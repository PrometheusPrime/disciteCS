Variables and Data Types

Variables

Use the var keyword to define a variable. If you define a variable and not use it, you'll get an error.

To declare a variable, write the var, variable name and then the data type. 
var favoriteNumber int
Assign or change variable's value. 
favoriteNumber = 5
Declare and initialise a variable 
var favoriteNumber int = 5
Go supports type inference. If you initialise a variable ,go will infer it's type from the value. 
var favoriteNumber = 5
Use := to declare and initialise a variable in a function, the var is keyword left out. 
age := 25
Declare and initialise multiple variables in a single line 
var firstName, lastName = "Nelson","Mandela"

var1, var2 := value1,value2

var (
    favoriteColor = "Dark Slate Gray"
    height = 5.3
)

Use the const keyword to define a constant. A constant's value can not be changed.

const pi = 3.1415

Data Types

  • Numbers

    • Integers are negative or positive whole numbers ...int int8 int16 int32(rune) int64
    To create an unsigned integer ,prefix the type with u uint uint8(byte) uint16 uint32 uint64 uintptr
    int,uint and uintptr are 32bits or more depending on the architecture of the computer.

    Floating Point (Floats) are Real Numbers with a Decimal Point.. float32 float64

    Complex : Go supports complex64 and complex128 with i as the imaginery part

  • Boolean

    • Boolean variables can have either true or false as values..

  • String

    • A String is a sequence of Characters, delimited by double quotes("").
