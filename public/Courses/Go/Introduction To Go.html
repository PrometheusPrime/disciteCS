Go

Go (Golang) is a statically typed, general purpose, compiled programming language. Go was made by a team at Google.

Installation

Go to go.dev/dl/ and download a Go installer for the OS you're using.

Write and Run Go programs

If you don't have Visual Studio Code installed you can download it from code.visualstudio.com/download.

Create a new file, name the it whatever you prefer with .go as the file extention and then write the code below. If you get pop ups requesting you to install go extension and any additional tools, click install.

If you're using a different editor and it support extentions, search and install go extension.

Alternatively, you can run go programs without installing go by going to go.dev/play/.

package main

import "fmt"

func main(){
  //print "Hello, world"
  fmt.Print("Hello, world")
}

To run the program, open a new Terminal window in the directory where your go file is located and type 'go run file.go' then click Enter .

package main : A go program is made up of packages. Go program execution starts in the main package.

import "fmt" : importing the format package which provides input and output functions

func main(){} : is the program execution starting point. Main runs automatically when the program is run.

// : comments starts with two forward slashes.

fmt.Print() : the print function in the format package is used to print text

To compile an executable file, go build package/path it needs a main package and main() function to be compiled as an executable.
