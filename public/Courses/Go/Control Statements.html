Control Statements

Control Statements are used to run code under certain conditions.

If, Else and Else If Statements

If Statements : if a certain condition is true, the code in the block will execute

if condition{
    code to be executed
}
Go uses cury brackets to delimit a block of code. 

if true{
    fmt.Println("That's True")
}

Nested If Statements: if statements can be nested.

var age = 22
if age > 18{
  if age <= 25{
    fmt.Println("You're between 18 and 25 years of age")
 }
}
Use logical Operators to chain conditions instead of nesting if statements. 
var is_cloudy = true
var is_raining = false
if is_cloudy || is_raining{
    fmt.Println("You'll need an Amburrella ")
}


var age = 22
if age > 18 && age <= 25{
    fmt.Println("You're between 18 and 25 years of age")
}

Else Statement : else statement will be executed if the condition or expression in the if Statement is false. 

var is_red = false
if is_red{
    fmt.Println("Stop")
}else{
    fmt.Println("Drive")
}
If - Else statements can be nested.

Else If Statement : else if checks additional conditions.

var name = "Johnson"
var gender = "female"
if gender == "male"{
    fmt.Println("Hello Mr " + name)
}else if gender == "female"{
    fmt.Println("Hello Ms " + name) 
}
An if statement can have any number of else if statements, but only one optional else statement that will handle any situations not handled by the preceding statements. 
var name = "Johnson"
var gender = "NA"
if gender == "male"{
    fmt.Println("Hello Mr " + name)
}else if gender == "female"{
    fmt.Println("Hello Ms " + name) 
} else{
    fmt.Println("Hello " + name)
}

Switch

The switch Statement is used do different operations depending on a particular value of a variable. It replaces multiple if-else statements. A variable is checked, if it matches a case, only that case's statement(s) is executed.

var num = 1
switch num{
case 1:
    fmt.Println("You're number 1!")
case 2:
    fmt.Println("You're number 2!")
default:
    fmt.Println("You're number something")
}
The default case is optional, it run if none of the cases matched. 
var num = 4
switch num{
case 1:
    fmt.Println("You're number 1!")
case 2:
    fmt.Println("You're number 2!")
default:
    fmt.Printf("Sorry,You're number %v",num)
}
A variable can be left out. When a variable is ommited a case doesn't have to be a constant, it can be an express or a call to a function.
