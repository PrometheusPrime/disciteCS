Operators

Arithematic Operators

Arithematic Operators are used in basic arithematic calculations

Addition Operator (+) : adds the operands.

a := 8
b := 5
fmt.Print(a + b)

Subtraction Operator (-) : subtracts one operand from another.

a := 8
b := 5
fmt.Print(a - b)

Multiplication Operator (*) : multiplies the operands.

a := 8
b := 5
fmt.Print(a * b)

Division Operator (/) : divides the operands. Dividing integers returns an integer, any remainder will be dicarded.

a := 8
b := 5
fmt.Print(a / b)
Modulo or Remainder Operator (%): returns the remainder of dividing the given operands. 

a := 8
b := 5
fmt.Print(a % b)
You can not do arithematic operations on mismatched types 
var age int = 25
var years int8 = 2 
fmt.Println(age + years)

Order of operation like in mathematics(Parenthesis,Exponetiation,Multiplication, Division,Addition,Subtraction)

Assignment Operators

Assignment Operators are used to assign values to variables...

Plus Equals (+=) adds the number on the right side of the equal sign to the value of the variable on left side of the equal sign and then assigns the result to that variable... 

a := 8
a += 6 // same as a = a + 6
fmt.Print(a)

Minus Equals (-=) subtracts the number on the right side of the equal sign from the value of the variable on left side of the equal sign and then assigns the result to that variable... 

a := 8
a -= 6 // same as a = a - 6
fmt.Print(a)

Multiplication Equals (*=) multiplies the number on the right side of the equal sign to the value of the variable on left side of the equal sign and then assigns the result to that variable... 

a := 8
a *= 6 // same as a = a * 6
fmt.Print(a)

Division Equals (/=) divides the variable on left side of the equal sign by the number on the right side of the equal sign and then assigns the result to that variable... 

a := 8
a /= 6 // same as a = a / 6
fmt.Print(a)

Remainder Equals (%=)

a := 8
a %= 6 // same as a = a % 6
fmt.Print(a)

Comparison Operators

Comparison Operators are used to compare given Variables, they return a Boolean value .

Greater Than Operator (>)

a := 8
b := 6
fmt.Print(a > b)

Less Than Operator (<) 

a := 8
b := 6
fmt.Print(a < b)

Greater Than or Equal To Operator (>=)

a := 8
b := 6
fmt.Print(a >= b)

Less Than or Equal To Operator (<=) 

a := 8
b := 6
fmt.Print(a >= b)

Equality Operator (==): checks whether the Operands are Equal.

a := 8
b := 6
fmt.Print(a == b)

Inequality Operator (!=): checks whether the Operands are Not Equal.Returns True if they are not equal.

a := 8
b := 6
fmt.Print(a != b)

Comparison Operators can be used to compare strings. Strings are compared according to the Alphabetical Order of their component Letters. 

fmt.Printf("bar < bus - %v\n","bar" < "bus")
fmt.Printf("bar <= bus - %v\n","bar" <= "bus")
fmt.Printf("bar > bus - %v\n","bar" > "bus")
fmt.Printf("bar >= bus - %v\n","bar" >= "bus")
fmt.Printf("bar == bus - %v\n","bar" == "bus")
fmt.Printf("bar != bus - %v\n","bar" != "bus")
Z is greater than A , lowercase letters are greater than Uppercase letters and letters are greater than Numbers.

Logical Operators

Logical Operators are used to chain Boolean variables or Boolean expressions, they return a Boolean value.

and (&&)Operator : Returns true if all the variables are true.

is_raining := true
is_sunny := false
fmt.Print(is_raining && is_sunny)

or (||) Operator : Returns true if one of the conditions is true.

is_raining := true
is_sunny := false
fmt.Print(is_raining || is_sunny)

not (!)Operator : Inverts a boolean value . If it was true, it becomes false. 

is_raining := true
fmt.Print(!is_raining)

Logical Operators can be chained and are evaluated according to Their order of precedence.
