Loop

Loops are used to repeatedly execute a block of code if a certain condition or an expression is true. In Go there's only one looping structure, the for loop.

for expression {
  //block of code to execute
}

var num = 1
for num <= 10 {
  fmt.Println(num)
}

If the condition does not evaluate to False, the Loop runs indefinitely (infinite Loop). Press Ctrl + C to stop execution.A condition or an expression can be updated inside the Loop so that it eventually evaluates to False. 

var num = 1
for num <= 10 {
  fmt.Println(num)
  num++
}

You can easily create an infinite loop by omiting an expression.

var num = 1
for{
  fmt.Println(num)
  num++
}
You can add the iteration variable, boolean expression and updation in the same line. 
for initialisation;expression;updation {
  //block of code to execute
}
First the iteration variable is initialised once before the first iteration. The condition or expression is evaluated before every iteration, if it evaluates to true, the body of the loop is executed. The updation is executed after the body of the loop is executed. 
for i := 1;i <= 10 ;i++ {
  fmt.Println(i*2)
}

You can not access the iteration variable outside of the loop.
