Functions

A Functions is a block of statement(s) that perform a specific Task. The func keyword is used to define a function. To create a function; write func, function name, followed by parentheses and then followed by curly brackets. Go uses curly brackets to delimmit a block of code. 

func function_name(){

}

A function is executed by calling it. To call a function, write the function name followed by parenthesis. Functions can be called as many times as needed.

package main
import "fmt"
        
func main(){
  
    greet()//calling a function
}
func greet(){
    fmt.Println("Hello World")
}

Parameters and Arguments

A Parameter is a variable(s) defined in the parenthes i when defining the function were if it takes arguments. Arguments are actual data passed into a function when calling it. A function can take any number of Parameters, comma seperate the parameters. A function must be called with the number of arguments it expects. 

func greet(first_name string,last_name string){
    fmt.Printf("Hello {first_name} {last_name}")
}
    greet('John','Doe')
If the parameters are of the same data type, you can omit the type from the parameters except the last one. 
func greet(first_name,last_name string){
    fmt.Printf("Hello {first_name} {last_name}")
}
greet('John','Doe')

positional and keyword args

Returning a value from a function

The return keyword is used to return a Value from a function. The type of data that will be returned from a function must be defined before the opening curly bracket. Statements after the return statement will cause an error.

package main
import "fmt"


func main(){
    
fmt.Println(greet())

}
func greet() string{
    return "Hello World"
}
