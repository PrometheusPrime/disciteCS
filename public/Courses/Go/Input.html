Getting Input

The Scanln() function from the fmt package is used to take input. Pass into the function the memory address of the variable where you want to store the input. 
var x,y int
fmt.Println("Enter a number :")
fmt.Scanln(&x)
fmt.Println("Enter another number :")
fmt.Scanln(&y)
fmt.Printf("sum of %v and %v is %v",x,y,x+y)
