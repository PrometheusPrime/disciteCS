Maps

A Map is a Derived Data Type with elements in key value pairs. A Map is defined with the following syntax. 

map := make(map[keyDataType] valueDataType)

daysInAWeek:= make(map[int8] string)
Add key value pairs to a map 
daysInAWeek[1] = "Sunday"
daysInAWeek[2] = "Monday"
daysInAWeek[3] = "Tuesday"
daysInAWeek[4] = "Wensday"
daysInAWeek[5] = "Thursday"
daysInAWeek[6] = "Friday"
daysInAWeek[7] = "Saturday"
Declare and initialise a map 
daysInAWeek := map[int8] string{
    1 : "Sunday",
    2 : "Monday",
    3 : "Tuesday",
    4 : "Wensday",
    5 : "Thursday",
    6 : "Friday",
    7 : "Saturday"
}
Get length of a map 
len(daysInAWeek)
Delete an entry delete() 
delete(daysInAWeek,1)
Map in a map 
programmingLanguages := map[string] map[string] string{
  "Python" := map[string] string{
    "released" : "",
    "by" : "",
    }
  "Go" := map[string] string{
    "released" : "",
    "by" : "",
    }
}
