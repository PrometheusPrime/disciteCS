Structures

A Structure is a collection of fields. To create a Structure write the type keyword, name of the structure, struct followed by curly brackets. A structure's properties are defined in the curly brackets.

type Employee struct {
    id        uint
    full_name, position  string
    salary    float32
}

manager := Employee{
    id:2022,
    full_name:"Jane Doe",
    position:"manager",
    salary:2500.0}
Alternatively, fields can be left out but you have pass the values in the order the properties are defined in the struct. 
doctor := Employee{2004,
    "Johnson Doe",
    "doctor",
    15000.0}

Struct riterals

var driver = Employee{}

var (
    secretary = Employee{}
)
Access struct fields by using the . notation 
fmt.Println(doctor.name)
doctor.name = "Gregory House"
fmt.Println(doctor.name)
