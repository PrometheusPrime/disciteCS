Packages

Import packages with the import keyword. If an imported package is not used, you'll get an error.

import "math"
Import multiple packages must be in seperate lines. 
import (
        "fmt"
        "math"
        "os"
)

Go to https://golang.org/pkg/" for a list of go packages

To get imformation on something run godoc package something in the terminal, for example to get imformation on the Print function in the fmt package godoc fmt Print().
