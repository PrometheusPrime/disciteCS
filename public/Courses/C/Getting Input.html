Getting Input

Get a character

To get a single character, use the getchar() function. 
char favoriteCharacter = getchar();
printf("Your favorite Character is %c", favoriteCharacter);

Get a String

Get a sequence of characters (string) using the gets() function. It takes the variable where you want to store the input as argument. Any input entered will be stored as a string. 
char name[12];
gets(name);

Another way of getting string input is by using the fgets function. Pass into the function the variable where to store, the number of characters and stdin

char firstName[9];
fgets(firstName,9,stdin);

Get input of any type

To get input of different data type use scanf(). It scans input that matches the format specifier, and places it at a memory address. scanf stops reading input when it encounters a space. 
float height;
scanf("%lf",&height);
Note; if the input you're getting is not a string, you have to append a & to the identifier.
