Loops

Loops are used to repeatedly execute code if a certain condition or an expression is True.

While Loop

The while loop is used to continually execute a block of statement(s) while a particular condition is true.

while(condition/expression){
    //TODO
}

The expression will be checked, if it evaluates to true; the statements in the while block will be executed. The while loop runs through the code block only if its condition is true, and will continue looping for as long as the condition remains true.

int num = 0;
while (num < 15){
    printf("%i\n",num);
}
If the condition does not evaluate to false, the Loop runs indefinitely (infinite Loop). Press Ctrl + C to stop execution. A condition or an expression can be updated inside the Loop so that it eventually evaluates to false. 
int num = 0;
while (num < 15){
    printf("%i\n",num);
    num++;
}

Do While Loop

The do while is the same as the while, except the condition is checked after each iteration. Therefore the statement(s) in the block are executed at least once, even if the condition is false.

int i = 10;

For Loop

The for Loop is used to run through a code block a specific number of times. 

for (initialisation; condition; updatation){
 //TODO
}
First an iteration variable is initialised and is always executed once before the loop. The second parameter holds the condition for the loop and is checked before each iteration. The third parameter contains the updatation of the iteration variable and is executed at the end of each loop. 
for (int num = 0;num <= 10;num++){
    printf("%i\n",num);
}
Explicitly create an infinte for loop 
for (;;) {
     printf("infinite loop ");
}

Break

The break keyword ends execution of a loop. 
int num = 0;
while (1==1){
    if(num > 100) break;
    printf("%i\n",num);
    num++;
}

Continue

Continue skips the rest of the current iteration and continues at the beginning of the next iteration. 
for(int num = 1;num <= 100;num++){
    if(num % 2 != 0) continue;
    printf("%i\n",num);
}
