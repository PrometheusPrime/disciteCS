Variables & Data Types

Variables

A variable is a container for storing data in memory. An identifier (variable name) can consist of letters, numbers and underscores, but it cannot start with a number, contain a space or special characters and must not be a reserved keyword. Variable names are case sensitive.

When declaring a variable, you have to specify the type of data it'll hold. The assignment operator = is used to assign a value to a variable.

char* firstName = "Jake";
Declare variable and then assign it a value later. 
char* lastName;
lastName = "Jones";
Declare multiple variables on a single line if they are of the same data type. 
float weight,height;
The value of a variable can be changed as many times as needed, but the type of that value must equal the type that variable can store. 
short age = 5;
age = 15;
age = 27;

Declare a constant variable with the const keyword, constants must be initialised.

const int DAYS_IN_A_WEEK = 7;

Data Types

There are four primitive data types in C; int, float, double and char.

int - Is used to store itegers (negative or positive whole numbers). The size of int depends on the architecture of the computer. If it is a 32 bit machine, then an int can store values from -2147483648 to 2147483647. 

int salary = 250000;
By default integers are signed, they store negative and positive numbers. If you only need to store positive values you can declare integer types as unsigned, their upper range will be doubled . 
unsigned int

float (floating point number) - real numbers with a decimal point. A float has an accuracy of up to 6 decimal places.

float height;

double - stores floating point numbers same as a float, but it is more precise. It has an accuracy of up to 15 decimal places

double pi = 3.141592653589793;

char - is used to store a single character. The value is enclosed between single quotes ('')

char questionMark = '?';
Alternatively, you can assign an ASCII code of the character. 
char hyphen = ;

Type conversion

Explicit cast is performed by placing the desired data type in parentheses before the variable or value that is to be converted. 
int num = 42;
double int_to_double = (double)num;
char int_to_char = (char)num;
When you convert a char to a number, you'll get an ASCII code of that character. 
int char_to_int = (int)'P';
