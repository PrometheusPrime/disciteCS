Strings

A String is an ordered, mutable sequence of characters. There no string type in C, string is an array of charaters. Strings are delimited by double quotes ("").

#include <stdio.h>
#include <string.h>
int main(){
            
char name[] = "Jane";
printf("%s",name);
            
return 0;
}

Indexing

The characters in a string are ordered, the first character is at position(index) 0. To get a character, write the name of the string followed by the index in square brackets.

printf("%c",name[2]);

We can change individual characters in a string. 

name[2] = 'k';
printf("%s",name);

Special characters

Use the Backslash (\) to escape any special Characters. For example \ has special function in a string, to display it we have to escape it. 
char path[] = "C:\\msys64\\clang64\\bin";
printf("%s",path);
Use (\n) to add a newline in a string. Any characters after (\n) will be pushed to a new line. 
char countDown[] = "3\n2\n1\n0";
printf("%s",countDown);

Strings and Comparison Operators

Comparison Operators can be used to compare strings. Strings are compared according to the Alphabetical Order of their component Letters. 
printf("%d\n","bar" < "bus");
printf("%d\n","bar" <= "bus");
printf("%d\n","bar" > "bus");
printf("%d\n","bar" >= "bus");
printf("%d\n","bar" == "bus");
printf("%d\n","bar" != "bus");
Z is greater than A , lowercase letters are greater than Uppercase letters and letters are greater than Numbers.

String functions

char name[] = "Jane";
The strlen() function returns the length (number of characters) of a string 
printf("%d",strlen(name));
Change all the letters in a string to uppercase. 
strupr(name);
printf("%s",name);
Change all the letters in a string to lowercase. 
strlwr(name);
printf("%s",name);
Reverse a string 
strrev(name);
printf("%s",name);
Copy a string 
char copy[] = "qwerty";
strcpy(copy,name);
printf("%s",copy);
strcmp() is used to compare strings. If the strings are the same, zero is returned. If the first string is greater than the second string, a positive number is returned. If the first string is less than the second string, a negative number is returned. 
printf("%i",strcmp("Hello","Hi"));
Merge strings 
char surName[] = " Doe";
strcat(name,surName);
printf("%s",name);
