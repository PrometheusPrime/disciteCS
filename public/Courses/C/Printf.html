Formating Printf

You can not directly print data that is not of type string (char*).

#include <stdio.h>
#include <stdlib.h>
            
int main(int argc, char const *argv[])
{
    /* code */
    int age = 42;
    printf(age);
                
    return 0;
}

If you want to print data of any other type, printf() function expects the first argument to be of type char * (a formating string), then the data.

Format Specifier

Format specifiers are placeholders and will be replaced by the by the corresponding argument after the formating string:

Format specifiers Type
%d or %i Integers
%ld or %li Long integers
%f Double or Float
%lf Double
%x Hexadecimal
%s Strings
%c Character 
#include <stdio.h>
#include <stdlib.h>
                
int main(int argc, char const *argv[])
{
    /* code */
    int age = 42;
    printf("%i",age);
                    
    return 0;
}
When formating multiple variables, the order of variables must match the order of the Format specifiers. 
#include <stdio.h>
#include <stdlib.h>
            
int main(int argc, char const *argv[])
{
    /* code */
    int age = 42;
    char name[] = "Morpheus";
    float height = 6.7 ;
    printf("My name is %s, i am %i years old and my height is %f",name,age,height);

    return 0;
}

printf("My name is %s, i am %i years old and my height is %.2f",name,age,height);

Other Output functions

putchar()

The putchar() function prints a single character.

char x =  '?';
putchar(x);

puts()

The puts() function is used to print a string. It automatically adds a new line. 

char *quote = "Life is really simple, but we insist on making it complicated - Confucius";
puts(quote);
It returns a positive integer if the string was successfully printed, else it returns a negative integer. 
char *quote = "Life is really simple, but we insist on making it complicated - Confucius";

int returnedValue = puts(quote);
printf("%d", returnedValue);
