Operators

Arithmetic Operators

Arithematic Operators are used in basic arithematic calculations.

Addition Operator (+) : adds the operands.

int x = 6;
int y = 8;
printf("%d",x + y);

Subtraction Operator (-) : subtracts one operand from another.

int x = 5;
double y = 2.5;
printf("%f",x - y);

Multiplication Operator (*) : multiplies the operands.

int x = 9;
int y = 25;
printf("%i",x * y);

Division Operator (/) : divides the operands. If you divide integers, an integer is returned.

int x = 27;
int y = 2;
printf("%i",x / y);
To get a floating point number, cast one of the operands to a float. 
int x = 27;
int y = 2;
printf("%f",(double)x / y);

Modulus or Remainder Operator (%) : returns the remainder after dividing the given operands. It can not be used with floating point numbers.

int x = 5;
int y = 3;
printf("%d", x % y);

Unary Operators

Unary Operator perform various operations on one operand.

Increment Operator (++) increments a variable by 1 

int x = 5;
x ++;
printf("%i",x);

Decrement Operator (--) decrements a variable by 1

int x = 5;
x --;
printf("%i",x);
When the unary operator prefixed, the operand is first changed and then the value is returned. If it is postfixed,the original value is returned before the operand is changed. 
int x = 5;
int y = 5;
printf("x = %i, y = %i", ++x, y++);

Assignment Operators

Operator (+=) 

int x = 8;
x+= 5;  //same as x = x + 5
printf("%i",x);

Operator (-=) 

int x = 8;
x-= 5;  //same as x = x - 5
printf("%i",x);

Operator (*=) 

int x = 8;
x*= 5;  //same as x = x * 5
printf("%i",x);

Operator (/=) 

int x = 8;
x/= 5;  //same as x = x / 5
printf("%i",x);

Operator (%=) 

int x = 8;
x%= 5;  //same as x = x % 5
printf("%i",x);

Comparison Operators

Comparison Operators are used to compare given Variables, they return a Boolean value 1(true) or 0(false).

Greater Than Operator (>) 

int x = 8;
int y = 5;
printf("%i",x > y);

Less Than Operator (<) 

int x = 8;
int y = 5;
printf("%i",x < y);

Equality Operator (==) : compares whether the Operands are Equal.

int x = 8;
int y = 5;
printf("%i", x == y);

Greater Than or Equal To Operator (>=) 

int x = 8;
int y = 5;
printf("%i",x >= y);

Less Than or Equal To Operator (<=) 

int x = 8;
int y = 5;
printf("%i",x <= y);

Inequality Operator (!=) : checks whether the Operands are Not Equal.Returns True if they are not equal.

int x = 8;
int y = 5;
printf("%i",x != y);

Logical Operator

Logical Operators are used to chain Boolean Variables, they return a boolean value (1 means true and 0 means false)..

And Operators (&&) : Returns true if all the variables are true.

int x = 8;
int y = 5;
printf("%i",x > 4 && y > 4);

Or Operator (||) : Returns True if one of the conditions is true.

int x = 8;
int y = 5;
printf("%i",x < y || y > x);

Ternary Operator

Ternary Operator(?:) can replace a single if - else statement. If the expression in parenthesis is true the variable is assigned the value after the question mark, else it is assigned the value after the colon.

char* x = (4 < 5) ? "4 is less than 5" : "4 is greater than 5";
printf("%s",x);
