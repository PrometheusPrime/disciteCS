Functions

A Function is a code blocks that does a task. To define a function, write the type of value it will return, followed by the function’s name, a set of parentheses and a code block. If a function does not return a value, use void as the return type.

void sum(){
    printf("5 + 5 = %i", 5 + 5);
}
A function will only execute when called. To call(invoke) a function, write the function’s name is specified followed by a set of parentheses. 
#include <stdio.h>

//function declaration
void sum(){
    printf("%i", 5 + 5);
}

int main(int argc, char const *argv[])
{
    sum();// function execution

    return 0;
}

Parameters and Arguments

Functions can accept data to work with called Parameters. A function can take any number of parameters, coma seperate them. Parameters appear in function definitions, while Arguments are actual data passed into a function when calling it.

//function declaration
void sum(int x,int y){
    printf("%i",x + y);
}
            
//in main
sum(5,4);
A function must be called with the number of arguments it expects, otherwise you'll get an error.

Default parameters

We can specify default values for parameters. If the argument for that parameter is not passed when the function is called, the default value will be used. The parameters with default values must come after those without default values.

//function declaration
void sum(int x,int y = 0){
    printf("%i", x + y);
}
                    
//in main
sum(5);
sum(5,4);

The Return statement

Return a value from a function with return keyword. Statement(s) after the return statement will not be executed.

int sum(int x,int y){
    return x + y;
}

//in main
sum(5);
sum(5,4);

Prototyping

Functions must be declared before they can be called. The following code will cause a compilation error.

#include <stdio.h>

int main(int argc, char const *argv[])
{
    printf("%d", sum(6, 9)); // function execution

    return 0;
}
// function definition
int sum(int x, int y)
{
    return x + y;
}

The function’s header(Prototype) needs to be specified before the point where the function is called, so that the compiler knows that the function exists. This is called Prototyping.

#include <stdio.h>

// prototype
int sum(int x, int y);

int main(int argc, char const *argv[])
{
    printf("%d", sum(6, 9)); // function execution

    return 0;
}
// function definition
int sum(int x, int y)
{
    return x + y;
}

Function Overloading

A Function can be defined multiple times with different number and or type of parameters. This is called Function Overloading

int sum(int x,int y){
    return x + y;
}

int sum(int x,int y,int z){
    return x + y + z;
}

double sum(double x,double y){
    return x + y;
}
double sum(int x,double y,double z){
    return x + y + z;
}

Pass by reference

A variable's address in memory is is passed in a function. When you make changes to it, the variable's actual value is changed. 
int main(int argc, char const *argv[]){

    char name[] = "John";
    printf("%s", name);
    changeName(name);
    printf("%s", name);

    return 0;
}

const char* changeName(char name[])
{
    name += " Doe";
    return name;
}

int main(int argc, char const *argv[]){
    changeName(name);
    return 0;    
}
