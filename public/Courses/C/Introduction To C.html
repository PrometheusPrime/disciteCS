C

C is a general purpose programming language. It was created by Dennis Ritchie in the 1970s.

C is a compiled Language, source code has to be compiled to machine code to run it. That means you need a Compiler to compile code.

Installing a compiler

Linux and MacOs

If you are using Linux or MacOS, you probably have everything you'll need to compile C code. Open a terminal window and run clang -version, if don't get any errors, you are good to go.

On Windows

  1. Go to https://www.msys2.org/ and download the installer. Run the installer then Follow all instructions on https://www.msys2.org/ carefully.
  2. Edit the system environment variables
    1. Click Windows + S to open the Search window.Type env,Click Edit the system environment variables.
    2. Under System Variables, select Path, then click Edit.
    3. Add new entrys C:\msys64\mingw64\bin and C:\msys64\usr\bin. Note you'll need change the paths if you installed msys64 on a different Drive. Click OK to save.
    4. On each window that was opened, click Apply or OK to dismiss it and apply the changes.
  3. Open a terminal window and type gcc --version then Enter to check if everything was installed and configured correctly.

Code Editor

You'll need a Text editor or an IDE to write your C programs.

A simple text editor that comes with your operating system will suffice, but you can use other code editors like Visual Studio Code , Atom and Sublime text.

I recommend you use Visual Studio Code. It's free and comes with a ton of features. You can download it from https://code.visualstudio.com/download. After installing Visual Studio Code, open it and install c/c++ extention.

Write a C program

Create a new file in the code editor of your choice, save it with .c as the file extention and write the code below.

#include <stdio.h>

int main(void)
{
    printf("Hello, World!\n");

    return 0;
}

Compile and Run a C program

In Visual Studio Code, press the play button in the top right corner of the editor to compile and run your program. Choose C/C++:gcc.exe build and debug active file.

Alternatively open a terminal window in the directory where your C file is , type make fileName without the file extention, then Enter. If no errors are shown it means the program compile successfully. There will be an executable file with the same file name as your C source file in the directory.

To run the executable file Linux or MacOS; type ./fileName, on Windows fileName then click Enter or return.

Output will be shown in the terminal.

Everytime you make changes to the source code, You have to recompile the code.

Sample Code Simplified

#include <stdio.h> - The preprocesor directive #include is used to add code from a header file in to our c source file before it is compiled. The <stdio.h> header file contains the printf function.

main() - is a special function in C where the execution of a program starts from.

{} - Curly brackets are used to delimit a block of code.

Comments are not executed , they are mainly used to document the Code.

// Single-line comments

/**/ Multi-line comments

printf() - Prints stuff passed to it to the console

\n - The new line sequence adds a new line.

return 0 - The exit status 0 indicates that the program executed successfully without any errors. We'll discuss the return statement later in the functions module.
