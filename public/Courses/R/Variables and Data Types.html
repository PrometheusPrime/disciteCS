Variables and Data Types

Variables

we can store data using variables. When defining a variable, it's name must start with a letter and can contain numbers, periods and underscores. Do not use Object names used in R as identifiers. There are several ways you can assign a value to a variable. 
x = 24
#Or
y <- 25
#Or
26 ‐> z
It is not reccommended to use = for assignment.

Data Types

Everything in R is an R Object. There are six R Objects; Vector, List, Matrix, Array, Factor and Data Frame. All of these will be looked at in details later.

Vectors (atomic Vectors)

There are six basic atomic Vector types, vectors with single values.

  • Numeric

    • Numeric any number; negative or positive, whole numbers or number with a decimal point. 
    # integers
student_age <‐ 16
student_height <‐ 6.24

  • Integers

    • Integers are negative or positive whole numbers. Append an L to the value. 
    # integers
student_age <‐ 16L

  • Complex

    • Complex with i as the imaginery part 
    # a complex
some_num <‐ 5+6i

  • Logical

    • logical variables can have either TRUE or FALSE / T or F as values. 
    # boolean
is_raining <‐ TRUE
is_sunny <‐ FALSE

  • Character

    • In R, a Character is a character or sequence of characters. Characters are delimited by either Single or Double quotes. 
    # single quotes
name <‐ 'John Doe'
# double quotes
grade <‐ "A"
    Raw Character my_var <- charToRaw("OMG")

Check Data Type

Use the class() function to check a variable's data type. 

student.height <‐ 18
class(student.height)
Check if a variable is of a specific type, returns a boolean. 
is.integer(student.height)
is.numeric(student.height)
is.character(student.height)
is.data.frame(student.height)

Convert Types

pi = 3.1415
as.integer(pi)
as.character(pi)
