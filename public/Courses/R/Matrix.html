Matrix

A matrix is a two-dimensional data set. values are stored in rows and columns use the matrix () function to create a matrix 
myMet <‐ matrix(2...4,ncol=,nrow=)
or 
my.Mtrx <‐ matrix(vector,ncol=,nrow=)
nrow - number of rows
ncol - number of columns
use the byrow argument to fill the row first 
my.Mtrx <‐ matrix(vector,ncol=,nrow=,byrow=T)
can also create a matrix by combining vectors using the rbind() function. 
my.mtrx <‐ rbind(1:6,7:12)
dim() function to get the dimension of the matrix 
dim(my.mtrx)
indexing 
myMet[row,column]

myMet[1,] #first row

myMet[,1] #first column

myMet[1,1] #first row,first column
change a value 
my.mtrx[row,col] <‐ new value
change entire row 
my.mtrx[row,] <‐ vector
change entire column 
my.mtrx[,col] <‐ vector
