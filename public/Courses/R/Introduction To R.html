Download and install R

Download R from https://cran.r-project.org and install it. On windows download the base release and add the folder where R is installed upto /bin to the path variable on the System variables in the System Environment Variables settings.

Run R program

Now that we have installed R, let's look at different ways we can run a simple R program below. 
#print 'Hello World' to the Console
print("Hello, World")

Using a Terminal

Open a terminal and type R, to run R interactively. You can write code and execute it directly in the Console.

R GUI

We can also use the R graphical user interface that comes with the installation of R. If you want to save your R code, save the file with .R as the file extention. You can run R code without saving the script and also directly in the console. You can even select the line(s) of code you wish to run.

Using R Studio

R Studio is the number one IDE used for R development, download it from here https://www.rstudio.com.

Open R Studio and select File, New File and then R Script. Just like in R GUI, you can ; run R code without saving the script, run code in the console. and also select the line(s) of code to run.
