Kotlin

Kotlin is a statically-typed, general-purpose programming language developed by JetBrain in 2011.
    Uses
  • Android App Development
  • Server side
  • Client side

You need a JDK (Java Development Kit) and IntelliJ to write and run a Kotlin program. IntelliJ comes bundled with Kotlin.

If you haven't already, download JDK from www.oracle.com/java/technologies/downloads/, install it and then add it to the enviroment variables

Download IntelliJ IDEA from https://www.jetbrains.com/idea/download/, the Community Edition will suffice.

Open IntelliJ and click NEW PROJECT. Write the project name. Make sure the language selected is Kotlin, the build system is Intellij and Add sample code is checked. Click CREATE.

Click the src folder and select Main.kt file, its where you'll be writing most of the code snipets in this course.

fun main(args: Array<String>){

    //print Hello world
    println("Hello World")

}

fun main() function is execution starting point of every kotlin program. Unless stated otherwise, it's where you'll be writing most of the code in this course.

// single-line comment in Kotlin

println() prints the argument passed to the Console.

Alternatively, you can write and execute Kotlin code snipets in the Kotlin REPL in IntelliJ. Click Tools, Kotlin, then Kotlin REPL.

Kotlin is interoperable with java, that means you can use Java code in Kotlin. So if you have you don't have to migrate everything to Kotlin.
