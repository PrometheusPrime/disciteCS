Encapsulation

Getters and Setters

Getters and Setters are automatically generated by the compiler, so when we access or modify properties we are not doing it directly.


    But sometimes the automatically generated Getters and Setters, for example the numberOfSatelites, angleOfRotation and distanceFromTheSun properties when modified outside the class can be assigned a negative numbers. 
    

    So we'll need to overwrite the default getters and setters.
    
class Planet(name: String, distanceFromTheSun: Double, surfaceArea: Double, numberOfSatelites: Int) {
    var name = name
        get() {
            return "The name of the Planet is $field"
        }
        set(value) {
            field = if (value.isEmpty()) {
                "Unkown"
            } else {
                value
            }
        }
    var distanceFromTheSun: Double = distanceFromTheSun
        set(value) {
            field = if (value < 0) {
                0.0
            } else {
                value
            }
        }
    var surfaceArea: Double = surfaceArea
        set(value) {
            field = if (value < 0) {
                0.0
            } else {
                value
            }
        }
    var numberOfSatelites: Int = numberOfSatelites
        set(value) {
            field = if (value < 0) {
                0
            } else {
                value
            }
        }

    init {
        if (name.isEmpty()) {
            this.name = "Unkown"
        } else {
            this.name = name
        }
        if (distanceFromTheSun < 0) {
            this.distanceFromTheSun = 0.0
        } else {
            this.distanceFromTheSun = distanceFromTheSun
        }
        if (surfaceArea < 0) {
            this.surfaceArea = 0.0
        } else {
            this.surfaceArea = surfaceArea
        }
        if (numberOfSatelites < 0) {
            this.numberOfSatelites = 0
        } else {
            this.numberOfSatelites = numberOfSatelites
        }
    }

    fun info(): String {
        return "Name : $name\nDistance From The Sun : $distanceFromTheSun\nSurface Area : $surfaceArea\nNumber Of Moons : $numberOfSatelites"

    }

}

    Note that the getters and setters must be defined imediately after the property.
