Maps

A collection of Key - value pairs . The key must be unique

fun main(){
    val mapName = mapOf(key to values, key to values, key to values)
}
By default a Map is immutable. To create a mutable Map, use the mutableMapOf() 
fun main(){
      val mapName = mutableMapOf(key to values, key to values, key to values)
}
create an empty Map 
val mymap = mutableMapOf<key-type, value-type>()
Add or change key - value pairs in a Map 
myMap[key] = value
//or
myMap.put(key, value)
remove an entry from a Map 
myMap.remove(key)
loop through a Map 
for((key, value) in myMap){
do something
}
