Loops

Loops are used to repeatedly execute a block of code.

while Loop

The while loop is used to repeat a block while a given condition or expression is true.

var num = 0

while(num <= 10){
    println(num)
    num++
}
Note we are updating the variable inside the Loop so that the condition eventually evaluates to false. If the condition does not evaluate to false, the Loop runs indefinitely (infinite Loop). 
var num = 0

while(num <= 10){
    println(num)
}
We can easily create an infinte loop by adding true as the condition. 
var num = 0

while(true){
    println(num)
    num++
}

Do... While Loop

The do... while loop is the same as the regular while, except in the do... while loop the condition is checked at the end. So it's block is executed at least once, even if the condition is false.

do {
    println("4 is greater than 5")
} while (4 > 5)

for loop

The for loop is used to loop through an iterable.

val myStr = "Whosoever Holds This Hammer, If They Be Worthy,Shall Posses The Power Of THOR"
for(character in myStr){
    println(character)
}

break and continue

continue skips execution of the rest of the code in the loop in the current interation and proceeds to the next interation.

val myStr = "Whosoever Holds This Hammer, If They Be Worthy,Shall Posses The Power Of THOR"
for(character in myStr){
    if(character == 'o' || character == 'O') continue
    print(character)
}

break exits the loop and execution goes to the first statement after the loop.

var num = 0
while(true){
    println(num)
    if (num == 100) break
    num++
}
