Conditional Statements

Control Statements are used to run code if certain conditions

The If statement

The if statement is used to execcute a block of statements if the given condition is true. The condition is either a boolean variable or expression that returns a boolean

val age = 25;
if(age > 18){
    print("You're more than 18 years of age")
}

If You want to execute some code if multiple conditions are true, place it in nested if statements.

val isCloudy = true
val isRaining = true
if(isCloudy){
    if(isRaining){
      println("It is croudy and raining ,You need an Amburrella")     
    }
}
Use Logical Operators to chain conditions instead of nesting if statements.

The Else statement

The else statement defines an alternate block of code to execute if the condition in the if statement is false.

val isRaining = false
if(isRaining){
    println("It is raining")     
}
else{
    println("It is not raining")
}

If Else statement

The else if statement is used to check for additional conditions. 
if(condition){

}else if(condition){

}

An if can have any number of elif statements, but only one optional else statement that will handle any situations not handled by the preceding statements.

if multiple conditions are true, only the first true statement will be executed other statements will be skipped.

Conditional Assignment

The if and else statements can be used to assign a value to a variable based on a condition.

val myVar = if(4 > 5) "4 > 5" else "4 < 5"

The When statement

The when statement is used we want to do different operations depending on a particular value of a variable.

val trafficLight = "Green"
when(trafficLight){
  "Red" -> println("Stop !")
  "Green" -> println("Drive")
  else -> println("Drive, if it is safe to do")

}

The when statement returns the last true expression, thus does not require a break statement. The else case is executed if non of the cases matched.

Since the when statement returns the last true case, it can be used as an expression .

val temperature = 35
val isCold = when(temperature){
    in -5..15 ‐> true
    else ‐> false
}
println(isCold)
