Variables and Data Types

Variables

A Variable is a placeholder for a value (data) in computer memory. Variables in Kotlin can be defined with the var and val keywords. Kotlin is statically typed, that means a variable's data type can not change.

Variables defined with var can be modified (reasigned a different value) while Variables defined with val are immutable (read only).

To create a variable, write var or val, the name of the variable. Followed by a colon, type of data the variable will hold, assignment operator (=) and the value. Assignment is evaluated from right to left.

val length: Int = 15

Kotlin has Type Inference, the compiler can figure out the data type of a variable based on the value assigned to it. 

var length  = 15
You can declare a variable and assign it a value later. Type must be specified if the variable is not initialised. 
val nextPresident: String

A top-level variable is a variable defined out side a block ()

Primitive Data Types

Kotlin supports the following Primitive Data Types :

Numbers

A Byte can store integers from -128 to 127 and uses 1 byte in memory.

val daysInAWeek : Byte = 7

A Short can store integers from -32768 to 32767 and uses 2 bytes in memory .

val studentAge : Short = 16

An Integer can hold a value ranging from -2147483648 to 2147483647 and uses 4 bytes in memory.

val secondsInAYear : Int = 31536000

A Long can hold a integer values from -9223372036854775808 to 9223372036854775807 . A Long uses 8 bytes in memory.

// a long
val milliSecondsInAYear : Long = 31536000000

Floating Point (Floats) are Real Numbers with a Decimal Point. Floats are less precise. Append an f to a value otherwise it will be infered as Double. 

// a float
val student_height : Float = 6.24f

Double are Real Numbers with a Decimal Point. Doubles are more precise than floats. 

// a double
val studentWeight : = 56.789

Boolean

Boolean variables can have either true or false as values..

// boolean
val isRaining = true
val isSunny = false

Character

Char character single quotes

// a character
val grade = 'A'
Top level variables, are variables declared in a Kotlin file outside a block of code.

Type Conversion

Numbers can not be implicitly converted, you have to cast them.

var pi = 3.14159
var piToInt = pi.toInt()
var piToLong = pi.toLong()
var piToChar = pi.toInt().toChar()
Note that you can call methods on numbers like they are objects even though they are primitive data types.
