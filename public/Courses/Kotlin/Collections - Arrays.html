Arrays

Use the arrayOf() function to create a array

val myArr = arrayOf('A',42,"Kotlin")
Create an array of a specific type 
val num : Array<Int> = arrayOf(6,24,42)
or use an appropriate typeArrayOf() function to create an array of a specific type 
val myCharArray = charArrayOf('V','W','X','Y','Z')
Alternatively use TypeArray class of the type of array you want to create. 
val evenNumsArr : IntArray = intArrayOf(2,4,6,8,10)
To print an array convert it to a string with either toString() method in the Arrays class or contentToString() 
println(Arrays.toString(evenNumsArr))
//Or
println(myCharArray.contentToString())

Indexing

Access an element using its index(position). Index starts from 0, that means the first element of an array is at 0.

fun main(){
    println(evenNumsArr[1])
}
Change a value in an array 
myCharArray[0] = 'Z'

Array Methods and Properties

Check the size or length of an Array with the size property. 
myCharArray.size
Use the .contains() to check if an element is in an Array, returns a boolean. 
myCharArray.contains('A')
The get() returns the array element at the specified index. 
fun main(){
    println(evenNumsArr.get(0))
}
Get the first element of an Array 
myCharArr.first()
Get the last element of an Array 
myCharArr.last()
Get the index of an element 
myCharArr.indexOf('X')
Get a number of elements from an Array as an Array. 
myArr = myCharArr.copyOfRange(1,3)
