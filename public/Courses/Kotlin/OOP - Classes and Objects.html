Class

A class is a blueprint for an object. Classes are declared using the class keyword. 

class Planet {
    var name = ""
    var distanceFromTheSun = 0.0
    var surfaceArea = 0.0
    var numberOfSatelites = 0


    fun info(): String {
        return "Name : $name\nDistance From The Sun : $distanceFromTheSun\nSurface Area : $surfaceArea\nNumber Of Moons : $numberOfSatelites"

    }
}

We've defined a class Planet with four properties and on function. A method in Kotlin is a function defined in a class.

We can define our Planet class in the Main.kt outside the main function or in a seperate Planet.kt file, in the same folder as Main.kt.

Objects

A Object is the actual thing created from the blueprint(class). Create an instance of a class using the following syntax. 
// in the main function
val earth = Planet()

We access the properties and methods of an object using the dot notation.

We can define a Primary Constructor by adding parenthesis after the class name. Then we define parameters in the constructor for each property so that we can initialise the properties when an object is created. 

class Planet(name:String,distanceFromTheSun: Double,surfaceArea: Double,numberOfSatelites: Int) {
    var name = name
    var distanceFromTheSun = distanceFromTheSun
    var surfaceArea = surfaceArea
    var numberOfSatelites = numberOfSatelites


    fun info(): String {
        return "Name : $name\nDistance From The Sun : $distanceFromTheSun\nSurface Area : $surfaceArea\nNumber Of Moons : $numberOfSatelites"

    }
}

If you're not doing anything with properties in the class body, they can be defined directly in the primary constructor.

class Planet(var name: String,var distanceFromTheSun: Double,var surfaceArea: Double,var numberOfSatelites: Int) {

    fun info(): String {
        return "Name : $name\nDistance From The Sun : $distanceFromTheSun\nSurface Area : $surfaceArea\nNumber Of Moons : $numberOfSatelites"

    }
}
We can give properties default values, so that an instance of the class can be created without passing arguments to the constructor. 
class Planet(
    var name: String = "",
    var distanceFromTheSun: Double = 0.0,
    var surfaceArea: Double = 0.0,
    var numberOfSatelites: Int = 0
) {

    fun info(): String {
        return "Name : $name\nDistance From The Sun : $distanceFromTheSun\nSurface Area : $surfaceArea\nNumber Of Moons : $numberOfSatelites"

    }
}

Initialiser Block

init{} is executed everytime an instance(object) of a class is created. we can use it to initialise the properties when an object is created. 
class Planet(var name: String, distanceFromTheSun: Double, surfaceArea: Double, numberOfSatelites: Int) {
    var distanceFromTheSun: Double
    var surfaceArea: Double
    var numberOfSatelites: Int


    init {
        if (distanceFromTheSun < 0) {
            this.distanceFromTheSun = 0.0;
        } else {
            this.distanceFromTheSun = distanceFromTheSun;
        }
        if (surfaceArea < 0) {
            this.surfaceArea = 0.0;
        } else {
            this.surfaceArea = surfaceArea;
        }
        if (numberOfSatelites < 0) {
            this.numberOfSatelites = 0;
        } else {
            this.numberOfSatelites = numberOfSatelites;
        }
    }

    fun info(): String {
        return "Name : $name\nDistance From The Sun : $distanceFromTheSun\nSurface Area : $surfaceArea\nNumber Of Moons : $numberOfSatelites"

    }
}
A class can have multiple init blocks, all of them will be executed in the order their defined in when a class is instantiated.

Secondary constructors

Add secondary constructors in the class body with the constructor keyword

class Planet(var name: String, distanceFromTheSun: Double, surfaceArea: Double, numberOfSatelites: Int) {
    var distanceFromTheSun: Double
    var surfaceArea: Double
    var numberOfSatelites: Int

    init {
        if (distanceFromTheSun < 0) {
            this.distanceFromTheSun = 0.0
        } else {
            this.distanceFromTheSun = distanceFromTheSun
        }
        if (surfaceArea < 0) {
            this.surfaceArea = 0.0
        } else {
            this.surfaceArea = surfaceArea
        }
        if (numberOfSatelites < 0) {
            this.numberOfSatelites = 0
        } else {
            this.numberOfSatelites = numberOfSatelites
        }
    }

    fun info(): String {
        return "Name : $name\nDistance From The Sun : $distanceFromTheSun\nSurface Area : $surfaceArea\nNumber Of Moons : $numberOfSatelites"

    }
//Secondary Constructors
    constructor() : this("", 0.0, 0.0, 0)
    constructor(name: String, numberOfSatelites: Int) : this(name, 0.0, 0.0, numberOfSatelites)

}
With this(), we are calling the primary constructor. Instead of defining secondary constructor(s) we can give properties default values like before, then the class can be instatiated in different ways.
