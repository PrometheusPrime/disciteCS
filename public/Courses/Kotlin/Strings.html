Strings

A String is an ordered collection of characters.

String Templates

String templates are use to interpolate a variable's value or result from an expression into a string. Add a $ before the name of the variable.

val age = 18
val myStr = "I'm $age years old"
println(myStr)
If you want to interpolate the value from an expression, add the expression in curly brackets. 
println("5 * 6 = ${5 * 6}")

Preformated String

To Display a string exactly the way it was formated in the code, delimit that string with three pairs of double quotes.

val myStr = """Whosoever Holds This Hammer,
If They Be Worthy,
Shall Posses The Power Of THOR."""
println(myStr)

indexing

val str = "Whosoever Holds This Hammer, If They Be Worthy,Shall Posses The Power Of THOR."
println(str[0])

String methods

val myStr = "Whosoever Holds This Hammer, If They Be Worthy,Shall Posses The Power Of THOR"
Check if the string is an empty string 
myStr.isEmpty()
Use the length property to check the length (number of characters) of a string. 
myStr.length
Get a character at a specified index. 
myStr.get(16)
The substring() method returns a sub string from the start index upto the stop index. 
myStr.substring(0,9)
The subSequence() mwthod returns a sub string from the start index upto the stop index. 
myStr.subSequence(0,9)
Use the equals() to compare strings, returns true if the strings are equal 
val greet = "Hello"
val str = "Hi"
greet.equals(str)
.compareTo() compares the strings, returns 0 if they are the same, else returns negative number. 
val greet = "Hello"
val str = "Hello"
println(greet.compareTo(str))
