Get Input

The readLine() function is used to get input from the user.

readLine()

To use the input from the user, you'll have to assign it to a variable.

val name = readLine()
println('Your name is ' , name)

By default the readLine() function returns a string...

val num1 = readLine()
val num2 = readLine()
println('''${num1} + ${num2} = ${num1 + num2}''')
In the example above, all the inputs are strings and are simply concatenated.

Convert numbers entered by the user to numbers(ints,double) , to use them as numbers.

val num1 = readLine()!!.toInt()
val num2 = readLine()!!.toInt()
println('''{num1} + {num2} = {num1 + num2}''')
The !! (not-null assertion operator) the expression on its left is not null. it is required.
