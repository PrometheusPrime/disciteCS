Strings

Strings are sequence of characters. There are two types of strings in Rust.

Primitive Strings

primitive strings are immutable 
let first_name = ""
let last_name: &str = ""

String Objects

When you create a string from the String class, You can modify it

fn main(){
    let mut greet = String::from("Hello");
    println!("{}",greet);
}

Formated Strings

let (firstName,lastName) = ("Jane","Doe");
println!("Your name is {} {} ?",firstName,lastName);
can also use index 
let (firstName,lastName) = ("Jane","Doe");
println!("Your name is {} {} ?",firstName,lastName);

String functions

check if a string is empty 
mystr.is_empty
count characters in a string 
mystr.chars().count()
check if a string is a sub-string 
myStr.contains(aStr)
replace 
myStr.replace(something,with)
Split a string at a specific index 
myStr.split_at(index)
