Operators

Arithmetic Operators

Arithematic Operators are used in basic arithematic calculations.

Addition Operator (+) : adds the operands.

let x = 6;
let y = 8;
println!({},x + y);

Subtraction Operator (-) : subtracts one operand from another.

let x = 5;
let y = 2.5;
println!({},x + y);

Multiplication Operator (*) : multiplies the operands.

let x = 9;
let y = 25;
println!({},x + y);

Division Operator (/) : divides the operands and returns a floating point number.

let x = 26;
let y = 2;
println!({},x + y);

Modulus or Remainder Operator (%) : returns the remainder of dividing the given operands.

let x = 5;
let y = 3;
println!({},x + y);

Unary Operators

Unary Operator perform various operations on one operand.

Increment Operator (++) increments a variable by 1 

let mut x = 5;
x ++;
println!({},x);

Decrement Operator (--) decrements a variable by 1

let mut x = 5;
x --;
println!({},x);
prefix vs postfix the post operator returns the original value before it changes the variable, while the pre-operator changes the variable first and then returns the value. 
let mut x = 5;
let mut y = 5;
println!({},++x);
println!({},y++);

Compound Assignment Operators

Operator (+=) 

let mut x = 8;
x+= 5;  //same as x = x + 5
println!({},x);

Operator (-=) 

let mut x = 8;
x-= 5;  //same as x = x - 5
println!({},x);

Operator (*=) 

let mut x = 8;
x*= 5;  //same as x = x * 5
println!({},x);

Operator (/=) 

let mut x = 8;
x/= 5;  //same as x = x / 5
println!({},x);

Operator (%=) 

let mut x = 8;
x%= 5;  //same as x = x % 5
println!({},x);

Comparison Operators

Comparison Operators are used to compare given Variables, they return a Boolean value (true or false).

Greater Than Operator (>) 

let x = 8;
let y = 5;
println!({},x > y);

Less Than Operator (<) 

let x = 8;
let y = 5;
println!({},x < y);

Equality Operator (==) : compares whether the Operands are Equal.

let x = 8;
let y = 5;
println!({}, x == y);

Greater Than or Equal To Operator (>=) 

let x = 8;
let y = 5;
println!({},x >= y);

Less Than or Equal To Operator (<=) 

let x = 8;
let y = 5;
println!({},x <= y);

Inequality Operator (!=) : checks whether the Operands are Not Equal.Returns True if they are not equal.

let x = 8;
let y = 5;
println!({},x != y);

Rust does not have a Ternary Operator, but you can use if and else statements to assign a value to a variable based on a condition.

let varX = if(condition) {value} else {anotherValue};
