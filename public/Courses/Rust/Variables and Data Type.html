Variables and Data Types

Variables

Variable are defined with the let keyword. Rust is Statically typed,everything has a type.

An identifier must start with a letter and can only contain letters, numbers and an underscore.

let greet: &str = "Hello";
 println!({},greet);
You need a placeholder ({}) to print a variable.

Rust supports type inference, The Rust Compiler will figure out the data type of a variable at compile time based on the value assigned to the variable. 

let greet = "Hello";
println!({},greet);
In Rust all variables are immutable by default. If you try reassigning a variable, I'll get an error (compile error). 
let greet = "Hello";
greet = "Hi";
println!({},greet);
Use the mut keyword to define a mutable variable. 
let mut greet = "Hello";
greet = "Hi";
println!({},greet);
Alternatively you can redefine a variable. Since the variable has been redefined, you can assign it a value of a different data type. 
let greet = "Hello";
let greet = 3110;
println!({},greet);
declare and assign multiple variables in a single line 
let (first_name,last_name) = ("Jane","Doe");
println!("Your name is {} {} ?",first_name,last_name);

Define a constant variable with the const keyword.

const NAME : &str = "";
println!({},NAME);
Note when defining a Constant, you have to explicitly state the type of data it'll hold.

Data Types

  • Numbers

    • Integers are negative or positive whole numbers. signed(i8,i16,i32,i64) and unsigned(u8,u16,u32,u64) integers.isize,usize space in memory is based on the buffer size of the computer. 
    // integers
let student_age: i8 = 16;

    Floating Point (Floats) are Real Numbers with a Decimal Point. Rust supports 32(f32) and 64(f64) bit floats. 

    // a float
let student_height = 6.24;

  • Boolean

    • Boolean variables can have either true or false as values.. 
    //explicitly state the type
let is_raining : bool = true;
//or
let is_sunny = false;

  • Characters

    Unicode characters 
    •  let grade = 'A'

    let char_var = '\u{code}';
    are delimited by single quotes.
