Rust

Rust is a C-like compiled programming language released by Mozilla in 2015. Rust is not Object Orientented.

Why Rust

  • Memory Safe

  • Free and Open Source You can use it freely.

  • Fast concurrency

  • Popular

  • Platform Independent

Installation

Go to https://rust-lang.org/tools/intall to install Rust on Windows.

On MacOS, Linux or WSL (Windows Subsystem for Linux) run curl --proto 'https' --tls in the Terminal to install Rust.

Alternatively run Rust online without installing anything at https://play.rust-lang.org

Create a Rust Project

Rust has a build system/package manager called Cargo. In the terminal run cargo init to create and initialise a Rust Project in the current directory.

cargo new ProjectName to creates a new folder ProjectName and initialises a new project in it. write source files in ProjectName/src folder.

cargo run builds and runs the project, cargo build just builds the project and cargo build --release builds a release version of the project.
