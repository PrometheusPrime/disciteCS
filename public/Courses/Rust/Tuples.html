Tuple

A Tuple is a Collection Data Type that ...

  • is ordered
  • is immutable
  • can contain elements of different Data Types
  • can contain duplicate entries
Tuple is create with parenthesis , comma seperate values... 
let a_tuple = ();
println!(a_tuple);

let a_tuple = ("Yes", 6,True,1.1);
println!(a_tuple);
