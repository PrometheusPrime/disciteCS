Loops

Loops are used to repeatedly execute code if a certain condition or an expression is True.

While Loop

While Loop is used to continually execute a block of statement(s) while a particular condition is true.

while(condition/expression){
    //TODO
}
The expression will be checked, if it evaluates to true; the statements in the while block will be executed.The while statement will continue checking the expression and executing its block of statements until the expression evaluates to false. The while loop runs through the code block only if its condition is true, and will continue looping for as long as the condition remains true. 
int num = 0;
while (num < 15)
{
    std::cout << num << std::endl;
}
If the condition does not evaluate to false, the Loop runs indefinitely (infinite Loop). Press Ctrl + C to stop execution.A condition or an expression can be updated inside the Loop so that it eventually evaluates to false. 
int num = 0;
while (num < 15)
{
    std::cout << num << std::endl;
    num++;
}
explicitly create an infinte loop 
int num = 0;
while (true){

    std::cout << num << std::endl;
    num++;
}

Do... While Loop

Do... While Loop is used to continually execute a block of statement(s) while a particular condition is true.It checks the condition after the code block. Therefore, the statement(s) in the block are executed at least once.

do {

    //TODO
        
} while (expr/condition);
 
int num = 20;

do {

    std::cout << num << std::endl;
    num++;

} while (num < 15);

For Loop

For Loop is used to run through a code block a specific number of times.

for (initialisation; condition; updatation){
 //TODO
}
The first one initializes a counter and is always executed once before the loop. The second parameter holds the condition for the loop and is checked before each iteration. The third parameter contains the increment of the counter and is executed at the end of each loop. 
for (int num = 0;num <= 10;num++){
    std::cout << num << std::endl;
}

for (;;) {

   cout << "infinite for-loop\n";
}

break

continue

continue skips the rest of the current iteration and continues at the beginning of the next iteration.
