C++

General Purpose, Compiled programming language , created by Bjarne Stroustrup. C ++ is heavyly based on The C Programming language, C code is valid C++ code.

To write a C++ program you need a text editor and a C++ compiler. Source code must be compiled into machine code inorder to run it. Note that each time you make changes ,you'll have to recompile it.

Linux and MacOs

If you are using Linux or MacOS, you probably have everything you'll need to compile and run C ++ code programs. Open a terminal window and run clang -version, if don't get any errors, you are good to go.

If you are on Windows, go to https://visualstudio.com/download and download Visual Studio Community Edition, Visual Studio installer selet .NET Desktop Development.
