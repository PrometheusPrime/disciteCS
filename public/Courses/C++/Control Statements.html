Switch

a variable that can have different value.

int temperature = 30;

switch (temperature) {
case  0:
    cout << "cold !";
    break;
case 30:
    cout << "warm";
    break;
default:
    cout << "hot !";
}
If the matching case has no break statement, all the cases that follow will be executed until a break statement is encountered, this is called Fall Through. 
int num = 4;

switch (num) {
case 5:
    cout << "5 !" << endl;
case 4:
    cout << "4 !" << endl;
case 3:
    cout << "3 !" << endl;
case 2:
    cout<< "2 !" << endl;
case 1:
    cout << "1 !" << endl;
default:
    cout << "Blast off!" << endl;
}
