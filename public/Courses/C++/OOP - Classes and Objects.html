Class

class keyword is used followed by a name, a code block and a semicolon. Fields(members or attributes) are variables defined in a class.

class Planet{
string name;
double surfaceArea;
int numberOfSatellites;
double distanceFromTheSun;

};

Create an Object

Classes, just like functions, must be declared before they can be referenced. Even with a prototype, you still cannot create an object of a class before it has been defined.

To create an object of the class, also called an instance; write the class name followed by the name of the object.

Planet earth;

Access modifiers

Access modifiers are used to control how the members of a class are accessed.

Class attributes are private by default, they can not be accessed outside the class. Other access modifiers include ; protected - accesible in the class and it's sub-classes, and public - accesible anywhere.

Write Access modifiers on a separate line, all fields and methods that follow will use that access modifier. 
class Planet{
public:
  string name;
  double surfaceArea;
  int numberOfSatellites;
  double distanceFromTheSun;
};

Planet earth;
earth.name = "Earth";
earth.surfaceArea = 510072000;
earth.numberOfSatellites = 1;
earth.distanceFromTheSun = 149600000;

Constructors

Constructors are special Methods called when instatiating an Object of a Class. Constructors have the same name as the class name and have no return type.We can initialise the members when an object is created with a constructor. 

class Planet{
public:
  string name;
  double surfaceArea;
  int numberOfSatellites;
  double distanceFromTheSun;

  Planet(string name,double surfaceArea,int numberOfSatellites,double distanceFromTheSun){

  name = name;
  surfaceArea = surfaceArea;
  numberOfSatellites = numberOfSatellites;
  distanceFromTheSun = distanceFromTheSun;

  }
};
Since we've overiden the default constructor, we can not create an instance without passing arguments to the constructor. 
Planet uranus("Uranus",8115600000,27,2870972200);
