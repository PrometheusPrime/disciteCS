Arrays

An Array is a container object that holds a fixed number of values of a single Data Type , referred by a single identifier. elements in an array are ordered.

To create an array, write the type of elements it'll hold, identifier, then the size of the array in square brackets. 
string fruitList[3];
You can assign values at the same time as the array is declared by enclosing them in curly brackets. The length of an array can be left out if you are declaring an array and adding element to it. 
string fruitList[] = {"Lime","Berries","Grapes"};

Indexing

Elements in an array can be accessed by referencing the elementâ€™s index inside square brackets. First element is at index zero. 
cout << fruitList[1] << endl;
Add or change an element at a specific index. 
fruitList[0] = "Apples";
Get the size of an array using the size() function. 
cout << size(fruitList) << endl;

Looping through arrays

Iterate through an array using a for loop 
int arrLength = size(fruitList);

for (int i = 0; i < arrLength; i++)
{
    cout << fruitList[i] << endl;
}
Or a for each loop 
for (string &fruit : fruitList)
{
    cout << fruit << endl;
}

Multi-Dimensional Arrays

string groceryList[4][3] = {
    {"Sugar","Flour","Baking Soda"},
    {"Spinach","Tomatoes","Carrots"},
    {"Lime","Berries","Grapes"},
    {"Beef","Pork","Fish"}
};
cout <<groceryList[1][2]<< endl;
Iterate through a Multi-Dimensional array using nested for loops 
int arraySize = size(groceryList);

for (int i = 0; i <= arraySize; i++)
{
    for (int j = 0; j <= i; j++)
    {
        cout << groceryList[i][j] << endl;
    }
}
