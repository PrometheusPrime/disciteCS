Lists

A container data structure,can expand and contract 
#include <list>
list<string> vegetablesList = {"Cucumbers","Lettuce","Tomatoes","Mushrooms","Potatoes","Zucchini","Peppers","Broccoli"};
Access and print list items using a for each loop 
for (string listItem : vegetablesList) {
    cout << listItem << "\n";
}
Add a value to the back of a list 
vegetablesList.push_back("Garlic");
Add a value to the front of a list 
vegetablesList.push_front("Carrots");
Get the size of the list 
vegetablesList.size();
Remove a value from the back of a list 
vegetablesList.pop_back();
Remove a value from the front of a list 
vegetablesList.pop_front();
Sort a list 
vegetablesList.sort();
Reverse a list 
vegetable.reverse();
Remove duplicates from a list 
vegetablesList.unique();
Remove a specific item from a list 
vegetablesList.remove("Broccoli");
Remove a specific item from a list if a particular condition is true 
vegetablesList.remove_if();
Merge lists with the merge(), list have to be sorted first. The list that was merged into the other list will be empty. 
list<string> groceryList{"Apples","Salt","Macaroni","Vinegar","Beef","Shampoo"};
groceryList.merge(vegetablesList);
