Geting Input

Use cin with the extraction operator (>>) to get input from the user.

int num1,num2;
cout <<"Enter a Number"<<endl;
cin >> num1;
cout <<"Enter another Number"<<endl;
cin >> num2;
cout <<num1 << " X " <<num2 <<"  = " <<num1 * num2<< endl;
Get more than one input in one line. 
int num1, num2;
cout << "Enter two numbers" <<endl;
cin >> num1 >> num2;
cout << num1 <<" X " << num2 <<"  = " << num1 * num2 << endl;

We can also use the getline() function

string name;
getline(cin,name)
getline() returns a string by default
