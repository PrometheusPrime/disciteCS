Operators

Arithmetic Operators

Arithematic Operators are used in basic arithematic calculations.

Addition Operator (+) : adds the operands.

int x = 6;
int y = 8;
std::cout << x + y;

Subtraction Operator (-) : subtracts one operand from another.

int x = 5;
double y = 2.5;
std::cout << x - y;

Multiplication Operator (*) : multiplies the operands.

int x = 9;
int y = 25;
std::cout << x * y;

Division Operator (/) : divides the operands. returns a floating point number if atleast on operand is a float.

int x = 26;
int y = 2;
std::cout << x / y;

Modulus or Remainder Operator (%) : returns the remainder of dividing the given operands.

int x = 5;
int y = 3;
std::cout << x % y;

Unary Operators

Unary Operator perform various operations on one operand.

Increment Operator (++) increments a variable by 1 

int x = 5;
x ++;
std::cout << x;

Decrement Operator (--) decrements a variable by 1

int x = 5;
x --;
std::cout << x;
prefix vs postfix the post operator returns the original value before it changes the variable, while the pre-operator changes the variable first and then returns the value. 
int x = 5;
int y = 5;
std::cout << ++x << std::endl;
std::cout << y++ << std::endl;

Compound Assignment Operators

Operator (+=) 

int x = 8;
x+= 5;  //same as x = x + 5
std::cout << x;

Operator (-=) 

int x = 8;
x-= 5;  //same as x = x - 5
std::cout << x;

Operator (*=) 

int x = 8;
x*= 5;  //same as x = x * 5
std::cout << x;

Operator (/=) 

int x = 8;
x/= 5;  //same as x = x / 5
std::cout << x;

Operator (%=) 

int x = 8;
x%= 5;  //same as x = x % 5
std::cout << x;

Comparison Operators

Comparison (or Relational) Operators are used to compare given Variables, they return a Boolean value (1 means true and 0 means false).

Greater Than Operator (>) 

int x = 8;
int y = 5;
std::cout << (x > y);

Less Than Operator (<) 

int x = 8;
int y = 5;
std::cout << (x < y);

Equality Operator (==) : compares whether the Operands are Equal.

int x = 8;
int y = 5;
std::cout << (x == y);

Greater Than or Equal To Operator (>=) 

int x = 8;
int y = 5;
std::cout << (x >= y);

Less Than or Equal To Operator (<=) 

int x = 8;
int y = 5;
std::cout << (x <= y);

Inequality Operator (!=) : checks whether the Operands are Not Equal.Returns True if they are not equal.

int x = 8;
int y = 5;
std::cout << (x != y);

Logical Operator

Logical Operators are used to chain Boolean Variables, they return a boolean value (1 means true and 0 means false)..

And Operators (&&) : Returns true if all the variables are true.

bool isRaining = true;
bool isSunny = false;
std::cout << (isRaining && isSunny);

Or Operator (||) : Returns True if one of the conditions is true.

bool isRaining = true;
bool isSunny = false;
std::cout << (isRaining || isSunny);

Not Operator (!) : inverts a boolean value . If it was True , it becomes False.

bool isRaining = true;
std::cout << (!isRaining);

Logical Operators can be chained and are evaluated according to Their order of precedence.

Ternary Operator(?:)Perfom assignment based on a condition. Ternary operator can replace a single if - else statement

std::string x = (4 < 5) ? "4 is less than 5" : "4 is greater than 5";
std::cout << x;
