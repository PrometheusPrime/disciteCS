Functions

Function are reusable code blocks that will only execute when called. To define a function, write the function's return type followed by the function name, a set of parentheses and a code block. If a function does not return a value, use void as the return type.

void greet(){

    cout << "Hello world!" << endl;
        
}
A function will only be executed when it is called. To call a function, write the function’s name followed by a set of parentheses. 
#include <iostream>

using namespace std;

void greet(){

    cout << "Hello world!" << endl;

}

int main(int argc, char const *argv[]){

    greet(); //calling the function
}

Parameters and Arguments

Parameters appear in function definitions, while arguments appear in function calls.A function can take any number of Parameters, comma seperate the parameters. A function must be called with the number of arguments it expects.

#include <iostream>
        
using namespace std;
        
void greet(string name){
        
    cout << "Hello, " + name << endl;
        
}
        
int main(int argc, char const *argv[]){
        
    greet("Jane"); //calling the function
}

Default parameters

You can specify default values for parameter(s) such that if that argument is not passsed when calling the function, the default value will be used instead. For this to work it is important that the parameters with default values are to the right of those without default values. All parameters after a parameter with a default value must have default values as well.

#include <iostream>
        
using namespace std;
        
void greet(string greeting,string name = "World"){
        
    cout << greeting + ", " + name << endl;
        
}
        
int main(int argc, char const *argv[]){
        
    greet("Hello");
    greet("Hi","Bob");

    return 0;
}

Return statement

The return keyword is used to return a Value from a function. Any statement after return will not be executed.

string greet(string greeting,string name)
{
    return greeting + ", " + name;
}
We can use the value returned from a function by printing it, assigning it to a variable, et cetra. 
cout << greet("Hi","Bob") << endl;
string greet(string greeting,string name)
{
    return greeting + ", " + name;
}
// in main
auto sayHello = greet;
cout << sayHello("Hello","James") << endl;

Prototyping

A function must be declared before it can be called.

#include <iostream>
        
using namespace std;
        
int main(int argc, char const *argv[]){
        
    cout << greet("Hi","Bob") << endl;

    return 0;
}
        
string greet(string greeting,string name)
{
    return greeting + ", " + name;
        
}
The function’s header needs to be specified before the point where the function is called, so that the compiler knows that the function exists. This is called Prototyping. And then add the function definition below.There's no need of declaring parameter names in the prototype, only the data types must be specified. 
#include <iostream>
                
using namespace std;

string greet(string,string);//prototype
                
int main(int argc, char const *argv[]){
        
    cout << greet("Hi","Bob") << endl;

    return 0;
}
        
string greet(string greeting,string name)
{
    return greeting + ", " + name;
        
}

Pass by Reference

By default arguments are passed by value to a function, the argument's value is copied.

#include <iostream>

using namespace std;

string changeName(string);

int main(int argc, char const *argv[])
{
    string name = "John";

    cout << name << endl;

    changeName(name);

    cout << name << endl;

    return 0;
}

string changeName(string x)
{
    x += " Doe";
    return x;
}

To pass an arguement's reference(memory location), add the Reference operator (&) before parameter name when defining the function.

#include <iostream>

using namespace std;
            
string changeName(string&);
            
int main(int argc, char const *argv[])
{
    string name = "John";

    cout << name << endl;

    changeName(name);

    cout << name << endl;
                    
    return 0;
}

string changeName(string &x)
{
    x += " Doe";
    return x;
}

Function Overloading

Function overloading is defining a function multiple times with different number and or type of parameters.

int sum(int x,int y){
    return x + y;
}

int sum(int x,int y,int z){
    return x + y + z;
}

double sum(double x,int y){
    return x + y;
}
double sum(double x,double y,double z){
    return x + y + z;
}
