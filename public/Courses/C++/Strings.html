Strings

A String is an ordered, mutable sequence of characters. String is not a primitive type in C++, so to work with strings include the string library or any library with string . Strings are delimited by double quotes ("").

string sayHello = "Hello World";
cout << sayHello;

Operating on Strings

The addition operator is used to concatenate strings

string firstName = "John";
string lastName = "Doe";
string name = firstName + " " + lastName;
cout << name;
You can not concatenate string literals. 
string sayHello = "Hello " + "World";
cout << sayHello;
To do this, one of the values has to be explicitly cast to a string. 
string sayHello = "Hello " + (string)"World";
cout << sayHello;
String literals will only be concatenated if the addition operator is left out. 
string sayHello = "Hello "  "World";
cout << sayHello;

The characters in a string are ordered, the first character is at position(index) 0. To get a character, write the name of the string followed by the index in square brackets.

string sayHello = "Hello World";
cout << sayHello[0];
We can change individual characters in a string. 
string myStr = "They";
myStr[3] = 'e';
cout << myStr << endl;
Copy a string 
string anotherStr = myStr;

Comparison Operators can be used to compare strings. Strings are compared according to the Alphabetical Order of their component Letters.

string bus = "Bus";
string bar = "Bar";
cout << (bar < bus) << endl;
cout << (bar <= bus) << endl;
cout << (bar > bus) << endl;
cout << (bar >= bus) << endl;
cout << (bar == bus) << endl;
cout << (bar != bus) << endl;
Z is greater than A , lowercase letters are greater than Uppercase letters and letters are greater than Numbers.
