Variables & Data Types

Variables

Variables are used to store data in computer memory.

A variable name can consist of letters, numbers and underscores, but it cannot start with a number. It also cannot contain spaces or special characters and must not be a reserved keyword. Use PascalCase or camelCase if a variable name has multiple words. Variable names are case sensitive.

When definig a variable, you need to specify the data type it will hold

char* firstName = "Peter";
You can declare a variable and then assign it a value later. 
char* lastName;
lastName = "Pan";
Multiple variables can be declared on a single line if they are of the same data type. 
char* firstName,lastName;
Use auto to let the compiler infer the data type of a variable based on the value assigned to it. The variable must be initialised. 
auto age = 25;
A Variable's value can as many times as needed. 
char* fullName = "Peter Pan";
fullName = "Peter Parker";

To declare a constant variable, use the const keyword

const int DAYS_IN_A_WEEK = 7;

Type conversion

Explicit cast is performed by placing the desired data type in parentheses before the variable or value that is to be converted. 
int num = 42;
cout <<(float)num << endl;
cout <<(double)num << endl;
cout <<(bool)num << endl;
cout <<(char)num << endl;
use functions 
cout << int(3.14159265);
cout << float(3.14159265);
cout << double(3.14159265);
When you convert a char to a number, you'll get an ASCII code of that character. 
cout << int('P');
When you convert a number to char, you'll get a character thats represented by that number in ASCII. 
cout << char(126);
If you convert any number other than 0 to Boolean, you'll get 1(true). 0 is false. 
cout << bool(42);

Variable Scope

A global variable is declared outside of any code blocks and is accessible from anywhere after it has been declared. A local variable, on the other hand, is declared inside a code block and will only be accessible within that block.

A Global variable will remain allocated for the duration of the program, while a local variable will be destroyed when its block of code has finished executing.
