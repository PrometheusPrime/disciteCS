Variables and Data Types

variables

Variables are used to store data in a computer memory variable names to access that data

A variable name can not be a reserved keyword, start with a number, contain a space or hyphen. Variable names are case sensitive . The equals sign (=) is used to assign a value to a variable. Assignment is evaluated from right to left.

Variables can be defined using three keywords; var, const and let.

var keyword is used to create a variable that has a Global Scope. Global variables are mutated in functions(other blocks), to avoid this behavior redefine that variable in the function(or block).

var name = 'Morpheus';

const keyword is used to create a constant variable. You have to initialise (give it a value) a constant variable otherwise you'll get an error.

const PI = 3.141592;

let keyword is used to create a variable that has a local scope(accessible within the block it declared in).

let num = 5;
Declare a variable and assign it a value later. 
let num;
num = 5;

Data Types

JavaScript is Dynamically typed, the data type of a variable can change thought the program. There are 7 data types in JavaScript :undefined, null ,boolean , String , symbol , number and Object.

Strings is a sequence of Characters.In JavaScript, strings are delimited by either Single quotes('') or Double quotes(""). 

let name = 'John';
console.log(name)

Numbers Integers (negative or positive whole numbers) or Floats (Real Numbers with a Decimal Point) 

const grade = 5;
let height = 6.4;
console.log(grade)
console.log(height)

Boolean variables can have either true or false as values

let isTall = true;
console.log(isTall)
note the values start with lowercase letters

Null value of nothing empty variable 

let name = null;
console.log(name)

Undefined without a value. have not set a value the variable is not defined 

let name = undefined;
let age;
console.log(name)
console.log(age)
variables declared with the let keyword are undefined by default.

Symbols immutable primitive type.

Use the typeof operator to check the data type of a variable.

var name = 'Morpheus';
console.log(typeof name);

parseInt() parseFloat()
