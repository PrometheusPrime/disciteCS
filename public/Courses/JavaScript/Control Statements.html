control Statements

If

The if statement's block will be executed if the condition is true.

if(4 < 5){
    console.log("4 is less than 5");
}

Else

The else statement defines an alternate block to execute if the conditon in the if statement is false

if(4 > 5){
    console.log("4 is greater than 5");
}else{
    console.log("4 is less than 5");
}

Else if

The else if statement is used to check additional conditions.

if(4 > 5){
    console.log("4 is greater than 5");
}else if(4 < 5){
    console.log("4 is less than 5");
}else{
    console.log("4 and 5 are equal");
}

If a Statement's condition is true that Statement's block will be executed, no other conditions will be evaluated and program execution go to the next line after the if elif else Statements. So be care when ordering the conditions.

We can only have one if statement; any number of else if statements; and one optional else statement that will handle any situations not handled by the preceding statements.

Switch Statement

The switch statement is used to do some operation depending on a value of a variable. The variable is compared to a case, if they match that case's statement(s) is executed. A case can only be a number, string or boolean.

const gender = "Male";
const lastName = "Smith";
switch (gender) {
  case "Male":
    console.log(`Hello Mr ${lastName}`);
    break;
  case "Female":
    console.log(`Hello Ms ${lastName}`);
    break;
  default:
    console.log(`Hello ${lastName}`);
    break;
}
The default case is executed if non of the cases matched.

if there isn't any break statement, all cases that follow will be executed until a break statement is found.

let num = 4;

switch (num) {
    case 5:
        console.log("5 !");
    case 4:
        console.log("4 !");
    case 3:
        console.log("3 !");
    case 2:
        console.log("2 !");
    case 1:
        console.log("1 !");
    default:
        console.log("Blast off!");
}
