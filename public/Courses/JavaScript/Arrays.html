Arrays

Arrays are ordered collection of multiple values referred by a single identifier. Array value can be of different data types. Arrays are objects, there is no array type in javascript.

Create an Empty Array 
let scores = new Array();
//Or
let anotherArr = [];
Create and initialise an Array 
const scores = new Array(3,6,87,3,2,1,9);
console.log(scores);
//Or
let anotherArr = [true,false,'Hello',5];
console.log(anotherArr);
Access an element using its index(position). The first element of an array is at 0. 
const scores = [3,6,87,3,2,1,9];
console.log(scores[0]);
Add or change a value in an array 
scores[0] = 77;
scores[7] = 69;
console.log(scores);
length(number of Elements) of an Array with the length property 
console.log(scores.length)
Delete an element from an array 
delete scores[2];
console.log(scores);
Delete all the elements of an array by setting it's length to 0. 
scores.length = 0;
console.log(scores);

Nested Arrays

A multi dimensional array an array containing other arrays (it's elements are arrays). 
let myArr = [
    [2, 4, 8, 16],
    [3, 9, 27, 81],
    [4, 16, 64, 256]
];
console.log(myArr[2][3]);
When indexing a 2 dimensional array, the first square brackets is seleting a nested array and the second brackets holds the index of an element in that array.

Methods

Add element(s) at the beginning of an array with the unshift() method. 
const scores = [3,6,87,3,2,1,9];
scores.unshift(false);
console.log(scores);
The push() add element(s) at the end of an array. 
const scores = [3,6,87,3,2,1,9];
scores.push(true);
console.log(scores);

The pop() function removes and returns the last value in an array

const scores = [3,6,87,3,2,1,9];
const lastValue = scores.pop();
console.log(lastValue);
console.log(scores);

The shift() function removes and returns the first value in an array

const scores = [3,6,87,3,2,1,9];
const firstValue = scores.shift();
console.log(firstValue);
console.log(scores);

To check whether something is an array use the isArray() method from the Array object.

const scores = [3,6,87,3,2,1,9];
console.log(Array.isArray(scores));

Get an index of a value indexOf(). If a value is not in an Array, indexOf() returns -1.

const scores = [3,6,87,3,2,1,9];
console.log(scores.indexOf(87));

The reverse() method reverses the elements in an array.

const scores = [3,6,87,3,2,1,9];
scores.reverse();
console.log(scores);

The sort() method sorts the elements in an array in ascending order.

const instruments = [
    "Saxophone",
    "Organ",
    "Ocarina",
    "Acoustic Guitar",
    "Trumpet",
    "Triangle",
    "Shakers",
    "Vibraphone",
    "Bagpipes",
    "Keyboard"
];
instruments.sort();
console.log(instruments);
From the example below, we can see that sorting numbers can produce unexpected results. This is because the method converts any element that is not undefined to string and sorts them according to their unicode number.
const scores = [3,6,87,3,2,1,9];
scores.sort();
console.log(scores);
We must pass a sorting function into the method, that returns 0 if the numbers being compared are the same (and therefore sorted), a negative number (also sorted if we are sorting in ascending order) and a positive number (first number is less than the second, they must be swaped if we are sorting in ascending order) 
const scores = [3,6,87,3,2,1,9];

scores.sort((a, b) =>  
    a - b
);

console.log(scores);

The splice() method removes elements from an array and returns array of the ramoved elements. Pass into the method an index where to start deleting. You can also pass a second arguement the number of elements you wish to remove.

const scores = [3,6,87,3,2,1,9];
const invalidScores = scores.splice(2,3);
console.log(invalidScores);
The toString() method returns a string representation of an array, with the elements comma-separated. 
const scores = [3, 6, 87, 3, 2, 1, 9];
console.log(scores.toString());
The join() method returns a string representation of an array. Pass a string to use to separate elements of the array. 
const scores = [3, 6, 87, 3, 2, 1, 9];
console.log(scores.join(":"));

Use the concat() method to join arrays.

const scores = [3, 6, 87, 3, 2, 1, 9];
let anotherArr = [true,false,'Hello',5];
console.log(scores.concat(anotherArr));

The .forEach(function) method executes the function on each element of the Array.

const scores = [3, 6, 87, 3, 2, 1, 9];
scores.forEach((score) => {
    console.log(score)
});

const scores = [3, 6, 87, 3, 2, 1, 9];
scores.forEach((score, index) => {
    console.log(`${score} is at ${index}`)
});

The .map(function) method executes the function on each element of the Array and returns a new Array.

const scores = [3, 6, 87, 3, 2, 1, 9];
const standardisedScores = scores.map((score) => score / 100);
console.log(standardisedScores);

values()

const scores = [3, 6, 87, 3, 2, 1, 9];
const itr = scores.values();
console.log(itr.next().value);
console.log(itr.next().value);

The includes() method checks if an element in the array.

const scores = [3, 6, 87, 3, 2, 1, 9];
console.log(scores.includes(5));

The find() method returns the first element that passes the test by the provided function. If no element passes the test, undefined is returned.

const scores = [3, 6, 87, 3, 2, 1, 9];
const evenScore = scores.find((score) => score % 2 == 0);
console.log(evenScore);

The .filter() method returns an array with elements that passes the test by the provided function, if no element passes the test an empty array is returned.

const scores = [3, 6, 87, 3, 2, 1, 9];
const oddScores = scores.filter((score) => score % 2 !== 0)
console.log(oddScores);

The reduce() method executes the function on each element of the array, passing the return value from the previous calculation. It returns a single value resulting from reducing the array.

const scores = [3, 6, 87, 3, 2, 1, 9];
const totalScore = scores.reduce((total,currentScore) => total + currentScore);
console.log(totalScore);

loop through Arrays

We can use different looping structures to transverse through an array.

Regular for loop 
const scores = [3, 6, 87, 3, 2, 1, 9];
for (let index = 0; index < scores.length; index++) {
    delete scores[index];
}
console.log(scores);
When you're transversing through an array's elements using their index, it is easy to get the index with the for ..in loop. 
const instruments = [
    "Saxophone",
    "Organ",
    "Ocarina",
    "Acoustic Guitar",
    "Trumpet",
    "Triangle",
    "Shakers",
    "Vibraphone",
    "Bagpipes",
    "Keyboard"
];
for (const index in instruments) {
    console.log(instruments[index]);
}
for ..of loop 
const week = [
    "Sunday",
    "Monday",
    "Tuesday",
    "Wensday",
    "Thursday",
    "Friday",
    "Saturday",
];
for (const day of week) {
    console.log(`Today is ${day}`);
}
