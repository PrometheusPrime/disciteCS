Strings

Strings are ordered sequence of characters delimited with single or double quotes

let greet = 'hi there !'
console.log(greet);
The backslash (\) is used to escape special characters in a string. 
let directory = "C:\\Users\\CSf\\.bash_history";
console.log(directory);

Accessing character at index

let greet = 'hi there !';
console.log(greet[0]);
Strings are immutable, You can not change individual characters in a string. 
let greet = 'hi there !'
greet[0] = 'H'
console.log(greet);

Template strings are delimited by back ticks ``, they are use to interpolate a variable's value or result from an expression into a string. Add the variable or expression in curly brackets prececeded by a $.

let a = 5;
let myStr = `${a} × ${a} = ${a * a}`;
console.log(myStr);

The addition operator is used to concatenate strings.

let x = 'Java';
let y = 'Script';
let language = x + y;
console.log(language);
If you add a string and a number, you'll get a string. The number is implicitly converted to a string and then concatenated with the other string. 
let a = 10;
let b = '24';
let c = a + b;
console.log(c);

String Properties and Methods

Use the length property to get the length of a string 
let greet = 'hi there !';
console.log(greet.length);

console.log(greet.toUpperCase());

let greet = 'HI THERE!';
console.log(greet.toLowerCase());
Get start index of a substring, returns -1 is a string is not found. 
console.log(greet.indexOf('there'));
Get a character at a specified index. 
console.log(greet.charAt(9));
Get a substring from specified index to the end of a string. You can add an optional stop index. 
console.log(greet.substring(3));
Slice a string from the start index upto but not including the end index. The end index is optional. 
console.log(greet.slice(0,3));
Replace a character in a string 
console.log(greet.replace('h','H'));
Convert a string to an Array, specify where to split the string. 
let greet = 'hi there !';
console.log(greet.split(' '));
