Sets

A collection of unique values. A set can contain values of any type. 
const mySet = new Set();

The add method adds a new element if it's not already in the set.

mySet.add(3.142);
mySet.add(42);
mySet.add(true);
mySet.add('False');
mySet.add(true);
console.log(mySet);

The size property returns the number of elements in the set.

console.log(mySet.size);

You can check if an item is in a set with the has() method.

console.log(mySet.has(false));

The .forEach(function) method calls the function on each element of the set.

mySet.forEach(element => {
    console.log(element)
});

The delete method deletes an element from the set.

mySet.delete();

The clear method deletes all elements from the set.

mySet.clear();
