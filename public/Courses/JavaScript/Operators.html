Operators

Arithmetic Operators

Addition Operators (+) : adds the operands.

let a = 6;
let b = 4.76;
console.log(a + b);

Subtraction Operators (-) : subtracts one operand from another.

let a = 8;
let b = 45;
console.log(a - b)

multiplication Operators (*) : multiplies the operands.

let a = 8;
let b = 6;
console.log(a * b)

Division Operators (/) : divides the operands.

let a = 126;
let b = 9.0;
console.log(a / b)

Modulus or Remainder Operators (%) : returns the remainder of dividing the given operands.

let a = 6;
let b = 4;
console.log(a % b)

increment Operators (++) : increments a variable by 1. 

let a = 6;
a++; // same as a = a + 1
console.log(a);

decrement Operators (--) : decrements a variable by 1.

let a = 7;
a--; // same as a = a - 1
console.log(a);
Postfix returns the original value and then updates it after,while prefix - the operator changes the variable first and then returns the value. 
let a = 5;
let b = 5;
console.log(a++);
console.log(++b);

Assignment Operators

do operations on a variable and other operand(s) and then assign the result of the operation to that variable.

Operators (+=) 

let a = 6;
a += 7; // same as a = a + 7
console.log(a)

Operators (-=) 

let a = 5;
a -= 9; // same as a = a - 9
console.log(a)

Operators (*=) 

let a = 8;
a*= 6; // same as a = a * 6
console.log(a)

Operators (/=) 

let a = 125;
a/= 2.5; // same as a = a / 2.5
console.log(a)

Operators (%=) 

let a = 15;
a %= 2; // same as a = a % 1
console.log(a)

Comparison Operators

Comparison Operators are used to compare given Variables, they return a Boolean value .

Greater Than Operators (>) 

let a = 8;
let b = 6;
console.log(a > b)

Less Than Operators (<) 

let a = 8;
let b = 6;
console.log(a < b)

Equality Operators (==) equal converts operands to a common type and then checks if the operands are equal.

let a = "8";
let b = 8;
console.log(a == b)

Strict Equality Operators (===)Checks if the operands are equal and of the same data type. Does not convert the operands to a common type.

let a = "8";
let b = 8;
console.log(a === b)

Greater Than or Equal To Operators (>=) 

let a = 8;
let b = 6;
console.log(a >= b)

Less Than or Equal To Operators (<=) 

let a = 8;
let b = 6;
console.log(a <= b)

Inequlity Operators (!=) : checks whether the Operands are Not Equal.Returns True if they are not equal.

let a = 8;
let b = 6;
console.log(a != b)

Operators (!==) Checks if the operands are Not equal and or Not of the same data type. 

let a = 8;
let b = "8";
console.log(a !== b)

Logical Operators

And Operators (&&) : Returns True if all the variables are true.

let isRaining = true
let isSunny = false
console.log(isRaining && isSunny)

Or Operators (||) : Returns True if one of the conditions is true.

let isRaining = true
let isSunny = false
console.log(isRaining || isSunny)

Not Operators (!) : inverts a boolean value . If it was True , it becomes False.

let isRaining = true
console.log(!isRaining)

Ternary operator

let a = 4 > 5 ? true : false;
console.log(a)
