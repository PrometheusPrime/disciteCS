Maps

A map is a collection of key-value pairs. The key must be unique but any thing can be used as a key (or value).

const independenceDay = new Map();

Add a new key-value entry to a map with the set() method. If an entry with the same key exists, that entry is updated.

independenceDay.set('Angola', '11/11/1975');
independenceDay.set('Botswana', '30/09/1966');
independenceDay.set('Zambia', '24/10/1964');
independenceDay.set('Zimbabwe', '18/04/1980');
console.log(independenceDay);

The size property returns the size of the map.

console.log(independenceDay.size);

The get() method returns the value associated with that key, if theres no entry with that key, undefined is returned.

console.log(independenceDay.get('Angola'));

The has() method checks if there is an entry with the specified key.

console.log(independenceDay.has('Angola'));

The entries() method returns an iterable of all key-value pairs in the map.

console.log(independenceDay.entries());

The keys() method returns an iterable of all keys in the map

console.log(independenceDay.keys());

The values() method returns an iterable of all values in the map

console.log(independenceDay.values());

We can execute a function on each map entry with the forEach() method.

independenceDay.forEach((date, country) =>
    console.log(`${country} got it's independence on ${date}`)
);

Use the delete() method to delete a key-value pair from the map. If the entry was in the Map and has been deleted the method returns true else it returns false.

independenceDay.delete('Zimbabwe');

The clear() method deletes all key-value entries from the map.

independenceDay.clear();
