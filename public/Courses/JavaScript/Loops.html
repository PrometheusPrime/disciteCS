Loops

Loops are used to repeatedly execute a block of code. Instead of writing n lines of the same code, use loops to repeatedly execute that code n times.

for loop

for block of statements to be executed, as long as a condition is true. for loop is used when

for(looping variable; expression; updatation){
    //Todo
}

for (let i = 1; i <= 10; i++) {
    console.log(i);
}
The for loop with in is used through all individual elements in an Iterable or a Collection. It stops execution when it reaches the last one. 
let alphabet = 'ABCDEFGHIJKLMNOPQRSTUVWXYZ';
for(let index in alphabet){
    console.log(alphabet[index]);
}

let alphabet = 'ABCDEFGHIJKLMNOPQRSTUVWXYZ';
for (const letter of alphabet) {
    console.log(letter);
}

While loop

while loop executes a block of statements, and repeats the block, while a condition is true. In a while loop, the condition is first checked so the loop will not be executed if the condition is false.

while(conditon){
    to do
}

Do ... while loop

do while loop executes a block of statements, and repeats the block, while a condition is true. In a do while loop, the condition is checked last so the loop will be executed atleast once even if the condition is false.

do{
    to do
}while(conditon);

Break and Continue

break exits the loop and execution goes to the first statement after the loop.

continue skips execution of the rest of the code in the loop in the current interation and proceeds to the next interation.
