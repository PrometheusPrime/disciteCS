Images, Audio and Videos

Images

The <img/> element is used to add an image to a web page.The source (src) attribute defines the image location. The width and height attributes are used to set the width and height of the image. The width and height of an image can be in pixels (px) or percentages (%). The alt attribute contains alternate text that describes the image if it can not be displayed. If the user uses a screen reader, the alternate text will be read to them.The alt attribute is required. The border attribute adds a border around the image 

<!DOCTYPE html>
<html lang ="eng">
<head>
  <title>Images</title>
</head>
<body>
 <img src="logo.jpg" width="100px" height="150px" border="1px" alt="site logo">
</body>
</html>

src="logo.jpg" means there is a file logo.jpg in the folder where the html file is. If it is in a different folder you'll have to spacify the path to the image. You can use an image from another website.. 

<!DOCTYPE html>
<html lang ="eng">
  <head>
    <title>Images</title>
  </head>
  <body>
   <img src="logo.jpg" width="50%" height="50%" alt="site logo">
  </body>
</html>
The width , height and border can be set with the style attribute..
style="width:50%;height:50%;border:0;"
Using large images can slow down the loading of your page.

Audio

The <audio> element is used to embed an audio in a web page.There are three supported audio formats : OGG, WAV and MP3. The source (src) attribute defines the audio location.The controls attribute adds control buttons. When the autoplay attribute is defined ,the audio will start playing once it is ready. When the loop attribute is defined ,the audio will replay every time it is finished.

<!DOCTYPE html>
<html lang ="eng">
  <head>
      <title>Audio</title>
  </head>
  <body>
  <audio src="audio.wav" controls autoplay loop></audio>
  </body>
</html>

You can add different file formats for the audio using the source elements nested in the audio. The first format that your browser supports will be played. If the browser doesn't support any of the files, the text will be printed.

<!DOCTYPE html>
<html lang ="eng">
 <head>
    <title>Audio</title>
 </head>
 <body>
    <audio controls autoplay loop>
       <source src="audio.ogg" type="audio/ogg">
       <source src="audio.wav" type="audio/wav">
       <source src="audio.mp3" type="audio/mpeg">
       Your Browser can not play this audio
    </audio>
 </body>
</html>

Videos

The <video> element is used to embed a video in a web page.There are three supported video formats : OGG, WebM and MP4. The source (src) attribute defines the video location.The controls attribute adds control buttons. When the autoplay attribute is defined ,the video will start playing once it is ready. When the loop attribute is defined ,the video will replay every time it is finished.

<!DOCTYPE html>
<html lang ="eng">
<head>
  <title>Video</title>
</head>
<body>
  <video src="video.webm" controls autoplay loop></video>
</body>
</html>

We can also add different video formats, just like the audio element.

<!DOCTYPE html>
<html lang ="eng">
<head>
  <title>Video</title>
</head>
<body>
  <video controls autoplay loop>
    <source src="video.ogg" type="video/ogg">
    <source src="video.webm" type="video/webm">
    <source src="video.mp4" type="video/mpeg">
    Your Browser can not play this video
  </video>
</body>
</html>
