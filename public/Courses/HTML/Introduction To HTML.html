HTML

HTML (Hyper Text Markup Language ) is the language used to describe the structure and content of a Web page. HTML is not a Programming Language, it is a Markup Language which uses Tags to discribe how content should be rendered by a programme( like a Browser).

Tags usually come in Opening and Closing Tag pairs with the content in the middle.Tag names are surrounded by angle brackets, the Closing Tag has a forward slash before the tag name.

<OpeningTag> Content </ClosingTag>

HTML Element are made up of the Opening Tag, Content and the Closing Tag.

Empty Elements do not have Closing Tags. The forward Slash (/) is optional, it comes after the element's name .

<br/>
The <br/> element adds a line break in text.

Types of Elements

Elements are either Block Level Elements (displayed with line breaks) or Inline Elements( displayed without line breaks).

How to create a Web page

HTML Documents can be created using a simple Text Editor.

Sample Code

<!DOCTYPE html>
<html>
 <head>
  <title>Hello HTML</title>
 </head>
 <body>
  <!-- My first HTML file -->
  Hello There !!
 </body>
</html>

Write the code above into your Text Editor. Name the file whatever you prefer with .htm or .html as the file extention and then save it. When saving the file, if the Text Editor you're using has an option to set character encoding, set it to UTF-8.

Go to the directory where you saved the file and double click it to open it.

The file will open in your default Browser.

Sample Code Simplified

<!DOCTYPE html> : Browsers can open different types of documents (like pdf et cetra). This declaration tells the browser that the Document type is HTML (and the version of HTML used), so that the document can be rendered correctly.

<html> : Is the root element, it marks the beginning and end of an HTML Document.

<head> : The head's content is not displayed on the page, it contains metadata which provides more information about the document which is important to search engines and browsers. Head also contains links to other resources (CSS, Javascript etc). More on this later.

<title> : Contains the title of the document which is shown in the browser's title bar or the page's tab, not on the page.

<body> : Contains all the visible content ( text, images, video, links et cetra) on the web page.

<!-- --> : Anything placed between these tags is a comment. Comments are not displayed by the Browser,they are mainly used to document the HTML Document. Note the exclamation point (!) in the opening, but not in the closing tag.

<!--[if IE 9]> HTML here <![endif]--> : Conditional comments defines HTML tags to be executed by Internet Explorer.

Attributes

Attributes are specified in the opening tag ,they provide additional information about HTML elements. Attributes come in name - value pairs.The value of the attribute is surrounded by either double quotes ("") or single quotes ('').

<html lang ="eng> Content </html>

The html element has a lang attribute which defines the language of the Document.

<!DOCTYPE html>
<html lang ="eng">
    <head>
        <title>HTML Attributes</title>
    </head>
    <body>
        <p>Attributes</p>
    </body>
</html>

If the value has double quotes, use single quotes as delimiters and vice versa.

<OpeningTag attribute='some "quoted" value'> Content </ClosingTag>

<OpeningTag attribute="some 'quoted' value"> Content </ClosingTag>
