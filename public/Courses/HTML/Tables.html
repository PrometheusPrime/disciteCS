Tables

Tables are used to present Tabular Data, they are defined with the <table> tag. Tables are divided into table rows with the <tr> tag. Table rows are divided into table data with the <td> tag. A table row can also be divided into table headings with the <th> tag.

<table>
  <tr>
    <th>Country</th>
    <th>Independence</th>
  </tr>
  <tr>
    <td>Angola</td>
    <td>11/11/1975</td>
  </tr>
  <tr>
    <td>Botswana</td>
    <td>30/09/1966</td>
  </tr>
  <tr>
    <td>Zambia</td>
    <td>24/10/1964</td>
  </tr>
  <tr>
    <td>Zimbabwe</td>
    <td>18/04/1980</td>
  </tr>
</table>

Border

Add a border with the border attribute. The size of the border is defined in pixels.

<table border="2px">
  <tr>
    <th>Country</th>
    <th>Independence</th>
  </tr>
  <tr>
    <td>Angola</td>
    <td></td>
  </tr>
  <tr>
    <td>Botswana</td>
    <td>30/09/1966</td>
  </tr>
  <tr>
    <td>Zambia</td>
    <td>24/10/1964</td>
  </tr>
  <tr>
    <td>Zimbabwe</td>
    <td>18/04/1980</td>
  </tr>
</table>

Width and Height

By default a table's width and height depends on the data it contains. You can define custom widths and heights with the width and height attributes. Values can be in pixels or percentages.

<table border="2px" width="100%" height="50%">
  <tr>
    <th>Country</th>
    <th>Independence</th>
  </tr>
  <tr>
    <td>Angola</td>
    <td></td>
  </tr>
  <tr>
    <td>Botswana</td>
    <td>30/09/1966</td>
  </tr>
  <tr>
    <td>Zambia</td>
    <td>24/10/1964</td>
  </tr>
  <tr>
    <td>Zimbabwe</td>
    <td>18/04/1980</td>
  </tr>
</table>
You can also change the width and height of a table row.
