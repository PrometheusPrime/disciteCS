Working With Text

Headings

HTML has 6 levels of headings (h1-h6),h1 is the most important(largest) heading and h6 the least important(smallest). 

<!DOCTYPE html>
<html>
 <head>
   <title>Hello HTML</title>
 </head>
 <body>
   <h1>Hello There !!</h1>
   <h2>Hello There !!</h2>
   <h3>Hello There !!</h3>
   <h4>Hello There !!</h4>
   <h5>Hello There !!</h5>
   <h6>Hello There !!</h6>
 </body>
</html>

Quotes

<q> create inline quotation

WWF <q>the world's leading conservation organization</q>

<blockquote> create a block quote

<blockquote>Lorem ipsum dolor sit amet consectetur adipisicing elit. Ea, quaerat. Rerum est tenetur provident quidem perferendis possimus? Error itaque, soluta molestiae quibusdam ratione hic saepe!</blockquote>

<strong>Important text

<small> Small text
<big> big text
<ins> Inserted text
<del>Deleted text
