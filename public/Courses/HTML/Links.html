Hyperlink

HTML links are defined with the <a> tag. The href attribute defines the link's address.The text between the <a> tags will be displayed. 

<a href ="url">text</a>

The target attribute specifies where to open the linked document
_self Opens the link document in the same frame as it was clicked. This is the default.
_blank Opens the link in a new window or tab
_parent Opens the link in the parent frame
_top Opens the linked document in the full body of the window
frame Opens the linked document in a named frame

<a href ="url" target ="_blank"> text </a>
