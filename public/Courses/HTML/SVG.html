SVG

SVG (Scalable Vector Graphics) is used to define vector-based graphics for the Web in XML format. SVG is not pixel based, so SVG graphics can be scaled without loss of quality. An SVG image can be added to an html document with the image tag. 

<img src="image.svg" height="" width = "" alt = "">
</img>

The <svg> element is used to directly draw shapes. The width and height attributes are required. Since SVG is based on XML, all elements must be properly closed. 

<svg height="" width = "">
</svg>

Circle

The <circle> element is used to draw a circle, cx and cy attributes define the X and Y coordinates of the center of the circle. If they are not specified, the circle's center is set to (0, 0).The r attribute defines the radius of the circle. The fill attribute refers to the color inside the circle. stroke and stroke-width attributes control how the outline of a shape appears.

<svg width = "" height="">
  <circle>
  </circle>
</svg>

Ellipse

The <ellipse> element is used to draw an ellipse. An ellipse is related to a circle. The difference is that an ellipse has an x and a y radius. The rx attribute defines the horizontal radius. The ry attribute defines the vertical radius.

<svg></svg>

Line

The <line> element is used to draw a line,The x1 attribute defines the start of the line on the x-axis. The y1 attribute defines the start of the line on the y-axis. The x2 attribute defines the end of the line on the x-axis. The y2 attribute defines the end of the line on the y-axis. 

<svg></svg>

Path

The <path> element is used to draw a path,The following commands are available for path data:

  • A = elliptical Arc
  • C = curveto
  • H = horizontal lineto
  • L = lineto
  • M = moveto
  • Q = quadratic Bézier curve
  • S = smooth curveto
  • T = smooth quadratic Bézier curveto
  • V = vertical lineto
  • Z = closepath
    • All of the commands can also be expressed with lower letters. Capital letters means absolutely positioned, lowercases means relatively positioned.

Rectangle

The <rect> element is used to draw a Rectangle,The width and height attributes of the <rect> element define the height and the width of the rectangle. The x attribute defines the position of the rectangle from the left margin. The y attribute defines the position of the rectangle from the top margin. The style attribute is used to define CSS properties for the rectangle. The CSS fill property defines the fill color of the rectangle. The CSS fill-opacity property defines the opacity of the fill color (from 0 to 1). The CSS stroke-width property defines the width of the border of the rectangle. The CSS stroke property defines the color of the border of the rectangle. The CSS stroke-opacity property defines the opacity of the stroke color (from 0 to 1). 

<svg></svg>
