Lists

There are three kinds of lists in HTML.

Ordered List

The <ol> tag defines an Ordered List. It is used when the order of the items matters. Each list item is placed between the <li> tags

<ol>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ol>
Use the start attribute to change where the list starts counting from. 
<ol start="6">
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ol>

The type of the list item marker can be changed with the type attribute. It has the default value of 1 (list items will be numbered with numbers), other values include A(list items will be numbered with uppercase letters), a(list items will be numbered with lowercase letters), i(list items will be numbered with lowercase roman numbers) and I(list items will be numbered with uppercase roman numbers) 

<ol type="I">
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ol>

Unordered List

The <ul> tag defines an Unordered List. It is used when the order of the items doesn't matters.

<ul>
  <li>Java</li>
  <li>Python</li>
  <li>C++</li>
</ul>

The style of the list item marker can be changed with the list-style-type property. It has the default value of disc, other values include none (the list items will not be marked), square and circle 

<ul style="list-style-type:square;">
  <li>Java</li>
  <li>Python</li>
  <li>C++</li>
</ul>

You can add an image as a list item marker with the list-style-image property.

<ul style="list-style-image: url('images/HTML5_Logo.svg')">
  <li>Java</li>
  <li>Python</li>
  <li>C++</li>
</ul>

Description list

A description list is a list of terms, with their descriptions. <dl> tag defines the description list, <dt> description term defines the term and <dd> description data that describes each term.

<dl>
  <dt>HTML</dt>
    <dd>Markup Language used to describe the content of a Web page and how it's structured.</dd>
  <dt>Python</dt>
    <dd>Python is a High-Level, General purpose, Interpreted programming language.</dd>
  <dt>CSS</dt>
    <dd>Style stylesheet used to control the way an HTML Document is presented or rendered.</dd>
</dl>

Nested Lists

Lists can be nested, the nested list element must be within a list item element.

<ul>
  <li>Web Development
    <ol>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ol>
  </li>
  <li>Android App Development
    <ol>
      <li>Java or Kotlin</li>
    </ol>
  </li>
</ul>
