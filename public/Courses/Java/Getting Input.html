Getting Input

The Scanner class in the java.util package is used to get input from the user. Import the Scanner class with the import keyword and then create an object of the class. Call the method of the data type you want to read on the object. To use the input from the user, you'll have to assign it to a variable.

There are different methods that read different data types :
  • nextByte()reads input of Byte data type.
  • nextShort()reads input of Short data type.
  • nextInt()reads input of Integer data type.
  • nextLong()reads input of Long data type.
  • nextFloat()reads input of Float data type.
  • nextDouble()reads input of Double data type.
  • nextBoolean()reads input of Boolean data type.
  • nextLine()reads input of String data type.
//outside the class
import java.util.Scanner;

//in the main method
Scanner inputScanner = new Scanner(System.in);

System.out.print("What is your Name ? ");

String name = inputScanner.nextLine();
inputScanner.close();

System.out.println("Hello, "+ name);

If the input is not of the data type expected, you'll get an Exception.

Alternatively, we can use the JOptionPane class to get string input.

//outside the class
import javax.swing.JOptionPane;

//in the main method
String name = JOptionPane.showInputDialog("Enter your name");
String age = JOptionPane.showInputDialog("Enter your age");
System.out.println("Your name is " + name + " and you're " + age +" years old");
