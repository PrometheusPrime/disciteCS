Variables and Data Types

Variables

A Variable is a container for storing data. Java is statically typed.

An variable name(identifier) can only start with a letter, $ or an underscore and can not be a reserved keyword.

To Create a Variable in Java , you'll have to write Data Type, Variable name = and then the Value; The Equals (=) sign is used to assign a value to a variable.

short age = 18 ;

You can also declare a variable and assign it a value later:

String name ;
name = "Jane";
The value of a variable can be changed as many times as needed, but the type of that value must equal the type that variable can store. 
short age = 18;
age = 19;
age = 22;
age = 30;

Multiple variables can be declared on a single line if they are of the same type.

float weight, height;

Us the final keyword to define a constant 

final String NAME = "John Doe";

You cannot use a variable before it is declared or outside it's scope (block of code it's declared in).

Variables declared in a class are given default values, therefore they can be used without being initialised.

Variables declared in a block (method,loop etc) don't have default values, they can not be used before they are initialised.

Data Types

Primitive Data Types

Primitive Data Types are predefined types of data. They are the most basic data types provided by a Programming Language. Java support the following primitive data types :

Byte

A byte can store integers from -128 to 127 (8-bit) and are 1 byte in size. 

byte daysInAWeek = 7;

Short

A short can store integers from -32768 to 32767 (16-bit). A short is 2 bytes in size. 

# integers
short studentAge = 16;

Integer

An int can hold a value ranging from -2147483648 to 2147483647 (32-bit). An int is 4 bytes in size. 

int secondsInAYear = 31536000;

Long

A long can hold a integer values from -9223372036854775808 to 9223372036854775807 (64-bit). A long is 8 bytes in size. The letter l (or L ) must be appended to the end of the value. 

long milliSecondsInAYear = 31536000000l;

Float

A float is 32-bit floating point numbers. A float is 4 bytes in size and have up to 6 decimal places precision. To create a float you must append the value with an f (or F), otherwise you'll get an error because it will be interpreted as a double. 

float  height = 6.4f ;
alternitively cast it to float. 
float  height = (float) 6.4;

Double

A double is a 64-bit double-precision numbers and is 8 bytes in size. Up to 15 decimal places precision. 

# integers
double pi = 3.141592653589793;

Character

A char is a single 16-bit Unicode character ranging from '\u0000' or 0 to '\uffff' or 65,535. A char is 2 bytes in size. Char values are surrounded by single quotes (''). 

char grade = 'A';

Boolean

A boolean can only take the values true or false. A boolean is 1 byte in size. 

boolean isRaining = false ;

Wrapper Classes

- Integer ,Byte ,Double, Float,Long,Char Non-primitive data types are called reference types because they refer to objects.

Type Conversion

A value can be converted from one data type to another. In Java, there are two types of casting:

Implicit Conversion - converting automatically from a small size data type to a large data type.

short years = 25;
int age = years;

Explicit Conversion - manually converting a larger type to a smaller size type.

int pay = 20000;
short salary = (short) pay;
Make sure the type you're converting to can hold the value, and even if it can some information may be lost. 
double pi = 3.141592653589793;
int num = (int)pi;
