Strings

String are immutable sequence (or Arrays) of characters. Unlike chars, strings are delimited by Double quotes(""). A string in Java is not a primitive type, it is an objects.

String greet = "Hello, World";
System.out.println(greet);
You can also create a string object with the new keyword. 
String sayHello = new String("Hello, World");

The addition operator is used to join strings

String firstName = "John" ;
String lastName = "Wick";
String fullName = firstName + " " + lastName;
System.out.println(fullName);
You can concatenate a string and a number, the number will be implicitly converted to a string.

String Methods

String phrase = "Whosoever Holds This Hammer, If They Be Worthy,Shall Posses The Power Of THOR";

System.out.println(phrase.isEmpty());

Use the length() method to check the length (number of characters) of a string.

System.out.println(phrase.length());
Get a character at a specified index. 
phrase.charAt(70);
The indexOf() method returns the index of a character or start index of a substring. It returns -1 if a character or string is not found. 
phrase.indexOf('X');
phrase.indexOf("THOR");
The contains() method checks if a string is a substring of the string, it returns a boolean. 
phrase.contains("Worth");
The toUpperCase() method return a new string with all the letters converted to uppercase. 
phrase.toUpperCase();
The toLowerCase() method return a new string with all the letters converted to lowercase. 
phrase.toLowerCase();
Use the equals() to compare strings instead of the equality operator. It returns true if the strings are exactly the same. 
String anotherStr = phrase.toUpperCase();
phrase.equals(anotherStr);
Even if they are the same, the equals() method will return false if the strings you're comparing are in different Cases. 
String anotherStr = phrase.toUpperCase();
phrase.equalsIgnoreCase(anotherStr)

String anotherStr = phrase.toUpperCase();
phrase.compareTo(anotherStr)

Replace a character or substring in the string with the replace() method. It returns a new string.

String anotherStr = phrase.replace("Hammer", "Axe");

The substring() returns a substring from specified index upto the end of a string. You can pass an optional end index as a second arguement. 

String character = phrase.substring(73);
String weapon = phrase.substring(21,27);

The trim() removes whitespace from the beginning and / the end of a string.

phrase.trim()

String myStr = "One Two Three Four Five Six Seven Eight Nine Ten";
String[] myArray = myStr.split();

String name = "Jane Doe";
String gender = "Female";
int age = 20;
String personalIfo = String.format("%s, %s, %d years old",name,gender,age);
System.out.println(personalIfo);

String Classes

StringBuilder

use StringBuilder if you're going to make changes to a string and don't care about thread safety. 
StringBuilder name = new StringBuilder("John");
System.out.println(name);

name.append(" Smith");
System.out.println(name);
name.replace(5, 10, "Wick");
System.out.println(name);

name.insert(index, something);
System.out.println(name);
The capacity() method returns the number of characters(including already written characters) that can be stored in a StringBuilder string. 
System.out.println(name.capacity());

name.delete()
System.out.println(name);
