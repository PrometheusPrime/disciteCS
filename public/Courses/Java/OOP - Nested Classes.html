Nested Classes

A Class can be defined in another class. Unlike regular classes, nested classes can be private or protected. A nested class can be either static or non-static. A non-static nested class is also known as Inner Class.

Inner Class

An inner class can access all members of the enclosing class.

class Outer {

    protected class Inner {
        void sayHello() {
            System.out.println("Hello world!");
        }
    }
}

To create an object of the inner class, we have to instantiate the outer class first. 

public class Main {

  public static void main(String[] args) {
  
    Outer outerObj = new Outer();

    Outer.Inner innerObj = outerObj.new Inner();

    innerObj.sayHello();
  }

}
We can use the following syntax to easily instantiate an inner class. 
Outer.Inner innerObj = new Outer().new Inner();

innerObj.sayHello();

If a nested class is declared private, it can not be accessed outside it's enclosing class.

Static Nested Class

A static nested class can not access non-static members of the outer class.

class Outer{

    protected static class Nested{
        void sayHello() {
            System.out.println("Hello world!");
        }
    }
}

A static nested class can be accessed without instantiating the enclosing class.

Outer.Nested nestedObj = new Outer.Nested();
nestedObj.sayHello();
