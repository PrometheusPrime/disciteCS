HashMap

A HashMap is a collection of objects in key - value pairs. Import the HashMap class from the java.util package.

HashMap<String,String> independenceDay = new HashMap<>();

The put(key,value) method adds a key - value entry to a HashMap.

independenceDay.put("Angola","11/11/1975");
independenceDay.put("Botswana","30/09/1966");
independenceDay.put("Namibia", "21/3/1990");
independenceDay.put("Malawi", "6/7/1964");
independenceDay.put("Zambia","18/04/1980");
independenceDay.put("Zimbabwe","18/04/1980");

System.out.println(independenceDay);

The put method modifiesputIfAbsent()

independenceDay.putIfAbsent("Zambia", "18/04/2000");

Use the get(key) method to get a value.

System.out.println(independenceDay.get("Zambia"));

Use the replace() method to change a value. Pass into the method the key and the new value.

independenceDay.replace("Zambia", "24/10/1964");
System.out.println(independenceDay);

Use the containsKey() and containsValue() methods to check whether a key or value is present in a HashMap respectively

System.out.println(independenceDay.containsKey("Angola"));
System.out.println(independenceDay.containsValue("El Aaiun"));

Check the size of a HashMap with the size() method.

System.out.println(independenceDay.size());

keySet() returns a collection with all the keys contained in the map.

System.out.println(independenceDay.keySet());

values() returns a collection with all the values contained in the map.

System.out.println(independenceDay.values());

The remove() method delete a key - value pair from the HashMap.

independenceDay.remove("Namibia");

System.out.println(independenceDay);

The clear() methods deletes all key - value pairs in a HashMap, returns an empty HashMap.

independenceDay.clear();

Iterate through a HashMap

Iterate through a HashMap using the forEach() method.

independenceDay.forEach(
    (country, day) -> System.out.println(country + " : " + day)
);

Iterate through a HashMap's keys or values using an enhanced for loop.

for (String key : independenceDay.keySet()) {
    System.out.println(key);
}

for (String value : independenceDay.values()) {
    System.out.println(value);
}
