HashSet

A HashSet is a collection of unique values. Import the HashSet class from the java.util package.

HashSet<String> instrumentSet = new HashSet<String>();

The add() method adds a value a HashSet.

instrumentSet.add("Saxophone");
instrumentSet.add("Organ");
instrumentSet.add("Ocarina");
instrumentSet.add("Acoustic Guitar");
instrumentSet.add("Trumpet");
instrumentSet.add("Triangle");
instrumentSet.add("Shakers");
instrumentSet.add("Vibraphone");
instrumentSet.add("Bagpipes");
instrumentSet.add("Keyboard");

System.out.println(instrumentSet);

Check whether a HashSet contains a certain value with the contains() method.

System.out.println(instrumentSet.contains("Vibraphone"));

Remove a value from the HashSet.

instrumentSet.remove("Keyboard");

System.out.println(instrumentSet);

Check the size of a HashSet with the size() method.

System.out.println(instrumentSet.size());

The clear() methods deletes all values in a HashSet, returns an empty HashSet.

instrumentSet.clear();

Iterate through a HashSet

Iterate through a HashSet using the forEach() method.

instrumentSet.forEach(
    instrument -> System.out.println(instrument)
);

Iterate through a HashMap's keys or values using an enhanced for loop.

for (String instrument : instrumentSet) {
    System.out.println(instrument);
}
