LinkedList

A LinkedList is a collection of objects of the same type. Each object it stored in a container which is linked to the cointainer holding the next and or previous object. Import the LinkedList class from the java.util package.

import java.util.LinkedList;
LinkedList<String> flowerList = new LinkedList<String>();

Add elements to a LinkedList with the add() method, it adds element to the end of an LinkedList.

flowerList.add("Snap Dragon");
flowerList.add("Butterfly Bush");
flowerList.add("Salvia");
flowerList.add("Poppy");
flowerList.add("Day Lily");
flowerList.add("Narcissus");
flowerList.add("Delphinium");

The addFirst() method adds element to the front of a LinkedList.

flowerList.addFirst("Clematis");
flowerList.addLast("Bleeding Heart");

The addLast() method adds element to the end of a LinkedList.

flowerList.addFirst("Clematis");
flowerList.addLast("Bleeding Heart");

To add multiple elements use the addAll() method, pass a list.

flowerList.addAll(Arrays.asList("Grape Hyacinth","Scilla","Kerria"));

Get the size of a Linked list

System.out.println(flowerList.size());

The indexOf() method returns the index of an element if it is in the Linked, else returns -1.

flowerList.indexOf("Bleeding Heart");

Check if a Linked list contains a certain valuecontains()

flowerList.contains("Snap Dragon");

Get the item at an index

System.out.println(flowerList.get(5));

The getFirst() returns the first item in a LinkedList

System.out.println(flowerList.getFirst());

The getLast() returns the last item in a LinkedList

System.out.println(flowerList.getLast());

Replace a value in a Linked list with the set(index, newValue) method.

flowerList.set(2, "Cosmos");

You can sort a Linked list using the sort method from Collections class in the java.util package.

//top of the file, outside of the class
import java.util.Collections;

//in main
Collections.sort(flowerList);

The remove() method removes an element from an Linked list, you can either pass into the method the element or it's index.
remove(element), removes the first instance of the element if there are duplicates.

flowerList.remove(3);
//or
flowerList.remove("Day Lily");
The removeFirst() method removes the first element while the removeLast() method removes the last element from an Linked list.

flowerList.removeFirst();
flowerList.removeLast();

Clear all items in a Linked list clear()

flowerList.clear();

Loop through an Linked List

for

for(int i = 0; i < flowerList.size(); i++){

    System.out.println(flowerList.get(i));
}

for(String flower : flowerList){
