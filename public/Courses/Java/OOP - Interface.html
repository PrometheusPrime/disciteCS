Interface

Another way of achieving Abstraction in Java is with interfaces. An interface is a fully abstract type.Interfaces help us achieve multiple inheritance.

interface Animal{

}

An interface cannot be instantiated, it is implemented by a class. Use the implements keyword to implement an interface.

interface Animal{
    
}
Abstract class Bird implements Animal{

}
class Person extends Human implements Animal{

}

A class can implement any number of interfaces but can only inherit from one class. A class that implements an interface must provide implementation for all abstract methods in the interface, unless it is an abstract class.

interface Animal{
    
}
Abstract class Bird implements Animal{

}
class Person extends Human implements Animal{

}
