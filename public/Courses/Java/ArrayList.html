ArrayList

An arrayList is just like an array ,except it doesn't have a fixed length and can only store objects. Import the ArrayList class from the java.util package.

//top of the file, outside of the class
import java.util.ArrayList;

//in main
ArrayList<String> bucketList = new ArrayList<String>();

Add elements to an arrayList with theadd() method, it adds element to the end of an ArrayList.

bucketList.add("Learn to Surf");
bucketList.add("Go Skydiving");
bucketList.add("Learn to play a Piano");
bucketList.add("Ride a horse");
bucketList.add("Start a Charity");
You can specify where to add an element, pass the index as the first arguement. 
bucketList.add(0,"Reconnect with an Old Friend");
bucketList.add(1,"Meet Someone Famous");
bucketList.add(2,"Go on a blind date");
System.out.println(bucketList);

The get() method returns an element from an ArrayList, pass into the method the index of the element.

System.out.println(bucketList.get(3));

The size() method returns the size or length of an array list

System.out.println(bucketList.size());

The set() method replaces an element at the specified position in the Arraylist with the specified value.

bucketList.set(3, "Own Horses");
System.out.println(bucketList);

Easily sort an ArrayList using the sort method from Collections class in the java.util package.

//top of the file, outside of the class
import java.util.Collections;

//in main
Collections.sort(bucketList);
System.out.println(bucketList);

The remove() method removes an element from an Arraylist, you can either pass into the method the element or it's index.
remove(element) removes the first instance of the element if there are duplicates.

bucketList.remove("Go on a blind date");
System.out.println(bucketList);

The clear() methods delete all elements in an ArrayList

bucketList.clear();

The isEmpty() method checks whether an array ArrayList is empty, returns a boolean value.

System.out.println(bucketList.isEmpty());

Loop through an ArrayList

for

for (int i = 0; i < bucketList.size(); i++){

    System.out.println(bucketList.get(i));
}

for each

for (String item : bucketList) {
    System.out.println(item);
}
