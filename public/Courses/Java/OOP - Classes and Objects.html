Class

A Class is a template for creating objects, it defines how an object will be. Java is an Object Oriented programming language. To create a class use the class keyword followed by the name of the class.

public class Person{
    String name;
    int age;
    String gender;

    String personalInfo() {
    return "Name : " + this.name + "\n" + "Gender : " + this.gender + "\n" + "Age : " + this.age;
  }
}

A file.java can have multiple Classes, but must contain one that matches the file name. Our Person class can be written in Main.java but outside of the main class. When compiled, separate .class files for each class will be generated.

Alternativelly, we can define our Person class in Person.java file, in the same folder as Main.java.

Object

Create an instance(object) of a class with the new keyword. You can create any number of objects from a class.

//in Main class, main method
Person jane = new Person();

Instance Variables

Instance Variables are variables declared in a class, their values are unique to each instance of a class. in our Person class name, age and gender are all instance variables.

Members (variables and methods) are accessed using the dot notation(.). 
  
jane.name = "Jane Smith";
jane.age = 34;
jane.gender = "Female";
System.out.println(jane.personalInfo());

Constructors

A Constructor is a method on a Class that is called when a new instance of the class is created. Constructors must have the public access modifier and same name as the class. 

public class Person{
    String name;
    int age;
    String gender;

    String personalInfo() {
    return "Name : " + this.name + "\n" + "Gender : " + this.gender + "\n" + "Age : " + this.age;
  }
    //Constructor
    public Person(String name,int age,String gender){
        this.name = name;
        this.age = age;
        this.gender = gender;

    }
            
}
The this keyword refers to the object the method is called on or whose instance variable being accessed.

Since we've overiden the default constructor, every time we create an instance, we have to pass arguments to the constructor.

//in Main class, main method
Person john = new Person("John Banda",58,"Male");
We can add the default constructor back, so that we can also just create an object and then initialise it instance variables later. In fact you can overload the constructor method as many times as you need. 
public class Person {
  String name;
  int age;
  String gender;

  String personalInfo() {
    return "Name : " + this.name + "\n" + "Gender : " + this.gender + "\n" + "Age : " + this.age;
  }

  // Default Constructor
  public Person() {

  }

  // Secondary Constructor
  public Person(String name, int age, String gender) {
    this.name = name;
    this.age = age;
    this.gender = gender;

  }

}

//uses the Default constructor
Person tony = new Person();
//uses the Secondary constructor
Person mike = new Person("Micheal Myers",40,"Male");

Static Variables

A static or class variable belong to the class, not to it's instances. There's one copy of this variable per Class,no matter how many times the class has been instantiated. All instances of the class share the same static variable.

We've modified the Person class so that we can track the number of Person objects we've created. The numberOfPeople variable is declared static because we don't want it to change on every instance of the class. The variable is being updated in the constructors. 
public class Person {
  static int numberOfPeople;
  String name;
  int age;
  String gender;

  String personalInfo() {
    return "Name : " + this.name + "\n" + "Gender : " + this.gender + "\n" + "Age : " + this.age;
  }

  // Default Constructor
  public Person() {
    numberOfPeople++;
  }

  // Secondary Constructor
  public Person(String name, int age, String gender) {
    this.name = name;
    this.age = age;
    this.gender = gender;
    numberOfPeople++;

  }

}
A static variable can be accessed on the object or the class itself. 
Person jane = new Person();
Person john = new Person("John Banda",58,"Male");
Person tony = new Person();
Person mike = new Person("Micheal Myers",40,"Male");

System.out.println(jane.numberOfPeople);
System.out.println(Person.numberOfPeople);
