Loops

Loops are used to repeatedly execute a block of code.

While Loop

The while loop is used to continually execute a block of statement(s) while a particular condition is true.

while(condition/expression){
    //TODO
}
The expression will be checked, if it evaluates to true; the statements in the while block will be executed.The while statement will continue checking the expression and executing its block of statements until the expression evaluates to false 
int num = 0;
while (num < 15){
    System.out.println(num);
}
If the condition does not evaluate to false, the Loop runs indefinitely (infinite Loop). Press Ctrl + C to stop execution. A condition can be updated inside the Loop so that it eventually evaluates to false. 
int num = 0;
while (num < 15){
    System.out.println(num);
    num++;
}
explicitly create an infinte loop 
int num = 0;
while (true){
    System.out.println(num);
    num++;
}

Do... While Loop

The do... while loop works the same way as the while loop, except the condition is checked after the block is executed. This means it's block is executed at least once, even is the expression is false.

do{
    System.out.println("1 is greater than 5");
} while (1 > 5);

For Loop

The for loop

for (initialisation; condition; updatation){
 //TODO
}
first the iteration variable will be initialised,then the expression is checked, if it evaluates to true the body of the loop is executed and then the iteration variable is updated. The iteration variable is created once. 
for (int num = 0;num <= 10;num++){
    System.out.println(num);
}

Jump Statements

continue skips the rest of the statements in a loop, restart execution from the begining.

for (int num = 0;num <= 10;num++){
    if (num == 6){
        continue;
    }
    System.out.println(num);
}

break exits the loop and execution goes to the first statement after the loop.

for (int num = 0;num <= 10;num++){
    if (num == 6){
        break;
  }
    System.out.println(num);
}
