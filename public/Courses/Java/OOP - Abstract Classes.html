Abstract Class

An abstract class is created with the abstract keyword. Abstract classes are similar to ordinary classes except they can not be instantiated and can contain abstract methods.

Abstract methods are also defined with the abstract keyword. Abstract methods have no implementation, they must be implementated by the sub-class. 
abstract class MusicalInstrument {
    String nameOfInstrument;

    // Getter and Setters
    public String getNameOfInstrument() {
        return nameOfInstrument;
    }

    public void setNameOfInstrument(String nameOfInstrument) {
        this.nameOfInstrument = nameOfInstrument;
    }

    // Non-abstract method
    String play() {
        return nameOfInstrument + " is making music";

    }

    // Abstract method
    abstract String info();

    // Constructors
    public MusicalInstrument() {

    }

    public MusicalInstrument(String nameOfInstrument) {
        setNameOfInstrument(nameOfInstrument);
    }
}

Abstract classes can not be instantiated, they are inherited. A class that inherits from an abstract class with abstract methods,must provide implementation for all the abstract methods. If not then it should also be declared abstract.

class Idiophone extends MusicalInstrument {

    public Idiophone() {
    }

    public Idiophone(String nameOfInstrument) {
        super(nameOfInstrument);
    }

    String info() {

        return nameOfInstrument + " produce sound by the vibration of the instrument's body";

    }

}

class Membranophone extends MusicalInstrument {
    public Membranophone() {
    }

    public Membranophone(String nameOfInstrument) {
        super(nameOfInstrument);
    }

    String info() {

        return nameOfInstrument + " produce sound by the vibration of a tightly stretched Membrane";
    }

}

Idiophone thumbPiano = new Idiophone("African Thumb Piano");
Membranophone drum = new Membranophone("Drums");

If an abstract class' sub-class is abstract, it doesn't have to override the abstract methods.

abstract class PercussionInstrument extends MusicalInstrument {
 
}

Abstract classes are used to achieve Abstraction, hiding unnessary details and only showing what is needed.
