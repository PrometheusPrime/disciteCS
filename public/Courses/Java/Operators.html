Operators

Arithmetic Operators

Arithematic Operators are used in basic arithematic calculations.

Addition Operator (+) : adds the operands.

int x = 6;
int y = 8;
System.out.println(x + y);

Subtraction Operator (-) : subtracts one operand from another.

int x = 5;
double y = 2.5;
System.out.println(x - y);

Multiplication Operator (*) : multiplies the operands.

int x = 9;
int y = 25;
System.out.println(x * y);

Division Operator (/) : divides the operands. Dividing integers returns an integer, any remainder will be discarded.

int x = 26;
int y = 2;
System.out.println(x / y);

Modulus or Remainder Operator (%) : returns the remainder of dividing the given operands.

int x = 5;
int y = 3;
System.out.println(x % y);

Unary Operators

Unary Operator perform various operations on one operand.

Increment Operator (++) increments a variable by 1 

int x = 5;
x ++;
System.out.println(x);

Decrement Operator (--) decrements a variable by 1

int x = 5;
x --;
System.out.println(x);
prefix vs postfix when doing an operation and incrementing (or decrementing), first the operation is done and then the variable is incremented (or decremented) 
int x = 5;
int y = 5;
System.out.println(++x); //first increment then print the variable.
System.out.println(y++); //first print the variable then increment it.

Compound Assignment Operators

Operator (+=) 

int x = 8;
x+= 5;  //same as x = x + 5
System.out.println(x);

Operator (-=) 

int x = 8;
x-= 5;  //same as x = x - 5
System.out.println(x);

Operator (*=) 

int x = 8;
x*= 5;  //same as x = x * 5
System.out.println(x);

Operator (/=) 

int x = 8;
x/= 5;  //same as x = x / 5
System.out.println(x);

Operator (%=) 

int x = 8;
x%= 5;  //same as x = x % 5
System.out.println(x);

Comparison Operators

Comparison Operators are used to compare given Variables, they return a Boolean value.

Greater Than Operator (>) 

int x = 8;
int y = 5;
System.out.println(x > y);

Less Than Operator (<) 

int x = 8;
int y = 5;
System.out.println(x < y);

Equality Operator (==) : compares whether the Operands are Equal.

int x = 8;
int y = 5;
System.out.println(x == y);

Greater Than or Equal To Operator (>=) 

int x = 8;
int y = 5;
System.out.println(x >= y);

Less Than or Equal To Operator (<=) 

int x = 8;
int y = 5;
System.out.println(x <= y);

Inequality Operator (!=) : checks whether the Operands are Not Equal.Returns True if they are not equal.

int x = 8;
int y = 5;
System.out.println(x != y);

Logical Operator

Logical Operators are used to chain Boolean Variables, they return a boolean value.

And Operators (&&) : Returns true if all the variables are true.

boolean isRaining = true;
boolean isSunny = false;
System.out.println(isRaining && isSunny);

Or Operator (||) : Returns True if one of the conditions is true.

boolean isRaining = true;
boolean isSunny = false;
System.out.println(isRaining || isSunny);

Not Operator (!) : inverts a boolean value . If it was True , it becomes False.

boolean isRaining = true;
System.out.println(!isRaining);

Logical Operators can be chained and are evaluated according to Their order of precedence.
