public class Person {
  private static int numberOfPeople;
  private String name;
  private int age;
  private String gender;

  String personalInfo() {
    return "Name : " + this.getName() + "\n" + "Gender : " + this.getGender() + "\n" + "Age : " + this.getAge();
  }

  // Setters
  public void setName(String name) {
    this.name = name;
  }

  public void setAge(int age) {
    if (age < 0) {
      this.age = 0;
    } else {
      this.age = age;
    }
  }

  public void setGender(String gender) {
    this.gender = gender;
  }

  // Getters
  public String getName() {
    return name;
  }

  public int getAge() {
    return age;
  }

  public String getGender() {
    return gender;
  }

  public static int getNumberOfPeople() {
    return numberOfPeople;
  }

  // Constructors
  public Person() {
    numberOfPeople++;

  }

  public Person(String name, int age, String gender) {
    setName(name);
    setAge(age);
    setGender(gender);
    numberOfPeople++;

  }

}

All the properties in the Super class,as long as they are not private, will be available in the Derived class. 
public class Employee extends Person {
  private String position;
  private double salary;

  // Getters
  public String getPosition() {
    return position;
  }

  public double getSalary() {
    return salary;
  }

  // Setters
  public void setPosition(String position) {
    this.position = position;
  }

  public void setSalary(double salary) {
    if (salary < 0.0) {
      this.salary = 0.0;
    } else {
      this.salary = salary;
    }
  }

  // Constructors
  public Employee() {

  }

  public Employee(String name, int age, String gender, String position, double salary) {
    super(name, age, gender);
    setPosition(position);
    setSalary(salary);
  }
}

Method Overiding

We can provide a different implementation of an inherited method in the Subclass, this is known as Method Overiding. static and final methods can not be overriden


  
We overriden the personalInfo() method so that it shows the position and salary properties.

  
public class Employee extends Person {
  private String position;
  private double salary;

  String personalInfo() {
    return "Name : " + this.getName() + "\n" + "Gender : " + this.getGender() + "\n" + "Age : " + this.getAge() + "\n"
        + "Position : " + this.getPosition() + "\n" + "Salary : " + this.getSalary();
  }

  // Getters
  public String getPosition() {
    return position;
  }

  public double getSalary() {
    return salary;
  }

  // Setters
  public void setPosition(String position) {
    this.position = position;
  }

  public void setSalary(double salary) {
    if (salary < 0.0) {
      this.salary = 0.0;
    } else {
      this.salary = salary;
    }
  }

  // Constructors
  public Employee() {

  }

  public Employee(String name, int age, String gender, String position, double salary) {
    super(name, age, gender);
    setPosition(position);
    setSalary(salary);
  }
}
