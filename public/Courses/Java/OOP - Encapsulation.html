Encapsulation

It it not recommended to directly access and modify fields in a class. Make fields private and use setter and getter Methods to modify and access them. We are going to modify our person class with the following getters and setter.

// Setters
public void setName(String name) {
  this.name = name;
}

public void setAge(int age) {
  this.age = age;
}

public void setGender(String gender) {
  this.gender = gender;
}

// Getters
public String getName() {
  return name;
}

public int getAge() {
  return age;
}

public String getGender() {
  return gender;
}

public static int getNumberOfPeople() {
  return numberOfPeople;
}
One of the uses of setter methods is to validate the value being assigned to an instance variable. For example age can not be negative. 
public void setAge(int age) {
    if (age < 0) {
      this.age = 0;
    } else {
      this.age = age;
    }
}
Here is our Person class fully encapsulated. 
public class Person {
  private static int numberOfPeople;
  private String name;
  private int age;
  private String gender;

  String personalInfo() {
    return "Name : " + this.getName() + "\n" + "Gender : " + this.getGender() + "\n" + "Age : " + this.getAge();
  }

  // Setters
  public void setName(String name) {
    this.name = name;
  }

  public void setAge(int age) {
    if (age < 0) {
      this.age = 0;
    } else {
      this.age = age;
    }
  }

  public void setGender(String gender) {
    this.gender = gender;
  }

  // Getters
  public String getName() {
    return name;
  }

  public int getAge() {
    return age;
  }

  public String getGender() {
    return gender;
  }

  public static int getNumberOfPeople() {
    return numberOfPeople;
  }

  // Constructors
  public Person() {
    numberOfPeople++;

  }

  public Person(String name, int age, String gender) {
    setName(name);
    setAge(age);
    setGender(gender);
    numberOfPeople++;

  }

}
Modifying instance variables with setter methods. 
//uses the Default constructor
Person tony = new Person();
tony.setName("Tony Stark");
tony.setGender("Male");
tony.setAge(43);

//uses the Secondary constructor
Person micheal = new Person("Micheal Myers",40,"Male");
micheal.setName("Micheal Mobius");
