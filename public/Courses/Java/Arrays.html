Arrays

Array is an ordered collection of values of the same data type,referred by a single identifier. An array has a Fixed size. the first element is at index( or position) 0.

type[] arrName = new type[numberOfElements];
Declaring an empty array. 
short[] arr = new short[4];

To add an element in an Array, write the name of the array followed by the index( position) in square brackets then equals sign followed by the value.

arr[0] = 3;
arr[1] = 1;
arr[2] = 4;
arr[3] = 2;

An Array in Java cannot be printed directly, first convert it to a string using the .toString() method in the Arrays class.

System.out.println(Arrays.toString(arr));

To access an element in an Array, write the name of the array followed by the index( position) in square brackets.

System.out.println(arr[2]);

Array literal

To create and add elements to an array, place the values in cury bracket, separated by commas.

String[] movies = {
    "The Sixth",
    "The Visit",
    "Split"
 };

Multidimensional Arrays

A multidimensional array is an array containing arrays. All the nested arrays must be of the same type

int[][] numsArray = new int[4][3];
numsArray[0][0] = 2;
numsArray[0][1] = 4;
numsArray[0][2] = 6;
numsArray[1][0] = 3;
numsArray[1][1] = 9;
numsArray[1][2] = 12;
numsArray[2][0] = 4; 
numsArray[2][1] = 16;
numsArray[2][2]= 64;
numsArray[3][0] = 5;
numsArray[3][1] = 25;
numsArray[3][2] = 125;
The first square brackets are for the position of the inner array , the second square brackets are for the position of an element in that array.

Use .deepToString() to convert a multidimensional array to string.

System.out.println(Arrays.deepToString(numsArray));
Multidimensional Array literal 
int[][] numsArray = {
    {2,4,6},
    {3,9,12},
    {4,16,64},
    {5,25,125} 
};

System.out.println(numsArray[1][2]);

Array properties and methods

The length property stores the number of elements in an Array.

System.out.println(movies.length);

The .fill() method in the Arrays class fills available space in an Array with a single value.

int[] evenNums = new int[5];
Arrays.fill(evenNums, 2);
System.out.println(Arrays.toString(evenNums));

Copy an array by using the .copyOf() method, pass the array and number of elements you wish to copy as arguements.

String[] watchList = Arrays.copyOf(movies, 3);

Compare arrays

String[] watchList = Arrays.copyOf(movies, 3);
System.out.println(Arrrays.equals(movies,watchList));

Sort arrays using the .sort() method.

int[] evenNums = { 2, 4, 10, 6, 14, 12, 16, 8 };
Arrays.sort(evenNums);
System.out.println(Arrays.toString(evenNums));
You can specify where to start and stop sorting. 
int[] evenNums = { 2, 4, 10, 6, 14, 12, 16, 8 };
Arrays.sort(evenNums, 2, evenNums.length);
System.out.println(Arrays.toString(evenNums));

Binary Search an array for an element using the binarySearch() method in the Arrays class. It returns the index of value if it is found. If the value is not in the array, a negative number is returned. For this to work, the array must be sorted.

String[] students = {
    "Grace",
    "Charles",
    "Elizabeth",
    "Joyce",
    "Catherine",
    "Moses",
    "Charity",
    "James",
    "Emmanuel"
};

Arrays.sort(students);
System.out.println(Arrays.toString(students));
System.out.println(Arrays.binarySearch(students, "Emmanuel"));
You can specify where to start and stop searching. 
Arrays.binarySearch(students, 3, 5, "Emmanuel");

Loop through Arrays

We can loop through an array using the simple for loop. 

for (int index = 0; index < movies.length; index++) {
    System.out.println(movies[index]);
}

The best looping structure for iterating through a one dimensional array's elements, is the Enhanced for loop.

for(String movie : movies){
    System.out.println(movie);
}
For a multidimensional array , we use nested for loops. 
int[][] numsArray = {
    {2, 4, 8},
    {3, 9, 27},
    {4, 16, 64},
    {5, 25, 125}
};
for (int i = 0; i <= 3; i++) {
    for (int j = 0; j < 3; j++) {

        System.out.print(numsArray[i][j]);
    }
    System.out.println();
}
In our two dimensional array example, the first loop selects an inner array and the second loop transverses through all the elements in that array.
