Conditional Statements

Control Statements are used to run code under certain conditions

The if, else and else if Statements

If statement

The code in the block will be executed if the condition is true. The condition is either a boolean variable or an expression which evaluates to true or false.

//write the code in main method
if(4 < 5){
    System.out.println("4 is less than 5");
}

if statements can be nested inorder to only execute a block if multiple conditions are true.

int age = 22;
if(age > 18){
  if(age <= 25){
      System.out.println("You're between 18 and 25 years of age");
  }
}
Instead of nesting if statements to check multiple conditions,we can combine logical expressions using logical operators. 
int age = 22;
if (age > 18 && age <= 25){
    System.out.println("You're between 18 and 25 years of age");
}

The Else statement

The else statement defines alternate block to execute if the condition in the if statement is false. 

if(4 > 5){
    System.out.println("4 is greater than 5");
}else{
    System.out.println("4 is less than 5");
}

Else if statement

The else if statement is used to check additional conditions.

boolean isRed = false;
boolean isGreen = true;

if (isRed){
    System.out.println("Stop !");
}else if(isGreen){
    System.out.println("Drive");
}else{
    System.out.println("Drive, if it is safe to do");
}
If a Statement's condition is true, that Statement's block will be executed and no other conditions will be evaluated. 
//write the code in main method
int age = 22;
if (age > 18){
    System.out.println("You're older than 18");
}else if(age < 25){
    System.out.println("You're less than 25 years of age");
}else{
    System.out.println("You're either less than 18 and older than 25");
}
The else if is skipped, even though its expression evaluates to true.

Switch

The switch statement is used to do different operations depending on a value of a variable.

String trafficLight = "Yellow";

switch(trafficLight){
case "Red":
    System.out.println("Stop !");
    break;
case "Yellow":
    System.out.println("Drive, with caution");
    break;
default:
    System.out.println("Drive");
}
The default case is executed if non of the cases matched. 
String trafficLight = "Green";

switch(trafficLight){
case "Red":
    System.out.println("Stop !");
    break;
case "Yellow":
    System.out.println("Drive, with caution");
    break;
default:
    System.out.println("Drive");
}
The break statement moves program execution out of the switch statement after a matched case is executed. If there isn't any break statement, all cases after the true case will be executed until a break statement is found. 
//fall  through
int num = 3;
switch(num){
case 4:
    System.out.println("4 !");

case 2:
    System.out.println("2 !");
case 1:
    System.out.println("1 !");
default:
    System.out.println("Blast Off !");
