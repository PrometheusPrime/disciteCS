Margin

The margin property is used to set margin around an element.


    If you provide a single value, that amount of margin is applied to all sides of the element.
    
    
When a two values are given, the first value is set to the and , the second value is set to the and .

    

    
If three values are given,  the first value is set to the and , the second value is set to the and and then third is applied to.

    

    And if four values are given , they are applied in the following order;
    

    
You can also set margin on each side of an element independently.

    
{   
    margin-top:;
    margin-bottom:;
    margin-left:;
    margin-right:;
}


Margin Collapse


Margins between elements will collapse into a single margin. If the margins are of different sizes, the larger of the two will be applied and the smaller margin becomes zero.

Centering an Element


Margin can be used to horizontally center an element in their container, set the left and right margin to auto.



If an element doesn't have a fixed width, it will shrink or expand to fill the available space while remaining centered.
