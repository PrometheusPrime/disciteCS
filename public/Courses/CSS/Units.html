Units

Absolute Units

Absolute units are based on actual physical units, and are ussally the same across devices

Pixel
Based on device pixels. Pixels vary depending on pixel density of the device 
p {
    text-indent: 10px;
}
Milimeters
One millimeters is 1/10th of a centimeter 
@page {
  margin: 5mm;
}
Centimeters
One centimeter is 37.8px 
@page {
  margin: 0.5cm;
}
Inches
One inch is 96px 
h1{
  font-size: 0.7in;
}
Points
One point is 1/72 of an inch 
h1{
  font-size: 24pt;
}
Picas
One pica is 1/6 of an inch 
h2{
  font-size: 3pc;
}

Except pixels, absolute units are mostly suitable for print.

Relative Units

Relative units base their dimensions on something else. They are great for Responsive Web Design.

Percentage
relative to the parent 
.main{
  max-width: 75%
}
Viewport Height
One viewport height is 1% of the Viewport's Height 
body {
  min-height: 100vh;
}
Viewport Width
One viewport width is 1% of the Viewport's Width 
.main{
  width: 75vw;
}
vmin
One vmin is 1% of whichever is smaller, viewport height or viewport width. 
div{
  height: 20vmin;
}
vmax
One vmax is 1% of whichever is larger, viewport height or viewport width. 
div{
  width: 25vmax;
}
ex
based on the height of x of the declared font 
p{
  line-height: 3ex;
}
em
based on the font size of the parent element. 
h1{
  font-size: 2em;
}
rem
based on the font size of the root element 
.main{
  font-size: 2rem;
}
ch
relative to the width of the 0 
p{
  font-size: 3ch;
}
