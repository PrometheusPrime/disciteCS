Class and Id Selectors

The Class Selector is used to style a specific element(s).A class can be used as many times as needed,on any number of elements. Use the class attribute to add a class to an element.To select an element(s) with a class, use the period (.) character, then follow it with the class name. A class name cannot start with a number.

<!DOCTYPE html>
<html>
  <head>
    <title>Hello CSS</title>
    <link rel="stylesheet" href="style.css">
  </head>
  <body>
    <h1 class = "heading">
  This is a paragraph.
    </h1>
    <p class = "paragraph">
      This is a paragraph.
    </p>
    <p>
      This is another paragraph.
    </p>
    <p class = "paragraph">
    This is also a paragraph.
    </p>
  </body>
</html>

.paragraph{ 
  color:aquamarine;
  background-color:darkslategrey;
}
.heading{
  color:cadetblue;
}

An element can belong to multiple classes, seperate the classes with a space.

<!DOCTYPE html>
<html>
  <head>
    <title>Hello CSS</title>
    <link rel="stylesheet" href="style.css">
  </head>
  <body>
    <h1 class = "heading italicText">
    This is a paragraph.
    </h1>
    <p class = "paragraph">
      This is a paragraph.
    </p>
    <p class = "paragraph italicText">
      This is another paragraph.
    </p>
  </body>
</html>

.paragraph{ 
  color:aquamarine;
      background-color:darkslategrey;
}
.heading{
  color:cadetblue;
}
.italicText{
  font-style: italic;
}

The Id Selector is used to style a specific element.An id is used only once per page ,it is used to select one unique element. Use the id attribute to add an id to an element. To select an element with an id, use the hash (#) character, then follow it with the id name. An id name cannot start with a number.

<!DOCTYPE html>
<html>
  <head>
    <title>Hello CSS</title>
    <link rel="stylesheet" href="style.css">
  </head>
  <body>
    <p class = "paragraph">
      This is a paragraph.
    </p>
    <p id = "paragraph">
      This is another paragraph.
    </p>
    <p class = "paragraph">
    This is also a paragraph.
    </p>
  </body>
</html>

.paragraph{
  color:aquamarine;
}
#paragraph{
  color:cadetblue;
}
