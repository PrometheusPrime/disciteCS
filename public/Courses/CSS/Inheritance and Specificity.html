Inheritance and Specificity

Inheritance

A child element will inherit styles from its parent(an element in which the child element is nested in). 

 
Applying styles to a child element will overwrite the inherited ones,the most specific style rule will be applied to an
  element.


<!DOCTYPE html>
<html>
<head>
  <title>Hello HTML</title>
  <style>
    body{
     color:#7f7f7f;}
   h1{
     text-align:center;
     font-family:'Times New Roman';
     color:darkcyan;}
    #myId{
     color:darkseagreen;font-family:verdana;}
  </style>
</head>
<body>
  <h1>
  This is a Heading with specific styles.
  </h1>
  <p>
  This paragraph is inheriting from parent.
  </p>
  <h2>
  This Heading is inheriting from parent.
  </h2>
  <p id="myId">
  This is a paragraph with specific styles.
  </p>
</body>
</html>



Child elements usually don't inherit certain properties from the parent. To force a child to inherit a property from the parent add inherit as that property's value.



button{
  font-style : inherit;
}
 

Specificity

if a Selector is defined multiple times, the last ruleset will be applied. The most specific rule applies

element -> class -> id

 !important;
