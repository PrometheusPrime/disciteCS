Variables

Define variable in the :root pseudo class

:root {
  --ff-codeBlock-: "Source Code Pro", "lucida console", Terminal, Consolas,
    Courier, "courier new";
  --ff-code-: "Cascadia Code", Inconsolata, Terminal, Consolas;
}

Use the var() function 

kbd,
samp,
code {
  font-family: var(--ff-code-);
  direction: ltr;
  unicode-bidi: bidi-override;
}
