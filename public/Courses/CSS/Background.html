Background

Background Color

Change the background Color of an element with the background-color property.


   
Background Image

   The background-image property defines an image to use as the background of an element.
   

   
By default, the image starts at 0,0 (top-left corner of an element) and is repeated both horizontally and vertically.

   
Background Image - Repeat

   
The repeat behavour of a background image can be changed by using the background-repeat property. Some of the values of the background-repeat property are ;repeat-y image will only be repeated vertically,repeat-x	image will only be repeated horizontally,no-repeat the background-image will not be repeated.
