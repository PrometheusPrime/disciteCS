Border

The borders of an element is styled using the border property. Using the property, we can change the width,color and style of the border.

{
    border:;
 }
We can style each of the border attributes seperately. 
{
    border-width:;
    border-color:;
    border-style:;
 }
Remember that the border has four parts, that means we can style each one of them independently. 
{
    border-right:;
    border-left:;
    border-top:;
    border-bottom:;
 }
As you might have guessed we can style each of the border attributes of any of the four parts of the border. 
{
    border-right-width:;
    border-left-color:;
    border-top-style:;
}
