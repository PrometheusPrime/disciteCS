CSS

CSS (Cascading Style stylesheet) is used to control the way the HTML Document is presented or rendered(text color, background color,font type, et cetra).

There are three ways to style (add CSS to) a web page.

Inline Styles

The Style Attribute is used to change the style of an HTML element. It has the following syntax : 

style = "property:value;"
Add the Style attribute to the Element you want to style. The property is what you want to change on an element. 

<!DOCTYPE html>
<html>
  <head>
    <title>Hello HTML</title>
  </head>
  <body>
    <p style="color:aquamarine;background-color:darkslategrey">
          This is a paragraph.
    </p>
    <p  style="color:cadetblue;">
          This is another paragraph.
    </p>
  </body>
</html>

Adding styles to an element with the style attribute is called Inline Styling.

You can add property-value pairs on an element.

<!DOCTYPE html>
<html>
  <head>
    <title>Hello HTML</title>
  </head>
  <body style="background-color:#7f7f7f;">
    <h1 style="text-align:center;color:rgb(255,255,255);">Hello There !!</h1>
    <p style="color:aqua;font-family:'Times New Roman';font-size:40px;">
    This is a paragraph.
    </p>
    <p style="color:aquamarine;font-family:verdana;font-size:30px;">
    This is another paragraph.
    </p>
  </body>
</html>

Internal/Embeded Styles

You can also style an HTML Document with the <style> element. The style element is nested inside the head element. Styling an HTML Document with the style element is called Internal Styling. It follows the following syntax : 

<style>
  selector {property:value;}
</style>
The selector is the part of the HTML Document you want to style , property:value; is called Declaration. 

<!DOCTYPE html>
<html>
  <head>
    <title>Hello HTML</title>
    <style>
      body{background-color: darkgrey;}
      h1{
        text-align:center
        ;font-family:cursive;
        color:darkcyan;
      }
      p{
        color:aquamarine;
        font-family:verdana;
      }
    </style>
  </head>
  <body>
    <h1>This is a Heading.</h1>
    <p>This is a paragraph.</p>
    <p>This is another paragraph.</p>
  </body>
</html>

You can add same styles to multiple selectors by comma seperating each selector.

<!DOCTYPE html>
<html>
  <head>
    <title>Hello HTML</title>
    <style>
      h1,h2{
        text-align:center;
        font-family:'Times New Roman';
        color:darkcyan;
      }
    </style>
  </head>
  <body>
      <h1>This is a Heading Level 1.</h1>
      <p>This is a paragraph.</p>
      <h2>This is a Heading Level 2.</h2>
  </body>
</html>
