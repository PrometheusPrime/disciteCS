Arrays

An array is a container object that holds a fixed number of values of a single Data Type , referred by a single name. Elements in an array are ordered. The first element is at index( or position) 0.

short[] numArray = new short[4];

To add or change a value in an array, write the name of the array followed by the index (position) in square brackets, assignment operator then the value.

numArray[0] = 3;
numArray[1] = 1;
numArray[2] = 4;
numArray[3] = 2;

To access a value in an array, write the name of the array followed by the index (position) in square brackets. The first element is at 0, so the last element is at numberOfElement - 1

Console.WriteLine(arr[2]);

Array Properties and Methods

Array Methods and Properties check the size or length of an Array with the Length property. 
Console.WriteLine(movies.Length);
Copy an array using the Copy() method in the Array class, pass in the method the source array, the array to copy to and number of elements to copy. 
String[] watchList = new String[5];
Array.Copy(movies,watchList,5);
Get the index of an element 
Array.IndexOf(movies, "Sonic the Hedgehog 2");
The Sort() method in the Array class sorts the elements in an array in ascending order. 
Array.Sort(movies);
The Join() method in the String class returns a string representation of an array. Pass a character to use to separate elements of the array. 
String str = String.Join(',',numArray);
Convert a string to an array with the Split() method. Pass into the method a character where to split the string into elements. 
string myStr = "The Batman";
String[] myStrArray = myStr.Split(' ');

Iterate through an Array

Iterate directly through an array's element use the for loop

for(int i = 0; i < movies.Length; i++){

    Console.WriteLine(movies[i]);
}

To iterate directly through each element in an array, use the foreach loop.

foreach (string movie in movies)
{

    Console.WriteLine(movie);
}

Multidimensional Arrays

A multidimensional array is an array whose elements are also arrarys. The Array and the nested arrays must be of the same type.

//multidimensional array riteral
int[,] nums = {
    {2, 4, 8, 16},
    {3, 9, 27, 81},
    {4, 16, 64, 256}
};

//multidimensional array declaration
int[,] numsII = new int[3,4];

//multidimensional array initialisation
numsII[0,0] =  5;
numsII[0,1] =  25;
numsII[0,2] =  125;
numsII[0,3] =  625;


numsII[1,0] =  6;
numsII[1,1] =  36;
numsII[1,2] =  216;
numsII[1,3] =  1296;

numsII[2,0] =  7;
numsII[2,1] =  49;
numsII[2,2] =  343;
numsII[2,3] =  2401;

Console.WriteLine(nums[2,0]);
Console.WriteLine(numsII[0,0]);

To traverse a multidimensional array, use nested for loops.

for (int i = 0; i < 3; i++)
{
    for (int j = 0; j <= 3; j++)
    {
        Console.Write(numsII[i, j]);
        Console.Write(" ");
    }
    Console.WriteLine();
}
