Variables and Data Types

Variables

A Variable is a container for storing data. C# is statically-typed ,once a variable's type is defined it can not change.

To Create a Variable in C# , you'll have to write the type of Data, variable name = and then the value; The Equals (=) sign is used to assign a value to a variable.

An identifier(variable name) can not start with a number, contain space or be a reserved keyword. 
short age = 18 ;
Another way of defining a variable is by using the var keyword. When defining a variable using the var you don't have to specify the data type. 
var characterName = "John Wick";
Console.WriteLine(charactername);

You can also declare a variable and assign it a value later.

String name ;
name = "Jane";
Before a variable is used, it must be initialised.

The value of a variable can be changed as many times as needed.

String name  = "Jane";
name = "John Wick";
name = "John Wick";

Constants

Use the const keyword to define a constant. A constant must be initialised. 
const double PI = 3.1415;

Primitive Data Types

Primitive data types are the most basic data types provided by a Programming Language. C# has 5 Primitive Data Types : int, long, decimal, float, double, character and boolean.

float are 32-bit single-precision floating point numbers, are 4 bytes in size and have a default value of 0.0 . To create a float you must end the value with an f, otherwise you'll get an error because it will be interpreted as a double. 

float  height = 6.4f ;
Alternitively, cast it to float. 
float  height = (float) 6.4;

double is a 64-bit double-precision numbers and is 8 bytes in size. 

# integers
double pi = 3.1415;

char is a single 16-bit Unicode character ranging from '\u0000' or 0 to '\uffff' or 65,535. A char is 2 bytes in size. Char values are surrounded by single quotes (''). 

char grade = 'A';

Boolean can only take the values true or false and are 1 byte in size. 

bool isRaining = false;
