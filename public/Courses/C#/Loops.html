Loops

Loops are used to repeatedly execute code if a certain condition or an expression is True.

While Loop : is used to continually execute a block of statement(s) while a particular condition is true.

The expression is checked, if it evaluates to true; the statements in the while block will be executed.The while statement will continue checking the expression and executing its block of statements until the expression evaluates to false 
int num = 0;
while (num < 15)
{
    Console.WriteLine(num);
}
If the condition does not evaluate to false, the Loop runs indefinitely (infinite Loop). Press Ctrl + C to stop execution.A condition or an expression can be updated inside the Loop so that it eventually evaluates to false. 
int num = 0;
while (num < 15)
{
    Console.WriteLine(num);
    num++;
}
Explicitly create an infinte loop by adding true as the condition. 
int num = 0;
while (true)
{
    Console.WriteLine(num);
    num++;
}

Do... While Loop

The do... while loop is the same as the regular while, except in the do... while loop The condition is checked at the end. So it's block is executed at least once, even if the condition is false.

do
{
    Console.WriteLine("4 is greater than 5");

} while (4 > 5);

For Loop

The for loop is used to repeatedly execute a block of code a known number of times.

for (int num = 0; num <= 10; num++)
{
    Console.WriteLine(num);
}

break and continue

break exits the loop and execution goes to the first statement after the loop. 

for (int num = 0; num <= 10; num++)
{
    if (num == 6)
    {
        break;
    }
    Console.WriteLine(num);
}

continue skips execution of the rest of the code in the loop in the current interation and proceeds to the next interation.

for (int num = 0; num <= 10; num++)
{
    if (num % 2 != 0)
    {
        continue;
    }
    Console.WriteLine(num);
}
