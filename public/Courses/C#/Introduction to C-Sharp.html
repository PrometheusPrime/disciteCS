How to write and run C# programs

Here is a simple Hello world program in C#. Remember to replace namespace with your project folder. 
namespace IntroToCSharp
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, World!");
        }
    }
}
There are several ways you can write and run C# programs

Using any code Editor

First install the .NET Framework, to do that go to https://dotnet.microsoft.com/download and download a .NET SDK for the OS you're using and install it.

To check if .NET was installed correctly,open a new terminal window and run
dotnet --version

Create a folder for your project, open a new terminal window in the project directory and run dotnet new console to create a new C# project.

Some files and folders will be generated, open Program.cs in a code editor of your choice, it is where we'll be writing most of the code in this course.

In the following examples , I'm using Visual Studio Code.

To run our console application, in a terminal run dotnet run. The output of our program is shown in the terminal.

Using Visual Studio

Download a Visual Studio installer for Visual Studio Community Edition. Install it and open it, then select .NET Desktop Development.

Open Visual Studio, select Create a new project, select Console app c# and click Next

Select the .Net Framework to use and click Create.

To run the program ,click Ctrl + F5 or the play button.

Another way of develop C# projects on Windows, Linux or MacOS is to use the Mono Project and xamarin. Search online on how to setup.
