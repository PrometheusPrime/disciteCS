Methods

A method is a group of statements that perform a single operation. Methods are defined in a class

To create a method, write the type of value that will be returned follow by name of the method and parenthesis. The name of a method is usually capitalised in C#. 
// in class Program, outside Main method
static void Sum(){
    Console.WriteLine(5 + 5);
}
We've used void as the return type because the method does not return a value.

To execute a method, you have to call it. To call a method, write the name of the method followed by parenthesis() and a semicolon. Methods can be called as many times as needed.

// in Main method
Sum();

Return values from a Method

Return a value from a method with the return keyword. Make sure the returned value matches the return type of the method.

// in class Program, outside Main method
static int Sum(){
    return 5 + 5;
}
We can assign a value returned from a method to a variable to use it. 
// in Main method
int total = Sum();
Console.WriteLine(total);

Parameters and Arguments

A method can take data to work with called Parameters. Parameters are defined in parenthesis when defining the Method. If a method takes multiple Parameters, comma seperate them. 
static int Sum(int a, int b){
    return a + b;
}
Arguments are actual data passed into a function when calling it. A method must be called with the number and type of arguments it expects. 
Sum(4,5);

Pass arguments in any order

Arguments are passed into a method in the order it's parameters are defined.

static void Info(string firstName, string lastName)
{
    Console.WriteLine($"Name : {firstName}\nSurname : {lastName}");
}
// in Main method
Info("Wick", "John");
When calling the method, you can specify which argument you're passing. 
Info(lastName: "Wick", firstName: "John");

Default Parameters

A parameter can be given a default value. When calling the method, passing an argument for that parameter is optional.

static int Sum(int a, int b = 0){
    return a + b;
}

// in Main method
Sum(4);
Sum(4,8);
Parameters with default values must be defined after required parameters.

Get Unkown number of Arguments

To get a varing number of arguments, add the params modifier before a parameter.

// in class Program, outside Main method
static int Sum(params int[] nums)
{
    int total = 0;
    foreach (int num in nums)
    {
        total += num;
    }
    return total;
}

Console.WriteLine(Sum(1,2,3,4,5,6,7,8,9));
Console.WriteLine(Sum(8,9));
Console.WriteLine(Sum(9));
Console.WriteLine(Sum());
A method can have additional parameters, but they must be defined before params, and only one params is allowed per method.

Method Overloading

Method Overloading is defining a method multiple times with different number and or types of parameters. 
// in class Program, outside Main method
static void Greet()
{
    Console.WriteLine("Hello");

}

static void Greet(string name)
{
    Console.WriteLine($"Hello, {name}");

}

static void Greet(String name, string gender)
{
    if (gender.ToLower() == "male")
    {
        Console.WriteLine($"Hello, Mr {name}");
    }
    else if (gender.ToLower() == "female")
    {
        Console.WriteLine($"Hello, Ms {name}");
    }
    else
    {
        Console.WriteLine($"Hello, {name}");
    }

}

Greet();
Greet("Jake");
Greet("Jane","female");

Pass by Reference

By default arguments are passed by value to a method, that means argument's value is copied. Therefore you can not modify a variable in a method.

// in Main method
int age = 9;
IncrementByTwo(age);
Console.WriteLine(age);

// in class Program, outside Main method
static int IncrementByTwo(int num)
{
    num += 2;
    return num;
}
To pass an arguement's reference(memory location), add the ref modifier before the parameter's type when declaring the method and before the argument when calling the method. 
// in Main method
int age = 9;
IncrementByTwo(ref age);
Console.WriteLine(age);

// in class Program, outside Main method
static int IncrementByTwo(ref int num)
{
    num += 2;
    return num;
}
