Conditional Statements

Control Statements are used to run code under certain conditions

The if, else and else if Statements

If statement

The code in the block will be executed if the condition is true. The condition is either a boolean variable or expression that returns a boolean

if (4 < 5)
{
    Console.WriteLine("4 is less than 5");
}

If You want to execute some code if multiple conditions are true, place it in nested if statements.

int age = 22;

if (age > 18)
{
    if (age < 25)
    {
        Console.WriteLine("You're between 18 and 25 years of age");
    }
}
Instead of using nested if statements, we can use logical operators to combine logical expressions. 
int age = 22;

if (age > 18 && age < 25)
{
    Console.WriteLine("You're between 18 and 25 years of age");
}

Else statement

The else statement defines alternate block of code to execute if the condition in the if statement is false.

if (4 > 5)
{
    Console.WriteLine("4 is greater than 5");
}
else
{
    Console.WriteLine("4 is less than 5");
}
You can nest if else statements.

Else If statement

The else if statement is used to check additional conditions.

bool isRed = false;
bool isGreen = true;

if (isRed)
{
    Console.WriteLine("Stop !");
}
else if (isGreen)
{
    Console.WriteLine("Drive");
}
else
{
    Console.WriteLine("Drive, if it is safe to do");
}

Make sure the conditions are mutually exclusive. Even if multiple conditions are true, only the first true statement will be executed other statements will be skipped.

bool isSunny = true;
bool isRaining = true;

if (isSunny)
{
    Console.WriteLine("It is Sunny");
}
else if (isRaining)
{
    Console.WriteLine("It is Raining");
}
else if (isSunny && isRaining)
{
    Console.WriteLine("It is Sunny and Raining");
}
else
{
    Console.WriteLine("It's neither Sunny nor Raining");
}

The Switch

The switch statement is used we want to do different operations depending on a particular value of a variable.

var gender = "Male";
var lastName = "Smith";

switch (gender)
{
    case "Male":
        Console.WriteLine($"Hello, Mr {lastName}");
        break;
    case "Female":
        Console.WriteLine($"Hello, Ms {lastName}");
        break;
    default:
        Console.WriteLine($"Hello, {lastName}");
        break;
}
The break statement goes out of the switch statement after a matched case is executed. It is required.

The default case is executed if non of the cases matched. 

string? gender = null;
var lastName = "Smith";

switch (gender)
{
    case "Male":
        Console.WriteLine($"Hello, Mr {lastName}");
        break;
    case "Female":
        Console.WriteLine($"Hello, Ms {lastName}");
        break;
    default:
        Console.WriteLine($"Hello, {lastName}");
        break;
}

Multiple cases can execute a single block.

var num = 4;

switch (num)
{
    case 5:
    case 4:
    case 3:
        Console.WriteLine("num is either 5, 4 or 3");
        break;
    case 2:
        Console.WriteLine("num is 2 ");
        break;
    case 1:
        Console.WriteLine("num is 1");
        break;
    default:
        Console.WriteLine("num is neither 5,4 nor 3");
        break;
}
