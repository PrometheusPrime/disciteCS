List

using System.Collections.Generic;
List<string> opposites = new List<string>();
Add an item to a list 
opposites.Add("contradictory");
opposites.Add("antagonistic");
opposites.Add("antithetical");
opposites.Add("contrary");
opposites.Add("polar");
Add multiple items to a list with the AddRange(), pass an array or a list. The items will be added at the end of the list. 
string[] oppositeWords = {
  "paradoxical",
  "contraindicate",
  "countermand",
  "irreconcilable",
  "u-turn",
  "diametrically",
  "opposite",
  "antonym"
 };

 opposites.AddRange(oppositeWords);
Insert an item at a specific index 
opposites.Insert(5, "inverse");
Get the number of items in a List Count property 
opposites.Count;
Check if an item is in a List 
opposites.Contains("paradoxical");
Get the index of a list item, returns a negative number is the item is not in the list. 
opposites.IndexOf("inverse");
Sort a List, the actual list is modified. 
opposites.Sort();
Copy an Array into a List 
list<type> myList = new list<type>(new type[]{value,value});
or 
list<type> myList = new list<type>(anArray);
Remove an item from a list 
opposites.Remove("opposite");
RemoveAt() removes an item at the specified index from the list. 
opposites.RemoveAt(9);
Remove all items in a list 
opposites.Clear();
