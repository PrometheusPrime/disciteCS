Strings

String are sequence of characters. Unlike chars, strings are delimited by Double quotes(""). A string is not a primitive type, it is an objects.

String greet = "Hello, World";
Console.WriteLine(greet);

Note that you can use String or string to create a string. string is an alias for the String class.

Escape Sequences

There are characters that have a special usage in strings. If we want to use them like ordinary characters, we have to escape them. Use the Backslash (\) to escape any special Characters in a string

string directory = "C:\\Users\\CSf\\.bash_history";
Console.WriteLine(directory);

String operations

Use the Length property to get the number of characters in a string. 
string name = "Jane";
Console.WriteLine(name.Length);
Characters in string are ordered, a get a character at an index 
string name = "Jane";
Console.WriteLine(name[2]);
Strings in C# are immutable, individual characters in a string can not changed. 
string name = "Jane";
name[2] = 'k';
Console.WriteLine(name);

The addition operator is used to join strings

string firstName = "John";
string lastName = "Wick";
string fullName = firstName + " " + lastName;
when you concatenate a string and a number, you'll get a string.

String Interpolation

You can substitute values of variables or expression into a string. Add a dollar sign before the opening quote and place the variable or expression in curly brackets. 
string person = "John Wick";
string greet = $"Hello, {person}";

Console.WriteLine(greet);

var num = 5;

Console.WriteLine($"{num} Ã— {num} = {num * num}");

String Methods

Remember strings are immutable, so any method or operation that appear to change the string returns a new string. 
string quote = "Everything has Beauty, but not Everyone Sees It - Confucius";

Use the .IsNullOrEmpty() method in the string class to check if a string is an empty string, it returns true if it is.

string.IsNullOrEmpty(quote);
Use the Contains() method to check if a string is a substring of the string or if the string contains a certain character, it returns a boolean. 
Console.WriteLine(quote.Contains("Beauty"));
Console.WriteLine(quote.Contains('Y'));
Get start start index of a substring in a string with the IndexOf() method, it returns -1 if a string is not part of the string. 
quote.IndexOf("Beauty");
Use the equals() in the String class to compare strings instead of the equality operator. if the strings you're comparing are the same but in different Cases the returns false. 
string aQuote = "everything has beauty, but not everyone sees it - confucius";
String.Equals(quote,aQuote);
Check if a string starts with a certain character or string. 
quote.StartsWith('E');
quote.StartsWith("Beauty");
Check if a string ends with a certain character or string. 
quote.EndsWith('e');
quote.EndsWith("Confucius");
Replace a character or substring with the Replace() method 
quote.Replace('e', ' ');
quote.Replace("Beauty", "it's Ending");
The Remove() method removes characters from a string. Pass into the method the index where to start removing, it will remove the characters upto the end of the string. 
Console.WriteLine(quote.Remove(47));
You can pass the number of characters you wish to remove as a second arguement. 
Console.WriteLine(quote.Remove(47,4));
Get a substring from specified index to the end of a string. 
quote.Substring(50);
You can specify the number of characters the method should return as a second arguement. 
quote.Substring(0,21);
The ToLower() method return a copy of the string with all the letters converted to lowercase. 
quote.ToLower();
The ToUpper() method return a copy of the string with all the letters converted to uppercase. 
str.ToUpper();
Remove whitespace from the start and end of a string. 
quote.Trim();
Removes whitespace atm the start a string. 
str.TrimStart();
Removes whitespace at the end of a string. 
str.TrimEnd();
