Classes and Objects

class Classes enable us to create a custom type to represent complex things in code.

class Person
{
    public String name;
    public String gender;
    public int age;

    public string GetPersonalInfo()
    {
        return $"Name : {name}\nGender : {gender}\nAge : {age}";
    }


}

Object

Create an instance(object) of a class with the new keyword. You can create any number of objects from a class. 
Person fury = new Person();

An object's members (variables and methods) are accessed using the dot (.) notation. 

fury.name = "Nick Fury";
fury.gender = "male";
fury.age = 30;

Console.WriteLine(fury.GetPersonalInfo());

Constructor

A Constructor is a special method that is called when an instance of a class is created. The Default Constructor is automatically generated, hence there's no need to include. Unless you add another constructor and still want to be able to create objects without passing arguments to the constructor. In fact you can overload the constructor method as many times as you need.

class Person
{
    public String name;
    public String gender;
    public int age;

    public string GetPersonalInfo()
    {
        return $"Name : {name}\nGender : {gender}\nAge : {age}";
    }

    //Default Constructor
    public Person()
    {

    }
    // Secondary Constructor
    public Person(string name, string gender, int age)
    {
        this.name = name;
        this.gender = gender;
        this.age = age;
    }
}



static Fields and methods are shared between the Objects of the class. Call static on the class instead of on the objects.


class Person
{
    public static int numberOfPeople;
    public String name;
    public String gender;
    public int age;

    public string GetPersonalInfo()
    {
        return $"Name : {name}\nGender : {gender}\nAge : {age}";
    }

    //Default Constructor
    public Person()
    {
        numberOfPeople++;

    }
    // Secondary Constructor
    public Person(string name, string gender, int age)
    {
        this.name = name;
        this.gender = gender;
        this.age = age;
        numberOfPeople++;
    }
}
