Getting Input

We can get input with the ReadLine() method from the Console class.

Console.ReadLine();
To use the input from the user, you have to assign it to a variable. 
Console.Write("What is your name ? ");
string name = Console.ReadLine();

Console.WriteLine($"Nice to meet you, {name}");
By default the ReadLine() method returns a string. Convert numbers entered by the user to numbers, to use them as numbers.

getting command line arguments

Environment.GetCommandLineArgs();
