Dart

General purpose programming language developed by Google.

Install Dart

You can write and run Dart programs online without installing anything by going to dartpad.dev

To write and run Dart programs locally on your computer, you'll need to install the Dart SDK. Go to dart.dev/get-dart for instructions on how to install Dart on your System.

If you want to manually download and install the Dart SDK, go to dart.dev/get-dart/archive. Make sure you download from the stable channel and add the Dart SDK directory upto /bin to your path after installation.

Run a Dart program

Any code editor can be used to write a dart program. If the code editor you're using supports programming language extensions/packages search and install one for Dart. It will help you with syntax highlighting,debugging etc.

Create a new file, name it whatever you prefer with .dart as the file extension. Write the simple dart program code below in the file and save it.

void main(){

    print("Hello, world !");
}

To run the program, open a terminal window in the folder where your dart source file is and run dart file.dart

The main() function is the programme execution entry point.

Curly brackets {} are used to delimit a block of code.

A semi colon ; is used to terminate a statement. It is required.
