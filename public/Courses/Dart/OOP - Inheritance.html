Inheritance

Suppose we want to create a class but already have a class with the same properties that our new class will have. we can simply inherit those properties and add properties unique to that class. The extends keyword is used to inherit from a class. 
class Person{
  string firstName
  string lastName
  int age

  Person(this.firstName,this.lastName,this.age){
  
  }
}

class Employee extends Person{

    static numberOfEmployees = 0
    int Id
    string position

    Employee(){
    Employee.numberOfEmployees ++;
  }

}
