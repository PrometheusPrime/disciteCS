Variables and Data Types

A variables is a container for storing data in computer memory. Dart is statically typed,this means that a variable's declared type can not change.

int age = 19;
You can also create a variable using the var keyword, it's type will be inferred from the value assigned to it. 
var height = 5.7;

You can also declare a variable and assign it a value later

var characterName;
int age;
characterName = 11;
age = 14;
If you just declare a variables with var, it will be a dynamic variable until it's assigned a value. A variable can not be used before it it initialised(given a value).

Although Dart is statically typed, you can define dynamic variables. A dynamic variable can be reasigned a value of a different type.

dynamic characterName = "Mike";
characterName = 11;
characterName = "Vecna";

A variable's value can not be null.

Constants

Constants are defined with the final and const keywords. A constant can not be reassigned new value.

A value of a constant variable defined with final can be determined at run time.

final  pi = 3.141592;
final currentDateAndTime = DateTime.now();

const makes an object constant at compile time. A const variable's value must be known at compile time, so it can not be for example a call to a function.

const piX2 = 3.141592 * 2;
const goldenRatio = 1.618033988749894;

Data Types

Everything in dart is an object

Numbers

Dart supports the following numeric type:

An integer is a negative or positive whole number. Integers are defined with the int keyword.

int annualSalary = 125000;

A double is a 64bit floating point number.

double weight = 67.8;
The num keyword is used to define a generic numeric variable. The variable will be converted to the appropriate numeric type, inferred from the value. 
num x;
x = 6;
num pi = 3.1415;
A variable defined with num can hold any numeric type. 
num x;
x = 6;
x = 3.1415;

String

A string is a sequence of characters. Strings in dart are delimited by single or double quotes.

String capitalCity = 'Lusaka';

Boolean

A boolean is either true or false. A boolean is defined with the bool keyword.

bool isTall = false;

List

A list is an ordered collection of objects.

var evenNums = [2, 4, 6, 8, 10];

Map

A map is a collection of values in key-value pairs.

var nums = {
    0: 'Zero',
    1: 'One',
    2: 'Two',
    3: 'Three',
    4: 'Four',
    5: 'Five',
    6: 'Six',
    7: 'Seven',
    8: 'Eight',
    9: 'Nine'
};

Strings, Maps and Lists will be discussed in details later.
