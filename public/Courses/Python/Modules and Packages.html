Modules and Packages

Modules

Modules are simply python files containing python code (classes ,functions etc). The file name is the module name. Use the import keyword to import a python module. 

import math
x = math.pow(5,3)
print(x)
To import a module and give it an alias use the as keyword, then the alias. 
import math as m
x = m.sqrt(9)
print(x)

Import specific class(es) or function(s) from a module using the from keyword

from math import pi, dist

Import specific class or function from a module and give it an alias

from math import fsum as sum
x = sum([1,2,3,4,5,6,7,8,9])
print(x)

Packages

Packages are collections of Modules. package name is the folder containing all the modules. __init__.py tells python that this is a package. it can be empty.
