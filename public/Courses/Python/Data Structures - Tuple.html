Tuple

A Tuple is a Collection Compound Data Type that ...

  • is ordered
  • is immutable
  • can contain elements of different Data Types
  • can contain duplicate entries
Tuple is create with parenthesis , comma seperate values... 
a_tuple = ()

a_tuple = ('year', 6,True)
print(a_tuple)

can also create a Tuple with the tuple() function from an Iterable eg a List. 

a_list = ['year', 6,True]
a_tuple = tuple(a_list)
print(a_tuple)

to create tuple with one element add a trailing comma

a_tuple = ('year',)
print(a_tuple)
Iterate through a Tuple with the for loop 
a_tuple = ('year', 6,True)
for x in a_tuple:
    print(x)
To check for an element in a Tuple use the in keyword. 
a_tuple = ('year', 6,True)
print('year' in a_tuple)

indexing

access element in a Tuple using index of an element, name of the tuple followed by square brackets with the index number. 

a_tuple = ('year', 6,True,1.1)
print(a_tuple[0])

negative indexing

a_tuple = ('year', 6,True,1.1)
print(a_tuple[-1])

slicing

a_tuple = ('year', 6,True,1.1)
print(a_tuple[1:3])

a_tuple = ('year', 6,True,1.1)
print(a_tuple[1:])

a_tuple = ('year', 6,True,1.1)
print(a_tuple[:2])

a_tuple = ('year', 6,True,1.1)
print(a_tuple[:])

Tuple methods and functions

Convert a Tuple to a List with the list() function 

a_tuple = ('year', 6,True)
a_list = list(a_tuple)
print(a_list)

The len() function is used to count the number of elements in a Tuple

a_tuple = ('year', 6,True, False, 6, 9)
print(len(a_tuple))

index() method is used to find the index of an element ,returns first occurance the element

a_tuple = ('year', 6,True, False, 6, 9)
print(a_tuple.index(6))

count() method is used to count an element occurance

a_tuple = ('year', 6,True, False, 6, 9)
print(a_tuple.count(6))
