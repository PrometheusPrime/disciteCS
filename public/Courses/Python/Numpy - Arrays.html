Array

collection ordered, same data types, can be changed, can contain duplicate value
  • ordered
  • mutable
  • elements are of the same data types
  • can contain duplicate value

NumPy

Python doesnt have Array data type,the closest thing to an Array in Python is a List. use Arrays in Python by importing the NumPy (Numerical Python) Library. NumPy is an Open Source project created by Travis Oliphant in 2005.

pip install numpy
import numpy using the import keyword can give numpy an alias( alternate name ) when importing it with the as keyword. 
import numpy as alias

create an array using the array() function in numpy. Pass a list or a tuple in the array function.

using a List 
import numpy as npy
arr = npy.array(['Spam','Eggs','Spam'])
print(arr)
using a Tuple 
import numpy as npy
arr = npy.array(('Spam','Eggs','Spam'))
print(arr)

Array Operations

len() function is used to count the number of elements in a Tuple

import numpy as npy
arr = npy.array(['Spam','Eggs','Spam'])
print(len(arr))

accessing elements/negative

import numpy as npy
arr = npy.array(['Spam','Eggs','Spam'])
print(arr[1])  

import numpy as npy
arr = npy.array((1,2,3,4,5))
print(arr[-1])

Multi Dimensional Arrays

import numpy as npy
arr = npy.array(((1,2,3),(1,4,9)))
print(arr)

check array dimension with the ndim attribute

import numpy as npy
arr = npy.array(((1,2,3),(1,4,9)))
print(arr.ndim)

changing value at index

import numpy as npy
arr = npy.array(['Spam','Eggs','Spam'])
arr[-1] = 'Pie'
print(arr)

concatenation

import numpy as npy
arr1 = npy.array(['Spam','Spam','Spam'])
arr2 = npy.array(['Eggs','Eggs','Eggs'])
arr3 = npy.concatenate((arr1,arr2))
print(arr3)
