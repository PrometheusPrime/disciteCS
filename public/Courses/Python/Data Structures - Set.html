Set

A Set is a Collection Type that ...

  • is unordered
  • is mutable
  • can contain elements of different Data Types
  • can only contain unique elements.

A Set defined with curly brackets.

a_set = {5,8,'p',0,3,4}
print(a_set)

You can also create a Set from an Iterable with the set() function.

a_set = set(iterable)

a_set = set("Hello")
print(a_set)

You can check if an item is in a set with the in operator.

a_set = {5,8,'p',0,3,4}
print(0 in a_set)

You can not access individual items in a set because they are not ordered. Use a for loop with the in operator to loop through all the elements in a Set.

a_set = {5,8,'p',0,3,4}
for element in a_set:
    print(element)

set comprehension

set1 = {5,8,'p',0,3,4}
set2 = {element for element in set1}
print(set2)

set1 = {5,8,'p',0,3,4}
set2 = {element expr for element in set1}
print(set2)

set1 = {5,8,'p',0,3,4}
set2 = {element for element in set1 if type(element) == int}
print(set2)

Set operations

in a but not in b

set1 = {5,8,6,2,0,3,4,'s'}
set2 = {9,0,6,50,3,2}
print(set1 - set2)

in a or b but not both

set1 = {5,8,6,2,0,3,4,'s'}
set2 = {9,0,6,50,3,2}
print(set1 ^ set2)

in both intersection of the sets

set1 = {5,8,6,2,0,3,4,'s'}
set2 = {9,0,6,50,3,2}
print(set1 & set2)

in a or b or both

set1 = {5,8,6,2,0,3,4,'s'}
set2 = {9,0,6,50,3,2}
print(set1 | set2)

Set Functions and Methods

The len() function return the number of elements in a set.

set1 = {5,8,6,2,0,3,4,'s'}
print(len(set1))

Add an element to a Set with the the add() method

a_set = set()
a_set.add('z')
a_set.add('a')
a_set.add(2)
print(a_set)
Add multiple elements to a Set using the update() method. 
a_set = set()
a_set.update(['z','a',2])
print(a_set)

The pop() method removes and also returns an element from a Set.

a_set = {5,8,'p',0,3,4}
poped = a_set.pop()
print(poped)

Remove an element from a Set with the discard() method . 

a_set = {5,8,'p',0,3,4}
a_set.discard(0)
print(a_set)

You can also remove an element from a Set with the remove() method . The remove() method will raise an error if the element is not a member of a Set.

a_set = {5,8,'p',0,3,4}
a_set.remove('p')
print(a_set)

Remove all elements from a Set with the clear() method .

a_set = {5,8,'p',0,3,4}
a_set.clear()
print(a_set)

The isdisjoint() method checks if the intersection of the sets is an empty set. Return True if the sets have no elements in common.

Use the issuperset() method to check whether a set is a superset of another set. 

set1 = {5,8,6,2,10,3,4,1,7,9}
set2 = {8,6,10,4,2}
print(set1.issuperset(set2))

Use the issubset() method to check whether a set is a subset of another set.

set1 = {5,8,6,2,10,3,4,1,7,9}
set2 = {8,6,10,4,2}
print(set1.issubset(set2))

update() method updates the set with its union with other set(s).

set1 = {5,8,6,2,0,3,4,'s'}
set2 = {9,0,6,50,3,2}
set1.update(set2)
print(set1)

union() method returns a set containing the union of the sets.

set1 = {5,8,6,2,0,3,4,'s'}
set2 = {9,0,6,50,3,2}
set3 = set1.union(set2)
print(set3)

intersection() method returns a set that is the intersection of the sets .

set1 = {5,8,6,2,0,3,4,'s'}
set2 = {9,0,6,50,3,2}
set3 = set1.intersection(set2)
print(set3)

difference() method returns a set containing the difference of the other sets.

set1 = {5,8,6,2,0,3,4,'s'}
set2 = {9,0,6,50,3,2}
set3 = set1.difference(set2)
print(set3)

difference_update() method .

set1 = {5,8,6,2,0,3,4,'s'}
set2 = {9,0,6,50,3,2}
set1.difference_update(set2)
print(set1)

symmetric_difference() method returns a set with the symmetric difference of the sets.

set1 = {5,8,6,2,0,3,4,'s'}
set2 = {9,0,6,50,3,2}
set3 = set1.symmetric_difference(set2)
print(set3)

symmetric_difference_update() method update the set with its symmetric difference and another.

set1 = {5,8,6,2,0,3,4,'s'}
set2 = {9,0,6,50,3,2}
set1.symmetric_difference_update(set2)
print(set1)

copying Sets

You can copy a Set by assigning the copy to the original Set. But if the copy (or original) is modified, all the Sets will be modified because they point to the same set object in memory.

set1 ={5,8,'p',0,3,4}
set2 = set1
print(f'set2 {set2}')
set2.add(101)
print('After modifying the copy')
print(f'set1 {set1}')
print(f'set2 {set2}')
Use the copy() method to copy a Set without pointing to the same Set object. 
set1 ={5,8,'p',0,3,4}
set2 = set1.copy()
print(f'set2 {set2}')
set2.add(101)
print('After modifying the copy')
print(f'set1 {set1}')
print(f'set2 {set2}')
The set() function can also be used to safely copy a Set. Pass the Set you want to copy as an argument to the function. 
set1 ={5,8,'p',0,3,4}
set2 = set(set1)
print(f'set2 {set2}')
set2.add(101)
print('After modifying the copy')
print(f'set1 {set1}')
print(f'set2 {set2}')

The frozenset() function is used to create a frozenset. A frozenset is immutable and hashable.
