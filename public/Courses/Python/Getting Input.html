Getting Input From The User

The input() function is used to get input from the user.

input()
when the run, the programme will wait for the user to click Enter, that's when the rest of the code will be executed.

You can pass a string into the function as a prompt to the user..

input('What is your name ? ')

To use the input from the user, you have to assign it to a variable.

name = input('What is your name ? ')
print(f'Nice to meet you, {name}')

By default the input function returns a string. 

num1 = input('Enter a number : ')
num2 = input('Enter another number : ')
print(f'{num1} + {num2} = {num1 + num2}')
In the example above, all the inputs are strings and are simply concatenated.

Convert numbers entered by the user to numbers, to use them as numbers.

num1 = int(input('Enter a number : '))
num2 = int(input('Enter another number : '))
print(f'{num1} + {num2} = {num1 + num2}')
In the example above, the input() function is nested in the int() function which converts the input to integers.
