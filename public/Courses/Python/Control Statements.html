Control Statements

Control Statements are used to run code under certain conditions.

If Statement

If Statements : if a certain condition is True, the code in the block will execute

if condition:
    code to be executed
Python uses Indentation to delimit a block of code.

Example 
if True:
    print("Thats True")

Nested If Statements

if statements can be nested if certain conditions are True.

is_cloudy = True
is_raining = True
if is_cloudy:
  if is_raining:
    print("It is croudy and raining ,You need an Amburrella")

age = 22
if age > 18:
  if age <= 25:
    print("You're between 18 and 25 years of age")
Logical Operators Chain conditions 
is_cloudy = True
is_raining = False
if is_cloudy or is_raining:
    print("You'll need an Amburrella ")

age = 22
if age > 18 and age <= 25:
    print("You're between 18 and 25 years of age")

Else Statement

The else statement will be executed if the if expression is false what to do instead if the condition or expression in the if Statement is false. 

is_red = False
if is_red:
    print("Stop")
else:
    print("Drive")
If - Else statements can be nested.

Else If Statement (ElIf)

The elif statement is used to check additional conditions. 

name = input("What is your name ? ")
gender = input("What is your gender ? ")
if gender.lower() == "male":
    print("Hello Mr " + name)
elif gender.lower() == "female": 
    print("Hello Ms " + name)
An if can have any number of elif statements, but only one optional else statement that will handle any situations not handled by the preceding statements.
