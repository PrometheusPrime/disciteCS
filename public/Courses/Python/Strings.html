String

A String is an ordered, immutable sequence or collection of characters (numbers , letters , spaces, et cetra). 

print("Hello World")

To display single quotes, use double quotes as delimiters and vice versa. 

print("'Monty Python's Flying Circus'")

Using the same quotes in a string as the ones you're using as delimiters will give you an error. 

#syntax error
print(''Monty Python's Flying Circus'')

Use the Backslash (\) to escape quotes or any special Characters.

print('\'Monty Python\'s Flying Circus\'')

Use (\n) to add a newline in a string. Any characters after (\n) will be pushed to a new line.

print("3\n2\n1\n0")

Prefix r before the opening quotes to create a Raw string, any special Characters in a Raw string are treated like ordinary characters.

print(r"3\n2\n1\n0")

To Display preformated or multiline strings, use three single (''') or Double (""") quotes. he output will be exactly the way it was formated in the code. 

print('''The Zen of Python, by Tim Peters

Beautiful is better than ugly.
Explicit is better than implicit.
Simple is better than complex.
Complex is better than complicated.''')

Operating On Strings

The Addition Operator (+) is used to join (concatenate) strings together. Numbers can not be added or combined with strings. 

first_name = "Jane"
last_name = "Doe"
name = first_name +" "+ last_name
print(name)

Strings can be multiplied by an integer, the result is that string repeated n times . A string can not be multiplied by another string. 

i = "I"
i_times_three = i * 3
print(i_times_three)

Indexing

Remember strings are ordered sequences of characters, we can get individual characters by writing the the name of a string followed by the position of the character in square brackets.

Index starts from 0, so the last character is at number_of_characters_in_a_string - 1.

greet = 'Hello'
print(greet[0])

print(greet[-1])

slicing

Slice a String to obtain substring. All the characters from the start index upto but not including the end index will be returned.

greet = 'Hello'
print(greet[1:5])
the characters from the start index upto the end of the string will be returned. 
greet = 'Hello'
print(greet[2:])
the characters from the start of the string upto the end index will be returned 
greet = 'Hello World'
print(greet[:5])

String Methods and Functions

The len() function returns the length (number of characters) of a string. 

first_name = "Jane"
print(len(first_name))

The strip() method removes whitespace from the beginning and / the end of a string. 

first_name = " Jane "
print("Number of characters before striping whitespace")
print(len(first_name))
first_name = first_name.strip()
print("Number of characters after striping whitespace")
print(len(first_name))
Strings are immutable, so replace() returns a new string 
greet = 'Hello World'
greet2 = greet.replace('H','Y')
print(greet)
print(greet2)
startswith() method 
greet = 'Hello'
print(greet.startswith('H'))
endswith() method 
greet = 'Hello'
print(greet.endswith('l'))

The find("C") method returns the index of a character in a String.

greet = 'Hello'
print(greet.find('e'))
count() method 
greet = 'Hello'
print(greet.count('l'))
The lower() method return a copy of the string with all the letters converted to lowercase. 
greet = 'HELLO WORLD'
print(greet.lower())

age = 20
