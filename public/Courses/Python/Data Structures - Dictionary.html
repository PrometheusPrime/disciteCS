Dictionary

A Dictionary is a Collection Data Type with elements in key value pairs, that ...

  • is unordered
  • is mutable
  • can contain elements of different Data Types
  • can not contain duplicate entries
Dictionary is written with curly brackets, keys and values are separated by colon. Separate key value pairs with a comma. 
a_dict = {'first_name':'John', 'last_name':'Doe', 'age':21}
The keys are the first element; they must be quoted, unique, hashable and of an Immutable Data Type. Values can be of any Data Type.

You can also create a Dictionary with the dict() function.When creating a Dictionary with the dict() function you don't have to quote the keys and the equals sign is used for assignment instead of the colon.

a_dict = dict(key="value",key="value")

a_dict = dict(first_name = 'John', last_name = 'Doe', age = 21)
print(a_dict)

accessing elements

keys 
a_dict = {'first_name':'John', 'last_name':'Doe', 'age':21}
print(a_dict.keys())
sorted() function returns a list of all the Keys sorted alphabetically. 
a_dict = {'first_name':'John', 'last_name':'Doe', 'age':21}
print(sorted(a_dict))
values 
a_dict = {'first_name':'John', 'last_name':'Doe', 'age':21}
print(a_dict.values())
Access a Value in a Dictionary by referring to it's Key in square brackets, returns an error if the key is not found. 
a_dict = {'first_name':'John', 'last_name':'Doe', 'age':21}
name = a_dict['first_name']
print(name)
you can also use the get() method to access an item in a Dictionary. 
a_dict = {'first_name':'John', 'last_name':'Doe', 'age':21}
name = a_dict.get('first_name')
print(name)

copying Dictionary

You can copy a Dictionary by assigning the copy to the original Dictionary. But if the copy (or original) Dictionary is modified, all the Dictionaries will be modified because all the Dictionaries are pointing to the same object in memory.

dict1 = {'first_name':'John', 'last_name':'Doe', 'age':21}
dict2 = dict1
print(f'dict2 {dict2}')
dict2['height'] = 6.91
print('After modifying the copy')
print(f'dict1 {dict1}')
print(f'dict2 {dict2}')
Use the copy() method to safely copy a Dictionary. 
dict1 = {'first_name':'John', 'last_name':'Doe', 'age':21}
dict2 = dict1.copy()
print(f'dict2 {dict2}')
dict2['height'] = 6.91
print('After modifying the copy')
print(f'dict1 {dict1}')
print(f'dict2 {dict2}')
The dict() function can also be used to safely copy a Dictionary. Pass the Dictionary you want to copy as an argument to the function. 
dict1 = {'first_name':'John', 'last_name':'Doe', 'age':21}
dict2 = dict(dict1)
print(f'dict2 {dict2}')
dict2['height'] = 6.91
print('After modifying the copy')
print(f'dict1 {dict1}')
print(f'dict2 {dict2}')
