Variables

Variables are used to store data in computer memory. Python is dynamically typed, variable's Data Type is evaluated at runtime. Python has Type inference; the Interpreter will figure out the variable's Data Type from the Value assigned to it. Since a variable's Data Type is evaluated at runtime, you can reassign a variable to a value of a different data type. Variable names are case sensitive . A variable name must start with a letter or an underscore and can only contain Alphanumeric characters and an underscore. Give Variables meaningful names eg student_height instead of sh .

The Assignment Operator (=) is used to assign a value to a variable. Assignment is evaluated from right to left. 

character_name = 'Morpheus'

Data Types

Primitive Data Types are predefined types of data. They are the most basic data types provided by a Programming Language. Python support the following Primitive Data Types :

  • Numbers

    • Integers are negative or positive whole numbers ... 
    # integers
student_age = 16

    Floating Point (Floats) are Real Numbers with a Decimal Point.. 

    # a float
student_height = 6.24

    Complex j as the imaginery part 

    # a complex
student_height = 6j

  • Boolean

    • Boolean variables can have either True or False as values.. 
    # boolean
is_raining = True
is_sunny = False
    note the values start with capital letters.

  • String

    • A String is a sequence of Characters. In Python, strings are delimited by either Single quotes('') or Double quotes(""). 
    # single quotes
a_string ='spam'
print(a_string)
# double quotes
another_string ="eggs"
print(another_string)
    Python does not have a Character Data Type, a character is simply a String of length 1.

Check Data Type

Use the type() function to check a variable's data type. 

is_sunny = True
type_of_is_sunny = type(is_sunny)
print(type_of_is_sunny)

Type Conversion

A variable can be converted from one data type to another (Type Casting)

Use the float() function to convert to a float. 

student_age = 16
print("before converting")
print(student_age)
print(type(student_age))
      
# from integer to float
print("after converting")
student_age = float(student_age)
print(student_age)
print(type(student_age))

Use the int() function to convert to an integer.

height = 6.24
print("before converting")
print(height)
print(type(height))
      
# from float to integer
print("after converting")
height = int(height)
print(height)
print(type(height))
Note when a float is converted to an int, any fractional part will be discarded.

Use the str() function to convert to a string.

height = 9.24
print("before converting")
print(height)
print(type(height))
      
# from float to string
print("after converting")
height = str(height)
print(height)
print(type(height))

To convert a string to a number, it must be numeric. Use the isnumeric() method to check wether the characters in a string are numeric.

# from string to integer
student_age = "16"
print(student_age.isnumeric())
print("before converting")
print(student_age)
print(type(student_age))
      
print("after converting")
student_age = int(student_age)
print(student_age)
print(type(student_age))

# from string to float
student_height = "9.24"
print(student_height.isnumeric())
print("before converting")
print(student_height)
print(type(student_height))
      
print("after converting")
student_height = float(student_height)
print(student_height)
print(type(student_height))

If you convert the Boolean True and False to Integers or Floats , you'll get 1 and 0 respectively.

print('From Boolean to int')
is_raining = True
is_sunny = False
true_to_int = int(is_raining)
false_to_int = int(is_sunny)
print(true_to_int)
print(false_to_int)

print('From Boolean to Float')
is_raining = True
is_sunny = False
true_to_float = float(is_raining)
false_to_float = float(is_sunny)
print(true_to_float)
print(false_to_float)
If you convert the Integers 1 and 0 to boolean, you'll get True and False. Use the bool() function to convert Integers or Floats to Boolean. Note that when any non-zero is converted to boolean, evaluates to True.

In Python all datatypes are Objects by default, unlike in other Programming Languages where you have to convert Primitive Datatypes to Objects in order to do operations with other Objects.
