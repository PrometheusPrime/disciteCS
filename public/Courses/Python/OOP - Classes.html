instance attributes are defined in the __init__() method. Their values are unique to each instance of the class.

Methods are function defined in a class. They are called on a class or its instance with the dot notation.

class Person:
    number_of_people = 0

    def __init__(self,name,age,height):
        self.name = name
        self.age = age
        self.height = height
        Person.number_of_people = Person.number_of_people + 1

    def personal_info(self):
        print(f'name : {self.name} \nage : {self.age} \nheight : {self.height}')

    def greet(self):
        print(f'{self.name} says hi')

person_john = Person('John Wick',40,5.4)
