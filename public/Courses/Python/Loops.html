Loops

Loops are used to repeatedly execute code if a certain condition or an expression is True.

While Loop

The While Loop is used to repeatedly execute code while the condition is True. it checks the condition first, if it is True, it's block will executed. go back and check the looping condition and will keep on doing so until the condition becomes False. 

while condition:
    code to be executed repeatedly

numbers = 0
while numbers < 16:
    print(numbers)
If the condition does not evaluate to False, the Loop runs indefinitely (infinite Loop). Press Ctrl + C to stop execution.A condition or an expression can be updated inside the Loop so that it eventually evaluates to False. 
numbers = 0
while numbers < 16: 
    print(numbers)
    numbers += 1

For Loop

The For Loop loops through all individual elements in an Iterable or a Collection, stops execution when it reaches the last one. An Iterable is an Object that can return it's elements one at a time. 

for everything in something:
    #do something

greeting ="Hello World"
for character in greeting:
    print(character)

range() generates a sequence of numbers. 

for numbers in range(16):
    print(numbers)
the range function takes optional start and step 
for something in range(start,stop,step):
    do something
It will generate numbers from the start, skip the specified step before printing the next number. 
for numbers in range(0,101,10):
    print(numbers)
