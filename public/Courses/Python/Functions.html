Functions

A Functions is a block of statement(s) that perform a specific Task and or returns a Value.The def keyword is used to define a function.

To create a function; write def, function name followed by parentheses and then a colon. Python uses identation to delimit a block of code. Use equal indentation for every line of code in the function body. Add an empty line to signal the end of a block of code.

def function_name():
    #function body

The body of a function can not be left empty .If you're going to implement the function later use the pass keyword otherwise you'll get an error. 

def greet():
    pass

A function is executed by calling it. To call a function, write the function name followed by parenthesis. Functions can be called as many times as needed.

def greet():
    print('Hello there')

#calling a function
greet()

The return keyword is used to return a Value from a function. Statements after the return statement will not be executed.

def greet():
    return'Hi there'

print(greet())
A Function returns a default value none, even if it doesn't have the return keyword.

Parameters and Arguments

Functions can accept data .

A Parameter is a Variable defined in the parenthesis when defining the function.

Arguments are actual data passed into a function when calling it. Arguments are usually shortened to args

def greet(first_name):
    print(f'Hello {first_name}')
            
greet('John')

def sqrd(x):
    return x ** 2
                
print(sqrd(2))
A function can take any number of Parameters, comma seperate the parameters. A function must be called with the number of arguments it expects. 
def greet(first_name,last_name):
    print(f'Hello {first_name} {last_name}')

greet('John','Doe')
