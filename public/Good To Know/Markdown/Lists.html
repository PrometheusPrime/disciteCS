Lists

Unordered List

To create Unordered List,use asterisks(*), pluses(+) or hyphens(-) as list item markers. Add a space between the list item markers and the list item. 

* Sunday
* Monday
* Tuesday
    
+ Wednesday  
+ Thursday
+ Friday
    
- Saturday
- Sunday
- Monday

Ordered List

To create an Ordered List, use numbers followed by a period as list item markers. Add a space after a period then the list item. The numbers don't have to be in numerical order, but they should start with one. 

1. Monday
2. Tuesday
3. Wednesday
    
When You can only count up to 1 
1. Monday
1. Tuesday
1. Wednesday  
    
Let the computer do all the work.
1. Monday
55. Tuesday
8. Wednesday
