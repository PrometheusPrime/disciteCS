Signed Binary

In the previous module unsigned binary positive, how can we represent negative and positive numbers (Signed Binary).

Sign and Magnitude

left most bit is to denote a sign and the remaining bits is the Magnitude (value) of the Number. If left most bit is 0 the number is positive, if it is 1 the number is negative. If we have a 5 bit binary number, only 4 bits are used to represent the actual value.
