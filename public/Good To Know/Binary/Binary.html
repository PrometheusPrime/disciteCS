Binary

Binary is a Number System used by computers. Binary is a base 2 Number System (there are two symbols to count with (0 and 1)),We can only count upto 1 and then we start reusing the symbols.

The 0 or 1 is called a Bit and is a smallest unit of data.A Bit has a maximum value of 1. A Binary number (Bitword) can be a sequence of 0s and / 1s of any length. The value of a digit in a Binary Number is that digit x 2n, where n is the position of the digit. The rightmost digit is at position 0.
